## 核心定位

大师兄是口述/讲座类内容,核心独特性在**"价值观层"**(长期主义/付出/扫地僧),而非"操作层"。这与**周易**(哲学层)和**炒股的智慧**(操作层)形成互补。

**大师兄的 6 大基石 skill**:

1. **dashixiong-long-termism** (长期主义) — 投资哲学基础
2. **dashixiong-keep-one-bite** (留一口) — 风险控制基础
3. **dashixiong-cash-cow** (现金奶牛识别) — 选股/选企业
4. **dashixiong-cultural-investment** (文化投资) — 跨域思维
5. **dashixiong-sweeper-monk** (扫地僧) — 做人/态度
6. **dashixiong-giving** (付出哲学) — 人生意义

## 大师兄的独特价值

大师兄的 6 个 skill 中,**sweeper-monk** 和 **giving** 是周易/炒股/经济学都不覆盖的"做人/态度/价值观"层。

这是大师兄的**独特价值**,也是 user 在做"长期人生选择"时的核心 skill。

**周易**: 给"哲学"和"原理"
**炒股的智慧**: 给"操作"和"方法"
**经济学原理**: 给"理论"和"框架"
**大师兄**: 给"价值观"和"人生态度"

四本书形成完整覆盖:
- 周易 → 决策哲学
- 炒股的智慧 → 投资操作
- 经济学原理 → 经济框架
- 大师兄 → 价值观与态度
