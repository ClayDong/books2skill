# 大师兄 — 术语候选 (glossary)

> 从大师兄口述稿中抽取的术语 (12个)

---

## g01 — 现金奶牛 (Cash Cow)
- **大师兄定义**: 持续产生现金流,不需要不断投入的企业
- **现代对应**: Free Cash Flow > CapEx 的企业
- **大师兄用法**: 选股/选企业的核心标准

## g02 — 留一口 (Keep One Bite)
- **大师兄定义**: 永远不满仓,永远保留调整能力
- **现代对应**: Risk Management / Margin of Safety
- **大师兄用法**: 投资/做人/做事

## g03 — 扫地僧 (Sweeper Monk)
- **大师兄定义**: 表面平凡,实则深厚
- **现代对应**: Humility + Excellence
- **大师兄用法**: 做人之道

## g04 — 付出 (Giving)
- **大师兄定义**: 人生的核心是付出
- **现代对应**: Contribution Mindset
- **大师兄用法**: 做人/职业/创业

## g05 — 火象星座 (Fire Sign)
- **大师兄定义**: 行动力/激情/担当
- **现代对应**: Action-oriented
- **大师兄用法**: 创业/危机/做决断

## g06 — 垂直制造 (Vertical Manufacturing)
- **大师兄定义**: AI 在制造端的垂直应用
- **现代对应**: Vertical AI / Industrial AI
- **大师兄用法**: AI 投资判断

## g07 — 表外/表内 (Off-balance / On-balance)
- **大师兄定义**: 金融资产的表外/表内
- **现代对应**: Off-balance Sheet / On-balance Sheet
- **大师兄用法**: 宏观风险判断

## g08 — 数字财富 (Digital Wealth)
- **大师兄定义**: 账户上的数字
- **现代对应**: Paper Wealth
- **大师兄用法**: 财富观

## g09 — 实物 (Real Asset)
- **大师兄定义**: 真实资产,与数字财富相对
- **现代对应**: Real Asset / Tangible Asset
- **大师兄用法**: 财富配置

## g10 — 长期主义 (Long-Termism)
- **大师兄定义**: 决策看长期
- **现代对应**: Long-termism / Patient Capital
- **大师兄用法**: 投资/创业/做人

## g11 — 跨域思维 (Cross-Domain Thinking)
- **大师兄定义**: 历史/文化/投资 互证
- **现代对应**: Interdisciplinary Thinking
- **大师兄用法**: 决策/学习

## g12 — 历史押韵 (Historical Rhyme)
- **大师兄定义**: 历史不会简单重复但会押韵
- **现代对应**: Historical Pattern Recognition
- **大师兄用法**: 投资/决策
