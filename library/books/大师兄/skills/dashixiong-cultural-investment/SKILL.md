# 文化投资决策 (Cultural Investment Decision)

## R — 原文 (Reading)

> 看完东方,我们来看西方。

> 为什么要研究高层的政治、经济、历史、文化?为了更好的去稳定,为了让老百姓,水可载舟,亦可覆舟,这儿安定了,整个国家才能定啊,明白吗?这就是精髓。

> 你看郑和当年,如此强大的实力,今天已经有很多的史实记载说,郑和欧亚非肯定是到过的,甚至连美洲他都是到过的。那为什么郑和没有在任何一块大陆上,留下大明朝的殖民地呢?

> 当时郑和带回来的,第一次、第二次下西洋,经历了明成祖、明仁宗、明宣宗,皇帝一任一任的过去,一任一任的去世。

> 郑和原先姓马对不对?小名三宝,所以郑和叫马三宝,明成祖朱棣赐他一个姓,赐他姓郑,叫郑和。

## I — 方法论骨架 (Interpretation)

大师兄文化投资的核心:**用历史/文化案例类比投资/商业决策**。这是"跨域思维"的具体应用。

### 文化-投资类比框架 (f05)

```
步骤 1: 找到历史/文化案例
  - 哪些历史事件与当前决策相似?
  - 找到 2-3 个类比

步骤 2: 提取类比的"模式"
  - 历史事件的"模式"是什么?
  - 与当前决策的"模式"是否相同?

步骤 3: 应用类比
  - 历史事件的结果是什么?
  - 当前决策可能的结果是什么?

步骤 4: 验证
  - 这个类比合理吗?
  - 有没有不同的"模式"?
```

### 大师兄的标志性类比

#### 类比 1: 郑和 vs 麦哲伦

```
郑和 (1371-1433):
  - 280 艘船
  - 2 万多人
  - 7 次下西洋
  - 贸易型 (不带警察)
  - 结果: 没有殖民地,但建立了朝贡体系

麦哲伦 (1480-1521):
  - 5 艘船
  - 300 人
  - 1 次环球 (未完成)
  - 征服型
  - 结果: 命都没有了

类比投资/创业:
  - 贸易型 = 与合作伙伴共赢
  - 征服型 = 试图压制竞争对手
  - 大师兄建议: 选贸易型
```

#### 类比 2: 皇室培养 vs 企业传承

```
皇室培养:
  - 中国历史上最在乎子孙培养的是皇室
  - 康熙/雍正/乾隆 3-4 岁开始启蒙
  - 长期主义
  - 选最优秀的师傅

企业传承:
  - 创始人最在乎企业传承
  - 早期培养接班人
  - 长期主义
  - 选最优秀的导师

类比:
  - 早期培养 > 临时学习
  - 长期主义 > 短期业绩
  - 选好师傅 > 选好学校
```

#### 类比 3: 大航海 vs 互联网革命

```
大航海 (15-17 世纪):
  - 新航线开辟
  - 全球贸易形成
  - 强国崛起 (西班牙/葡萄牙/英国)
  - 弱者被殖民

互联网革命 (1990s-):
  - 新技术出现
  - 全球连接
  - 巨头崛起 (Google/Apple/Microsoft)
  - 传统行业被颠覆

类比:
  - 大航海: 抓住新航线
  - 互联网: 抓住新技术
  - 共同的"模式": 新技术/新航线改变世界
```

#### 类比 4: 表外/表内 vs 隐性风险

```
表外/表内 (金融):
  - 2008 次贷危机
  - 隐性风险最终会暴露
  - 大师兄: 200 多年没有发生,但发生了

隐性风险 (其他领域):
  - 创业的隐性风险
  - 关系的隐性风险
  - 健康的隐性风险

类比:
  - 表外风险: 不要被表面稳定迷惑
  - 隐性风险: 关注"看不见"的风险
```

### 跨域思维的 3 个方法

#### 方法 1: 历史模式识别
- 当前问题 → 历史上有什么相似?
- 提取"模式"
- 应用到当前

#### 方法 2: 东西方对比
- 东方如何处理?
- 西方如何处理?
- 综合两种智慧

#### 方法 3: 多领域互证
- 历史/文化/投资/做人 → 互为印证
- 跨域一致性 → 强信号

## A1 — 书中的应用 (Past Application)

### 案例 1: 郑和 vs 麦哲伦 — 商业模式选择

- **大师兄用类比**:
  - 郑和贸易型 → 与合作伙伴共赢
  - 麦哲伦征服型 → 试图压制
- **应用**:
  - 创业选贸易型 (合作) 而非征服型 (垄断)
  - 投资选"现金流共享" (股票) 而非"控制" (私募)
- **现代案例**:
  - 苹果: 开放 App Store (贸易型) → 成功
  - 某些试图控制一切的厂商 → 失败

### 案例 2: 皇室培养 — 长期主义

- **大师兄用类比**:
  - 皇室 3-4 岁启蒙
  - 现代企业早期培养
- **应用**:
  - 子女教育: 早期 > 临时
  - 员工培养: 早期 > 临时
  - 投资: 长期 > 短期
- **现代案例**:
  - 巴菲特从小开始投资
  - 稻盛和夫 27 岁创立京瓷

### 案例 3: 大航海 vs 互联网

- **大师兄用类比**:
  - 大航海: 强国崛起
  - 互联网: 巨头崛起
- **应用**:
  - 抓住新技术 (AI/区块链/新能源)
  - 警惕被颠覆 (传统巨头)
- **现代案例**:
  - 微软抓住互联网
  - 诺基亚错过智能手机

### 案例 4: 表外/表内 — 隐性风险

- **大师兄用类比**:
  - 表外风险最终会暴露
  - 隐性风险最终会暴露
- **应用**:
  - 投资: 不要被表面稳定迷惑
  - 创业: 关注隐性风险
  - 健康: 重视体检
- **现代案例**:
  - 2008 次贷危机
  - 安然事件
  - 长期不锻炼的健康风险

## A2 — 触发场景 (Trigger Scenarios)

### 场景 1: 投资决策

- 用户:"这家公司值得投资吗?"
- 应用: 文化-投资类比
  - 历史上类似公司怎样?
  - 商业模式类比 (贸易型 vs 征服型)
  - 长期模式 (皇室培养类比)
- 输出: 投资建议

### 场景 2: 创业方向

- 用户:"这个创业方向对吗?"
- 应用: 大航海/互联网类比
  - 这是"新航线"还是"老业务"?
  - 抓得住吗?
- 输出: 创业方向建议

### 场景 3: 子女教育

- 用户:"该怎么教育孩子?"
- 应用: 皇室培养类比
  - 早期培养 > 临时学习
  - 长期主义 > 短期成绩
- 输出: 教育建议

### 场景 4: 国际化决策

- 用户:"该不该出海?"
- 应用: 郑和/麦哲伦类比
  - 贸易型 vs 征服型
  - 准备是否充分 (280艘 vs 5艘)
- 输出: 出海建议

### 场景 5: 风险评估

- 用户:"这个风险是真的吗?"
- 应用: 表外/表内类比
  - 表面稳定 ≠ 真实稳定
  - 关注隐性风险
- 输出: 风险评估

## B — 边界 (Boundary)

### 不适用场景

- **没有历史/文化背景的全新问题**: 类比无从下手
- **纯技术问题**: 用其他 skill
- **短期决策**: 类比通常需要长期视角
- **紧急情况**: 没时间做类比

### 局限

- **类比可能牵强**: 历史不会简单重复
- **过度类比**: 把所有事都比作历史,可能失真
- **文化差异**: 东西方类比需谨慎
- **"押韵"而非"重复"**: 大师兄自己也说"历史会押韵,但不会简单重复"

### 失效条件

- **真正全新的事**: 历史上没发生过,类比失效
- **过度解读**: 把巧合当必然
- **缺乏领域知识**: 找不到合适的类比
