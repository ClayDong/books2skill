# 留一口资金管理 (Keep-One-Bite Position Management)

## R — 原文 (Reading)

> 我们一直都留一口好吗?留一口。

> 留一口。留一口是什么意思?就是说你的账户里,永远要留一点钱,不是为了赚更多,是为了在市场最坏的时候,你还有子弹。

> 我跟大家讲了未来十年不要,不要高杠杆啊。我怕它崩啊,我怕它崩啊,我怕它在你庆祝利润最高点的时候,但是它是金融衍生啊,它不是实物啊,它只是你账户里面的数字啊。

> 我怕它崩啊,我怕我怕你承受不了,我怕你赢了十年、20年,那一秒钟你承受不了啊。

---

## I — 方法论骨架 (Interpretation)

大师兄"留一口"原则:**永远不满仓,永远保留调整能力**。这是大师兄投资哲学的两大基石之一。

### "留一口"的核心定义

**留一口 ≠ 留固定的百分比**
- 不是机械地"30% 不投"
- 而是"保留调整能力"

**留一口的 3 个层次**:
1. **资金层面**: 账户里永远有钱
2. **心态层面**: 永远有"退出"的能力
3. **机会层面**: 市场最坏时还有子弹

### 留一口的 4 个目的

```
目的 1: 危机中加仓
  - 市场崩盘时,你有子弹抄底
  - 不是为了赚更多,而是为了"还有机会"

目的 2: 心理稳定
  - 满仓 100% 时,任何下跌都让你崩溃
  - 留一口时,下跌你不会"被迫"行动

目的 3: 机会成本
  - 满仓意味着错过其他机会
  - 留一口意味着"选择权"

目的 4: 长期生存
  - 投资是 30 年的事
  - 短期崩一次就退场,不值
```

### 留一口的 4 个错误

| 错误 | 表现 | 后果 |
|---|---|---|
| **满仓梭哈** | 100% 投入 | 下跌时无法行动 |
| **加杠杆** | 借钱投资 | 爆仓风险 |
| **不止损** | 越跌越买,但满仓 | 没有子弹 |
| **频繁进出** | 留不住"一口" | 错过机会 |

### 留一口的 4 个规则

**规则 1: 永远不满仓**
- 最高仓位: 80-90%
- 留 10-20% 不动
- "一口" 不是具体的百分比,而是"调整能力"

**规则 2: 永远不加杠杆**
- 不借钱投资
- 不融资融券
- 不参与高杠杆衍生品

**规则 3: 越涨越卖一点点**
- 涨 50%: 卖 10%
- 涨 100%: 卖 20%
- 涨 200%: 卖 30%
- 让"留一口"动态增加

**规则 4: 越跌越买一点点**
- 跌 30%: 加仓 5%
- 跌 50%: 加仓 10%
- 跌 70%: 加仓 15%
- 用"留一口"的钱加仓

## A1 — 书中的应用 (Past Application)

### 案例 1: 大师兄本人的"留一口"

- **背景**: 大师兄多次强调"留一口"
- **应用**:
  - 即使看好,也不满仓
  - 即使涨了,也留一口
  - 即使跌了,也不恐慌 (有子弹)
- **关键**: 长期主义 + 留一口 = 穿越牛熊

### 案例 2: 巴菲特现金为王 (1969-1980s)

- **背景**: 巴菲特在 1969 年市场高估时主动退出
- **大师兄视角**: 巴菲特在顶峰时"留一口" (没把伯克希尔变成 100% 股票)
- **应用**:
  - 巴菲特保留大量现金
  - 1973-74 跌到底部时,巴菲特有子弹抄底
  - 这是"留一口"的最佳实践
- **结果**: 巴菲特成为史上最成功投资者之一

### 案例 3: 2008 年次贷危机

- **背景**: 多数人满仓+杠杆
- **大师兄视角**: 表外进表内,虚拟全部打开
- **应用**:
  - 满仓+杠杆 → 血本无归
  - 留一口的人 → 危机中抄底
- **结果**: 幸存者偏差明显

### 案例 4: 雷曼时刻

- **背景**: 2008 年 9 月雷曼破产
- **错误示范**: 满仓+杠杆
- **结果**: 财富一夜归零
- **大师兄警示**: "我怕你赢了十年、二十年,那一秒钟你承受不了"

### 反例 1: 满仓被套

- **表现**: 100% 投入,跌 50%
- **后果**:
  - 没有子弹加仓
  - 心理崩溃
  - 长期无法回本
- **大师兄警示**: "留一口"是为了"还有机会"

### 反例 2: 杠杆爆仓

- **表现**: 借钱投资,跌 30% 爆仓
- **后果**:
  - 不只是亏钱,还欠债
  - 心理彻底崩溃
- **大师兄警示**: 即使长期看好,也不要高杠杆

## A2 — 触发场景 (Trigger Scenarios)

### 场景 1: 是否加仓

- 用户:"我看好这只股票,该不该加仓到 100%?"
- 应用: 留一口原则
  - 最高 80-90%
  - 留 10-20%
- 输出: 不满仓 建议

### 场景 2: 是否使用杠杆

- 用户:"融资融券能放大收益,该不该用?"
- 应用: 留一口原则
  - 永远不加杠杆
  - 杠杆 = 满仓的平方
- 输出: 不用杠杆 建议

### 场景 3: 是否止盈

- 用户:"股票涨了 100%,该全卖吗?"
- 应用: 留一口原则
  - 全卖 = 0 仓位
  - 留 20% 长期持有
  - 卖 80% 落袋为安
- 输出: 分批止盈 建议

### 场景 4: 市场崩盘应对

- 用户:"市场崩了 30%,我满仓了,该怎么办?"
- 应用: 留一口原则
  - 如果你"留一口" → 没关系,有子弹
  - 如果没"留一口" → 已无法行动,只能等待
- 输出: 复盘 → 重新开始时一定要"留一口"

### 场景 5: 创业资金管理

- 用户:"创业资金该全投吗?"
- 应用: 留一口原则
  - 创业也要"留一口"
  - 留 6-12 月运营资金
- 输出: 留现金流 建议

## E — 执行 (Execution)

### 步骤 1: 当前仓位评估 (5 分钟)

```
1. 我现在的仓位是多少?
   - 100% (满仓)
   - 80-90% (高位)
   - 50-70% (中位)
   - 30-50% (低位)
   - 0-30% (基本空仓)

2. 我是否使用了杠杆?
   - [ ] 是
   - [ ] 否

3. 我的"留一口"在哪里?
   - 现金: ___%
   - 备用资金: ___%
   - 加仓能力: ___%
```

### 步骤 2: 留一口设计

```
设计原则:
- 最高仓位: 80-90%
- 最低仓位: 30-50%
- 留 10-20% 不动
- 永远不加杠杆

我的设计:
- 目标仓位: ___% (建议 60-80%)
- "留一口": ___% (建议 20-40%)
- 杠杆使用: 0%
```

### 步骤 3: 越涨越卖计划

```
涨 30%: 卖 ___% (建议 5%)
涨 50%: 卖 ___% (建议 10%)
涨 100%: 卖 ___% (建议 20%)
涨 200%: 卖 ___% (建议 30%)
涨 300%: 卖 ___% (建议 40%)

原则: 越涨越卖,让"留一口"动态增加
```

### 步骤 4: 越跌越买计划

```
跌 10%: 不动
跌 30%: 加仓 ___% (建议 5%,前提是基本面没变)
跌 50%: 加仓 ___% (建议 10%)
跌 70%: 加仓 ___% (建议 15%)

原则: 越跌越买,但要"小步"加,不要梭哈
```

### 步骤 5: 月度复盘

```
复盘问题:
1. 我是否做到了"留一口"?
2. 我是否在涨时越涨越卖?
3. 我是否在跌时越跌越买?
4. 我是否避免了杠杆?
5. 心理是否稳定?
```

## B — 边界 (Boundary)

### 不适用场景

- **纯价值分析**: 用 dashixiong-cash-cow
- **一次性事件**: 用其他 skill
- **已满仓无法调整**: 接受现实,用 stock-psychology-check 调整心态
- **没有资金管理能力的场景**: 不适合 (如买房首付)

### 局限

- **"留一口"会降低收益**: 满仓的回报 > 留一口
- **需要纪律**: 涨时想多投,跌时想止损,都需要纪律
- **不同市场不同**: A 股 80% 仓位可能比美股 100% 仓位风险更高
- **心理挑战**: 看着别人满仓赚更多,很难坚持"留一口"

### 失效条件

- **市场长期向上**: "留一口"会错过收益
- **个人风险承受力极强**: 满仓也能承受
- **市场效率极高**: 没有"抄底"机会

### 与现代投资理论的整合

| 大师兄 | 现代工具 | 整合方式 |
|---|---|---|
| 永远不满仓 | Kelly Criterion / 仓位管理 | 用 Kelly 公式计算最优仓位 |
| 不加杠杆 | 风险敞口管理 | 用 VaR 控制风险 |
| 越涨越卖 | 动态再平衡 | 用再平衡策略 |
| 越跌越买 | 价值平均法 | 用 Value Averaging |
| 心态稳定 | 行为金融学 | 用纪律+认知偏差识别 |
