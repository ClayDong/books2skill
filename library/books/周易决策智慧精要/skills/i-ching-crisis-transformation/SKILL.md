## I — 方法论骨架 (Interpretation)

易经的危机观:"危机"是"危险+机会"的双面词。易经不否认危机的痛苦,但强调**在危机中如何转化是关键**。

### 两种危机的区分

**坎 (Kǎn) — 外险型危机**
- 性质: 来自外部的险境 (战争/疫情/政策变化/行业颠覆)
- 应对: "维心亨,行有尚" — 内心通透,行动有目标
- 关键: 不被外险压垮,主动寻找突围方向
- 现代案例: 疫情对线下零售/制造业的冲击 (外部风险)

**困 (Kùn) — 内困型危机**
- 性质: 来自内部的困境 (能力不足/资源枯竭/团队崩盘)
- 应对: "困,亨,贞,大人吉无咎" — 坚守正道,寻找大人(导师) 帮助
- 关键: 不要乱动 (有言不信),要主动求助
- 现代案例: 创始人能力跟不上公司发展 (内部困境)

### 危机转化四步

#### 第一步: 维心亨 — 内心通透

> "习坎,有孚,维心亨,行有尚"

- 危机中,内心必须通透
- "维"是"系/连结" — 把心系在某个核心上
- 核心建议:
  1. 写下"危机对我的 3 个教训"
  2. 写下"我在危机中坚守的 3 个原则"
  3. 每天冥想 10 分钟 (传统: 静坐/打坐)
- 内心不通透,任何行动都加剧危机

#### 第二步: 辨坎困 — 区分危机类型

```
外险型(坎) →  行动突围 → 改变外部环境
内困型(困) →  静守求助 → 改变内在能力
混合型     →  分步处理 → 外险先突围,内困后解决
```

**反向判断**:
- 如果乱动加剧危机 → 可能是内困型,该静守
- 如果不动加剧危机 → 可能是外险型,该行动

#### 第三步: 变则通 — 主动求变

> "穷则变,变则通,通则久"

- 危机的本质是"穷" (走到尽头)
- 走到尽头必须变,不变就是死路
- 变的 3 个层次:
  - **小变**: 调整方法/工具 (例: 转行不转岗)
  - **中变**: 调整方向/领域 (例: 转行+转岗)
  - **大变**: 调整身份/价值观 (例: 创业 → 打工)
- 选哪一层变: 评估"不变会死吗"+"变了能否更好"

#### 第四步: 大人吉 — 寻找"大人"帮助

> "困,大人吉无咎"

- 危机中独自难以脱困,要找"大人" (导师/贵人/专家)
- 找"大人"的 3 个标准:
  1. 经历过类似危机并脱困
  2. 没有利益冲突 (不为推销)
  3. 能给出具体的行动建议 (不只是鼓励)
- 不要"有言不信" — 别人建议不信,独自硬撑

### 危机中的"勿做"清单

| 勿做 | 原因 |
|---|---|
| 情绪化决策 | 危机中容易冲动,要"维心亨" |
| 病急乱投医 | "有言不信" — 不要什么都试 |
| 逃避现实 | 喝酒/游戏/疯狂购物都是逃避 |
| 死撑到底 | "非所困而困焉" — 该退就退 |
| 抱怨他人 | 抱怨不解决问题,反而消耗能量 |
| 隐瞒实情 | 瞒到最后无法收拾,早暴露早解决 |

### 危机后的"化" (Transformation)

危机不是终点,是转折点。化危为机的关键:
- 重新认识自己 (危机前的盲点)
- 重新选择方向 (危机前的路径错误)
- 重新建立支持 (危机前的孤立无援)
- "困而后通" — 通之后的境界比之前更高

## A1 — 书中的应用 (Past Application)

### 案例 1: 文王囚羑里 — 坎中转化

- **问题**: 商纣王囚文王于羑里七年,生命危在旦夕
- **危机类型**: 外险型 (商纣)+ 内困型 (周国被监控) 混合
- **方法论的使用**:
  1. 维心亨 — 在囚禁中保持内心通透
  2. 变则通 — 把囚禁时间用于"重八卦为六十四卦"
  3. 大人吉 — 找散宜生等大臣传递消息
- **结果**: 囚禁成为创造期,周易诞生
- **化**: 从"西伯侯"变成"文王",奠定周朝 800 年基业

### 案例 2: 越王勾践 — 困中转化

- **问题**: 越被吴灭,勾践为奴三年,身心俱辱
- **危机类型**: 内困型 (从霸主变为奴)+ 外险型 (吴国强大)
- **方法论的使用**:
  1. 维心亨 — "勾践之地,南至于句无,北至于御儿,东至于鄞,西至于姑蔑" (内心理想不灭)
  2. 辨坎困 — 表面臣服 (应对外险), 内部准备 (应对内困)
  3. 变则通 — 22 年变奴隶为霸主
  4. 大人吉 — 文种/范蠡/西施 都是"大人"
- **结果**: 灭吴称霸

### 案例 3: 王阳明龙场悟道 — 困中精神突破

- **问题**: 王阳明被贬贵州龙场 (今修文),瘴气弥漫,语言不通,生死未卜
- **危机类型**: 内困型 (精神危机)
- **方法论的使用**:
  1. 维心亨 — 在石棺中"格物致知"
  2. 辨坎困 — 这是内困,不是外险
  3. 变则通 — 突破朱熹理学的"格物"框架
  4. 大人吉 — 当地人/学生/书友的支持
- **结果**: 龙场悟道,创立心学
- **化**: 从"被贬官员"变成"心学宗师",影响东亚 500 年

### 反例: 王莽改革失败 — 困中未化

- **问题**: 王莽以外戚身份代汉,推行激进改革
- **错误**:
  1. 维心亨失败 — 改革脱离民心,内心仍执念"新政理想"
  2. 辨坎困失败 — 没区分内困 (民众不支持) 和外险 (旧势力反对)
  3. 变则通失败 — 改革没考虑节奏,一步到位
  4. 大人吉失败 — 杀尽反对者,无人敢提意见
- **结果**: 改革失败,新朝 15 年而亡

## A2 — 触发场景 (Trigger Scenarios)

### 场景 1: 失业
- 应用: 区分"行业衰退" (坎) 和"个人能力" (困)
- 输出: 应对策略 (转行 vs 提升能力)

### 场景 2: 创业失败
- 应用: 多为"内困型" (能力/资源/团队)
- 输出: 复盘方向 + 重新出发 vs 转换轨道

### 场景 3: 关系破裂 (离婚/分手)
- 应用: 内困型为主 (关系处理能力)
- 输出: 修复 vs 接受 vs 重建

### 场景 4: 健康危机 (重大疾病)
- 应用: 坎+困 混合 (身体外险 + 心理内困)
- 输出: 治疗策略 + 心理建设

### 场景 5: 财务困境
- 应用: 多为内困型 (现金流管理)
- 输出: 紧急止血 + 长期重建

## E — 执行 (Execution)

### 步骤 1: 维心亨 — 内心通透练习 (立即执行)

每日 10 分钟,持续到危机解除:
```
1. 写下今天危机中的 1 个教训 (5 分钟)
2. 写下今天我坚守的 1 个原则 (3 分钟)  
3. 静坐/冥想 2 分钟 (保持内心清晰)
```

### 步骤 2: 辨坎困 — 危机类型诊断 (1-2 天)

回答以下问题:
```
1. 危机主要来自外部还是内部?
   - 外部: 行业/政策/市场/对手 → 坎型
   - 内部: 能力/资源/团队/心态 → 困型

2. 危机中"乱动"会加剧还是缓解?
   - 乱动加剧 → 困型,应静守
   - 乱动缓解 → 坎型,应行动

3. 是否有人可以求助?
   - 有可信的"大人" → 困型,可求助
   - 无人可信 → 坎型,需自力
```

### 步骤 3: 变则通 — 设计变化方案 (1 周)

```
小变方案 (调整方法/工具):
  - 例: 销售方法 / 工作工具 / 沟通方式
  - 评估: 不变会死吗? 变了能否更好?

中变方案 (调整方向/领域):
  - 例: 转岗 / 转行 / 转换合作方
  - 评估: 沉没成本 vs 未来收益

大变方案 (调整身份/价值观):
  - 例: 创业→打工 / 结婚→单身 / 国内→海外
  - 评估: 我是谁 vs 我想成为谁
```

### 步骤 4: 大人吉 — 寻找并联络"大人" (持续)

- 列出 3 个"大人"候选 (经历过类似危机的人)
- 主动联络,坦诚相告,寻求建议
- 不要"有言不信" — 至少执行一个建议试试

### 步骤 5: 持续监测 (每周)

```
危机转化进度 (1-10):
  - 维心亨: ___ (内心清晰度)
  - 辨坎困: ___ (类型判断准确度)
  - 变则通: ___ (变化方案执行度)
  - 大人吉: ___ (获得支持度)

下一步行动: ___
```

## B — 边界 (Boundary)

### 不适用场景

- **日常挫折**: 失败一个项目/一次争吵 — 不算真正的危机
- **需要专业心理治疗**: 抑郁/焦虑/创伤 — 请就医
- **需要紧急医疗**: 危及生命的健康问题 — 先救命
- **法律紧急情况**: 拘留/起诉 — 先找律师

### 局限

- **不解决具体操作问题**: 危机应对仍需具体方案
- **不替代专业意见**: 法律/医疗/财务问题需专业人士
- **文化适配**: 易经概念需翻译为现代语言,可能损失原意

### 失效条件

- **多重危机叠加**: 同时失业+离婚+疾病 — 需要更复杂的框架
- **长期慢性危机**: 持续多年的困境 (如照顾慢性病家人) — 易经框架可能不够
- **极端生存威胁**: 战争/饥荒 — 易经框架失效,先保命
