# 乾卦生命周期决策 (Life-Cycle Decision)

## R — 原文 (Reading)

>
> 九三,君子终日乾乾,夕惕若厉,无咎。
>
> 九四,或跃在渊,无咎。
>

>

> 潜龙勿用,下也。见龙在田,时舍也。终日乾乾,行事也。或跃在渊,自试也。飞龙在天,上治也。亢龙有悔,穷之灾也。
>
> — 《乾卦·文言》

## I — 方法论骨架 (Interpretation)

**乾卦六爻 = 人生/事业/项目生命周期六阶段**。每阶段有明确的"状态"+"行动"+"风险"+"信号"。

### 六阶段总览

```
初九 潜龙勿用  ──  积累期   状态: 隐藏/学习/等待
                行动: 蓄力,不强行出头
                风险: 急躁冒进
                信号: 能力不够,机会未到

九二 见龙在田  ──  显露期   状态: 初步展现,小试牛刀
                行动: 谨慎尝试,寻找"大人"(导师/伯乐)
                风险: 过度表现,树敌太多
                信号: 有人开始认可你

九三 终日乾乾  ──  奋斗期   状态: 日夜精进,警惕自满
                行动: 全力拼搏,日乾夕惕
                风险: 过劳/失去平衡
                信号: 处于"下不上下不田"的位置,需特别警惕

九四 或跃在渊  ──  抉择期   状态: 处于关键节点,大抉择
                行动: 评估跃升 vs 维持的代价
                风险: 错误选择,可能跌落深渊
                信号: 多个机会同时出现,需要取舍

九五 飞龙在天  ──  鼎盛期   状态: 达到顶峰,大展宏图
                行动: 巩固基础,服务他人
                风险: 居功自傲,听不进意见
                信号: 行业/领域/团队的领导者

上九 亢龙有悔  ──  衰退期   状态: 顶峰已过,势头减弱
                行动: 主动退让,转换轨道
                风险: 不知退,必有大悔
                信号: 失去群众基础 (高而无民)
```

### 关键转折点

#### 九三 → 九四 (奋斗 → 抉择) — 最大风险
- "终日乾乾" 转为 "或跃在渊"
- 从"做好自己的事"到"决定未来方向"
- 决策影响整个后半生
- **核心建议**: 不要因为"已经投入太多"而继续,要"评估未来收益"

#### 九四 → 九五 (抉择 → 鼎盛) — 最大机会
- 成功跃升后进入鼎盛
- 此时**最危险的是居功自傲**
- **核心建议**: 飞龙在天,利见大人 — 仍要"见" (依靠) 优秀的人

#### 九五 → 上九 (鼎盛 → 衰退) — 最大危机
- 顶峰时容易"亢"
- "贵而无位,高而无民,贤人在下位而无辅"
- **核心建议**: 主动"亢前退" — 在九五后期就准备退出

### 各阶段的"勿做"清单

| 阶段 | 千万不要做 |
|---|---|
| 初九 | 急于显露,参与不熟悉的领域,透支资源 |
| 九二 | 过度自信,认为"已经够了",忽视继续学习 |
| 九三 | 拼到过劳,失去家庭/健康,不停下来反思 |
| 九四 | 因为沉没成本而勉强继续,未真正评估选项 |
| 九五 | 居功自傲,听不进意见,任人唯亲 |
| 上九 | 死守已有成就,不愿转换轨道 |

## A1 — 书中的应用 (Past Application)

### 案例 1: 越王勾践 — 完整的六阶段历程

| 阶段 | 勾践的表现 |
|---|---|
| 初九 潜龙 | 被夫差击败为奴三年 (被动的潜龙) |
| 九二 见龙 | 归国后"见龙在田",表面臣服,暗中准备 |
| 九三 乾乾 | "十年生聚,十年教训",日夜操劳 |
| 九四 或跃 | 夫差北上争霸时,勾践面临"跃"或"不跃"的选择 |
| 九五 飞龙 | 灭吴称霸,成为春秋最后一位霸主 |
| 上九 亢龙? | 范蠡及时劝退,勾践本人略晚但未大悔 |

**结论**: 勾践完整经历六阶段,范蠡"知进退"是真正的"上九"处理方式。

### 案例 2: 文王 — 被动潜龙,主动见龙

- **问题**: 文王 50 岁前,商纣王还算贤明,文王作为"西伯侯"积累实力
- **方法论的使用**: 文王在"九三 终日乾乾"阶段积蓄力量
- **转折**: 商纣暴虐,文王被囚 (意外的"潜龙勿用")
- **应用**: 囚禁中把八卦重为六十四卦 (潜龙阶段的最大成就)
- **结果**: 获释后 "见龙在田" (诸侯归心),奠定周朝基础

### 案例 3: 微软 — 九五飞龙与革鼎转型

- **问题**: 2014年微软处于"亢龙"状态 — PC 时代没落,市值跌至 3000 亿
- **方法论的使用**: 鲍尔默下台,纳德拉接任 CEO
- **革**: 革掉"Windows 为中心"的思想
- **鼎**: 确立"移动优先+云优先" (鼎新)
- **结果**: 10 年后市值达 3 万亿 (重新"飞龙在天")

**关键教训**: 微软从"亢龙"通过"革"主动转为新的生命周期 (而非等待衰退)。

## A2 — 触发场景 (Trigger Scenarios)

### 场景 1: 职业迷茫
- 用户:"我 35 岁了,不知道下一步该做什么"
- 应用: 用六阶段定位 → 多半是"九三 乾乾"或"九四 或跃"阶段
- 输出: 阶段定位 + 阶段对应行动

### 场景 2: 事业顶峰
- 用户:"公司已经做到行业第一,该守还是该攻"
- 应用: 判断是否处于"九五飞龙"或已到"上九亢龙"
- 输出: 阶段判断 + 继续 vs 转型的建议

### 场景 3: 创业期
- 用户:"创业 2 年了,感觉卡住了"
- 应用: 判断是"初九潜龙"(积累不够)还是"九三乾乾"(需要日乾夕惕)
- 输出: 阶段判断 + 突破方向

### 场景 4: 退休决策
- 用户:"我 55 岁,该不该退休"
- 应用: 用"亢龙有悔"原则判断 — 是否有主动退的智慧
- 输出: 退 vs 不退的评估

### 场景 5: 项目周期
- 用户:"项目已经运营 5 年,该转型还是继续"
- 应用: 把项目看作"乾卦",判断处在哪一阶段
- 输出: 阶段判断 + 转型/继续/退出的建议

## E — 执行 (Execution)

### 步骤 1: 阶段定位 (15 分钟)

回答以下问题:

```
A. 状态自评 (1-10 分)
  - 我现在在积累/显露/奋斗/抉择/鼎盛/衰退的哪个状态?
  - 评分: 1=积累期, 6=衰退期

B. 行动评估
  - 我现在的行动是否符合当前阶段?
  - 例如: 潜龙阶段在大量应酬 (不符)
  - 例如: 飞龙阶段不主动退 (不符)

C. 风险评估
  - 当前阶段最大的"勿做"是什么?
  - 我是否正在做这件事?

D. 转换信号
  - 是否有从当前阶段转向下一阶段的信号?
  - 信号包括: 机会出现/能力匹配/环境变化
```

### 步骤 2: 阶段匹配检查

| 阶段 | 关键行动 | 关键不行动 |
|---|---|---|
| 初九 | 学习/积累/等待 | 急于显露 |
| 九二 | 小试/见大人 | 过度表现 |
| 九三 | 全力拼搏/警惕自满 | 拼到过劳 |
| 九四 | 评估跃升 vs 维持 | 因为沉没成本勉强 |
| 九五 | 巩固基础/服务他人 | 居功自傲 |
| 上九 | 主动退让/转换轨道 | 死守成就 |

### 步骤 3: 制定阶段行动计划

```
我现在的阶段: [填入]
本阶段最关键 3 件事: [填入]
本阶段绝对不做的事: [填入]
本阶段转换的信号: [填入]
```

### 步骤 4: 持续监测

每月/每季度重新评估:
- 我是否还在当前阶段? 还是已经转换?
- 当前阶段的"勿做"我是否在做?
- 下一阶段的"关键行动"我是否在准备?

## B — 边界 (Boundary)

### 不适用场景

- **纯技术/操作问题**: 本 skill 是阶段判断,不解决具体操作
- **没有明确目标的场景**: 本 skill 假设用户有清晰目标
- **多线并行的复杂场景**: 如果同时在多个领域,六阶段模型难以套用 (建议每个领域独立评估)

### 局限

- **模型简化**: 人生/事业不一定严格按六阶段线性发展
- **主观判断**: 阶段定位有主观性,建议与可信他人讨论
- **跨领域迁移困难**: 不同领域的"潜龙"含义不同 (个人/企业/项目)

### 失效条件

- **多阶段同时进行**: 事业巅峰但感情低谷,六爻难以同时表达
- **外部环境剧变**: 行业被颠覆 (如诺基亚),六爻模型失效
- **个人特殊情况**: 长期病患/特殊家庭情况,六爻需调整

### 与其他 i-ching-* skill 的边界

| skill | 关注点 | 不关注 |
|---|---|---|
| i-ching-life-cycle | 阶段判断 + 阶段行动 | 具体决策的"时位中正" |
| i-ching-time-position | 决策前自检 | 阶段判断 |
| i-ching-advance-retreat | 进退时机 | 整体生命周期 |
| i-ching-decision-omen | 早期信号 | 阶段判断 |
| i-ching-crisis-transformation | 危机中转化 | 正常阶段管理 |

**协同使用**: 先用 i-ching-life-cycle 定位阶段 → 再用 i-ching-time-position 做具体决策 → 用 i-ching-decision-omen 监测信号 → 必要时用 i-ching-crisis-transformation 处理危机。
