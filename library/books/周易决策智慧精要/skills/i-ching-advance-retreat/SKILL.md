# 进退决策 (Advance-Retreat Decision)

## R — 原文 (Reading)

> 亢龙有悔。
>
> — 《乾卦·上九》

> 亢之为言也,知进而不知退,知存而不知亡,知得而不知丧。其唯圣人乎!知进退存亡而不失其正者,其唯圣人乎!
>
> — 《乾卦·文言》

> 遁,亨,小利贞。
>
> — 《遁卦》

> 否之匪人,不利君子贞,大往小来。
>
> — 《否卦》

> 损,有孚,元吉,无咎,可贞,利有攸往。曷之用,二簋可用享。
>
> — 《损卦》

## I — 方法论骨架 (Interpretation)

易经进退观的核心:**"知进退存亡而不失其正"** 是圣人的标志。多数人只知进不知退,导致亢龙有悔。

### 进退的 4 个维度

| 维度 | "进"的含义 | "退"的含义 |
|---|---|---|
| **进退** | 进一步,加码,扩张 | 退一步,减仓,收缩 |
| **存亡** | 保持现状,延续 | 放弃,终结,转换 |
| **得失** | 获得更多收益 | 接受损失,落袋为安 |
| **取舍** | 抓住机会 | 放弃机会 |

**核心原则**: 这 4 维是联动的 — 只知"进"不知"退",必然"存亡"出问题,进而"得失"失衡。

### 退的 4 种类型

**1. 急流勇退 (主动退)**
- 性质: 在顶峰主动退让
- 时机: "飞龙在天"后期
- 心态: 主动选择,不是被迫
- 风险: 早退可惜,晚退必悔
- 案例: 范蠡助勾践灭吴后主动离开

**2. 功成身退 (完结退)**
- 性质: 任务完成后退场
- 时机: 目标达成后
- 心态: 完成使命,转向新使命
- 风险: 恋战导致失败
- 案例: 张良助刘邦后隐居

**3. 战略转移 (转换退)**
- 性质: 从一个领域转到另一个
- 时机: 旧领域已无空间
- 心态: 主动转型,非被动
- 风险: 转错方向
- 案例: 微软从 PC 转向云服务

**4. 壮士断腕 (被动退)**
- 性质: 在危机中被迫退
- 时机: 不退则亡
- 心态: 痛苦但必要
- 风险: 退得不彻底
- 案例: 越王勾践被吴击败后的退

### 进退的"勿"清单

#### 知进而不知退的表现

| 表现 | 后果 |
|---|---|
| 顶峰时继续加杠杆 | 亢龙有悔 |
| 事业巅峰不愿分享权力 | 团队离心 |
| 投资盈利不愿止盈 | 利润回吐 |
| 关系中控制欲过强 | 关系破裂 |
| 决策时不愿承认错误 | 错上加错 |

#### 知退不知进的表现

| 表现 | 后果 |
|---|---|
| 还没成功就退缩 | 功亏一篑 |
| 见好就收过度 | 错过更大机会 |
| 频繁切换方向 | 难以积累 |
| 关系中过度退让 | 被利用 |

### 进退判断的 5 问

```
1. 我现在处在什么阶段? (用 i-ching-life-cycle)
   - 潜龙/见龙/乾乾/或跃/飞龙/亢龙?

2. "退"和"进"哪个更紧迫?
   - 退的紧迫性高 → 退
   - 进的紧迫性高 → 进

3. 退的代价 vs 进的代价?
   - 退的代价 < 进的代价 → 退
   - 进的代价 < 退的代价 → 进

4. 退之后还有什么?
   - 退后有更好的选项 → 退
   - 退后无路可走 → 谨慎,可能是死路

5. 退得是否彻底?
   - 退要彻底,半退不退最危险
   - 进要果决,半进半退消耗最大
```

### 进退的时机信号

#### 该"进"的信号
- 行业/公司/项目 处于上升期
- 自身能力匹配新的挑战
- 资源/团队/时机 都准备好
- 机会窗口有限,必须抓住

#### 该"退"的信号
- 顶峰已持续较长时间,边际效益下降
- 团队/家人/股东 出现普遍疲劳
- 行业/政策 有不利变化的早期信号
- 内部矛盾开始激化
- 自己的身体/精神发出警告

#### 该"止"的信号 (既不进也不退)
- 行业/公司/项目 处于平台期
- 投入产出比稳定
- 没有明显机会也没有明显威胁
- 团队/家人 需要稳定期

## A1 — 书中的应用 (Past Application)

### 案例 1: 范蠡三散三聚 — 主动退的艺术

- **第一次退**: 助勾践灭吴后,范蠡主动离开越国
  - 文种不愿走,被杀
  - 范蠡预见"飞龙"之后的"亢"风险
- **第二次聚**: 范蠡到齐国经商,改名"鸱夷子皮",治产千万
- **第二次散**: 散尽家财给朋友,只带少量
- **第三次聚**: 到陶地 (今山东) 经商,改名"陶朱公",再致千金
- **第三次散**: 再散家财

**核心**: 范蠡三次"飞龙"都主动退,符合"知进退存亡"。

### 案例 2: 张良 — 功成身退的典范

- **问题**: 辅佐刘邦建立汉朝后,张良是开国功臣,位极人臣
- **退的智慧**:
  - 不争功: 让萧何/韩信领大头
  - 不恋权: 主动请辞丞相
  - 不结党: 退居幕后,专心修道
- **结果**: 善终,子孙繁衍 (张家 2000 年世家)
- **对比**: 韩信不知退,被杀;萧何半退,差点被杀

### 案例 3: 王莽 — 不知退的典型

- **问题**: 王莽以外戚身份位极人臣,代汉称帝
- **进的表现**:
  - 当了皇帝后仍激进改革
  - 权力极度集中
  - 听不进反对意见
- **不知退的代价**:
  - 改革失败
  - 全国起义
  - 自己被杀
- **教训**: "贵而无位,高而无民,贤人在下位而无辅" — 不知退必大悔

### 案例 4: 巴菲特 — 退的现代应用

- **第一次退**: 1969 年,巴菲特在合伙人基金顶峰时解散基金
  - 当时市场仍牛
  - 巴菲特认为已经找不到便宜股票
  - 解散后市场又涨了 2 年,但巴菲特躲过 1973-74 的大跌
- **第二次退**: 2020 年新冠时,巴菲特"抄底"航空股后又割肉
  - 主动承认错误 (知退)
  - 不恋战
- **核心**: 巴菲特说"在别人贪婪时恐惧,在别人恐惧时贪婪" — 知进退的具体化

## E — 执行 (Execution)

### 步骤 1: 阶段定位 (5 分钟)

```
我现在处于:
  [ ] 潜龙 (积累期)
  [ ] 见龙 (显露期)
  [ ] 乾乾 (奋斗期)
  [ ] 或跃 (抉择期)
  [ ] 飞龙 (鼎盛期)
  [ ] 亢龙 (衰退前期)
```

### 步骤 2: 进退紧迫性评估 (10 分钟)

```
1. "进"的机会成本:
   - 不进会失去什么? (机会)
   - 不进会保住什么? (稳定)

2. "退"的机会成本:
   - 退会失去什么? (地位/财富/关系)
   - 退会得到什么? (自由/健康/新机会)

3. "止"的代价:
   - 止 = 维持现状
   - 现状能否持续? (3-5 年视角)
```

### 步骤 3: 退的 4 问 (10 分钟)

```
1. 现在是"飞龙"还是"亢龙"?
   - 飞龙: 还可以进,不必急退
   - 亢龙: 必须退,否则大悔

2. 退得是否彻底?
   - 半退不退最危险 (人心散了,事也做不成)
   - 要退就退干净

3. 退后还有什么?
   - 退后有具体的新方向/新生活 → 退得值
   - 退后空虚无聊 → 退得苦

4. 退的时机是否合适?
   - 顶峰时退 (范蠡) > 衰退时退 (被动)
   - 主动退 > 被动退
```

### 步骤 4: 决策矩阵

| 阶段 | 优先 | 行动 |
|---|---|---|
| 潜龙 | 进 | 主动学习,抓住机会 |
| 见龙 | 进 | 适度显露,寻找大人 |
| 乾乾 | 进 | 日乾夕惕,警惕自满 |
| 或跃 | 评估 | 多方案评估,选择最佳 |
| 飞龙 | 守 | 巩固基础,服务他人 |
| 亢龙 | 退 | 主动退让,转换轨道 |

### 步骤 5: 执行与监测

```
退的执行:
  - 退要有时间表 (3-6 个月内)
  - 退要彻底 (权力/资源/关系 全部交接)
  - 退后要有新方向 (避免空虚)

进的执行:
  - 进要有节奏 (不要一口吃成胖子)
  - 进要有止损 (知道何时停)
  - 进要持续 (不能三天打鱼两天晒网)

监测信号:
  - 退后是否真的 "无悔"
  - 进后是否真的 "亢"
  - 每季度复盘一次
```

## B — 边界 (Boundary)

### 不适用场景

- **完全没进退需求**: 日常稳定期 — 不必套用本 skill
- **没有退路的死战**: 用 i-ching-crisis-transformation
- **刚入门还没开始进**: 用 i-ching-life-cycle 定位阶段

### 局限

- **"退"的判断主观**: 何时是"亢龙"难以精确判断
- **"退"的机会成本难以量化**: 退的隐性损失难估
- **文化适配**: 西方文化更鼓励"坚持",易经的"退"需要翻译
- **个人心理**: 多数人心理上不愿退 (沉没成本/不甘心)

### 失效条件

- **被动退 (被迫退场)**: 已不是"退"而是"败"
- **战略退却 (战术性)**: 短退是为了长进,不算本 skill 范围
- **紧急避险**: 用 i-ching-crisis-transformation

### 与 stock-* skill 的协同

| 场景 | 推荐 skill 组合 |
|---|---|
| 投资止盈 | i-ching-advance-retreat + stock-profit-taking-decision + stock-trailing-stop |
| 投资止损 | i-ching-advance-retreat + stock-stop-loss-decision |
| 职业顶峰 | i-ching-advance-retreat + i-ching-life-cycle (阶段判断) |
| 创业退出 | i-ching-advance-retreat + i-ching-revolution-timing |

**互补关系**: 
- stock-stop-loss-decision 解决"具体怎么止损" (操作)
- i-ching-advance-retreat 解决"该不该退的时机" (战略)
- 协同使用: 先用 i-ching-advance-retreat 判断时机,再用 stock-stop-loss-decision 执行
