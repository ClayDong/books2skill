# 时位中正决策 (Time-Position-Balance Decision)

## R — 原文 (Reading)

> 二与四同功而异位,其善不同。二多誉,四多惧,近也。柔之为道,不利远者,其要无咎,其用柔中也。
>
> — 《系辞下传》

> 君子居则观其象而玩其辞,动则观其变而玩其占。
>
> — 《系辞上传》

> 初辞拟之,卒成之终。若夫杂物撰德,辩是与非,则非其中爻不备。
>
> — 《系辞下传》

> 君子黄中通理,正位居体,美在其中,而畅于四支,发于事业,美之至也。
>

## I — 方法论骨架 (Interpretation)

易经决策哲学的**四维自检框架**。任何重大决策前,逐一回答以下四个问题,只有四个答案都"通"才能执行。

### 四维定义

**1. 时 (Time) — 是不是合适时机?**
- 这件事的发生时刻是否正确?
- 现在做,还是等等再做?
- 时机不对,任何努力都白费
- "时乘六龙以御天" — 把握时机才能驾驭变化

**2. 位 (Position) — 你处在什么位置/角色?**
- 你在这件事中是什么角色 (决策者/执行者/旁观者)?
- 这个角色应该做什么?不应该做什么?
- 不同位置做同样的事,结果完全不同 ("二多誉,四多惧,近也")
- "二"在下卦之中多赞誉,"四"近上卦多忧惧 — 位置决定风险

**3. 中 (Balance) — 决策是否不偏不倚?**
- 决策是否走极端 (全押/全空)?
- 是否考虑了所有相关方?
- 是否有折中方案?
- "黄中通理" — 中不是平庸,而是通达事理

**4. 正 (Rightness) — 决策是否合乎正道?**
- 决策是否符合法律/伦理/长期价值观?
- 即便短期能获利,长期是否站得住脚?
- "利贞"中的"贞"就是"正" — 正道是基础
- "德薄位尊,鲜不及矣" — 不正之位迟早崩塌

### 四维关系

```
时 ──→ 是基础 (无时则无法)
位 ──→ 是前提 (无位则无权)
中 ──→ 是方法 (无中则失衡)
正 ──→ 是底线 (无正则必败)
```

### 三层决策法 (观象→玩辞→占变)

**第一层:观象 (Observe Signs)**
- 决策前先观察"象" — 外在表现、数据、现象
- 不观象的决策是"瞎子摸象"

**第二层:玩辞 (Study Principles)**
- 决策中要"玩" (反复研究) 内在规律
- 卦辞/爻辞是经验总结,要"玩味"而非死记
- 不玩辞的决策是"鹦鹉学舌"

**第三层:占变 (Divine Changes)**
- 决策终要"占" — 在不确定中给出判断
- "占"不是"算命",而是"承担"
- 不占变的决策是"纸上谈兵"

### 与炒股的智慧的核心差异

| 维度 | 炒股的智慧 | 时位中正 |
|---|---|---|
| 时间观 | 技术周期 (4 阶段) | 天时/时机 (哲学层) |
| 空间观 | 临界点/位置 | 角色/位置 (社会层) |
| 平衡 | 概率胜算 | 中道 (伦理层) |
| 正当 | 行为纪律 | 正道/合规 (法理层) |

**互补关系**:炒股的智慧解决"具体怎么做",时位中正解决"为什么这么做"+"什么时候做"。

## A1 — 书中的应用 (Past Application)

### 案例 1: 文王被囚演周易 — "时" 的智慧

- **问题**: 商纣王囚禁文王于羑里,七年不释,生死难料
- **方法论的使用**: 文王在囚禁中,接受"时" (商朝气数未尽),通过"位" (臣位) 坚守臣道,在"时"和"位"允许的范围内,通过"观象玩辞"创造性地把八卦重为六十四卦
- **结论**: 七年囚禁中完成周易经部,既保全自己,又为周朝建立奠定思想基础
- **结果**: 后获释,辅武王灭商,周易成为儒家六经之首

### 案例 2: 诸葛亮借东风 — "时" 的精准判断

- **问题**: 208年赤壁之战,孙刘联军需借东风才能火攻
- **方法论的使用**: 诸葛亮"观天象" — 观察长江冬季气候规律,知道隆冬时节偶尔有暖流导致短期转南风。这是"观象"+"玩辞" (研究气象规律)
- **结论**: 在"时" (转风) 精准出现时,主动"占变" (承担) 借东风
- **结果**: 火攻成功,赤壁之战奠定三国格局

### 案例 3: 越王勾践 — 失时失位后的"位"重建

- **问题**: 越被吴灭,勾践为奴三年。归国后国贫兵弱
- **方法论的使用**: 勾践在"位"上重建 (从霸主变为奴),用"潜龙勿用"接受"时"的不利;同时用"时行则行,时止则止"把握吴王夫差北上争霸的"几"
- **结论**: 22 年后"飞龙在天",灭吴称霸
- **结果**: 春秋最后一位霸主,范蠡功成身退 — 符合"知进退存亡"

## A2 — 触发场景 (Trigger Scenarios)

### 场景 1: 投资决策
- 决策前自检:
  - **时**: 当前是市场的哪一阶段? (用 stock-market-timing 辅助)
  - **位**: 我的角色是? (主力/散户/机构)
  - **中**: 仓位是否合理?是否过度集中?
  - **正**: 投资标的是否合法合规?是否违背价值观?
- 输出:四维自检表 + GO/NO-GO 决策

### 场景 2: 职业转型
- **时**: 当前经济环境如何? 行业趋势如何?
- **位**: 我在当前公司的角色? 转型后的角色?
- **中**: 转型的代价/收益是否平衡? 风险是否可承受?
- **正**: 转型是否符合我的长期价值观?

### 场景 3: 创业决策
- **时**: 市场窗口是否成熟? 是否有红利期?
- **位**: 我的能力是否匹配创业? 团队是否到位?
- **中**: 资源/野心/能力是否平衡?
- **正**: 这个创业是否解决真问题? 是否符合伦理?

### 场景 4: 重大关系决策
- **时**: 现在是关系变化的好时机吗?
- **位**: 我在这段关系中是什么角色?
- **中**: 各方利益/情感是否平衡?
- **正**: 这个决策是否符合双方长期福祉?

### 场景 5: 道德困境
- **时**: 时机紧迫是否影响判断? 是否被"急"牵着走?
- **位**: 我在事件中的利益相关度?
- **中**: 短期利益/长期利益的平衡?
- **正**: 决策是否经得起公开检验?

## E — 执行 (Execution)

### 步骤 1: 准备决策素材 (10 分钟)
- [ ] 列出决策涉及的所有事实 (观象)
- [ ] 收集所有相关方的观点
- [ ] 写下决策的 3 个备选方案
- [ ] 写下"不行动"的代价 (重要: 多数人忽略)

### 步骤 2: 时位中正四问 (15 分钟)

```
时问: 这是不是合适的时机? 
  - 时机过早/过晚的标志是什么?
  - 如果等 3 个月,情况会怎么变?
  评分: [1-10]

位问: 我在这个位置应该做什么? 
  - 同样的事,不同位置的人做结果一样吗?
  - 我是否有权做这个决定?
  评分: [1-10]

中问: 决策是否不偏不倚? 
  - 这是不是最极端的方案?
  - 是否考虑了所有相关方?
  - 折中方案的成本/收益比如何?
  评分: [1-10]

正问: 决策是否合乎正道? 
  - 这个决策能否告诉家人/朋友?
  - 5 年后回看,会不会后悔?
  - 是否符合长期价值观?
  评分: [1-10]
```

### 步骤 3: 总分评估
- 总分 = 时分 + 位分 + 中分 + 正分 (满分 40)
- **>32**: 强 GO,放心执行
- **24-32**: 谨慎 GO,做好风险预案
- **16-24**: 暂缓,3 个月后再评估
- **<16**: NO-GO,放弃或重新设计

### 步骤 4: 占变 — 主动承担
- 即便四维都通过,仍要"占变" — 主动承担后果
- 写下"如果失败,我能接受的最坏结果是什么"
- 准备 Plan B (应对失败)
- 决策后,不要反复纠结,坚持执行

### 步骤 5: 复盘 (决策后 1 周)
- 决策实际结果 vs 预期
- 四维自评: 哪一维最准确/最不准
- 学习: 哪一维需要加强 (例如"时"判断不准 → 加强对外部环境的观察)

## B — 边界 (Boundary)

### 不适用场景

- **快速日常决策**: 吃什么/买什么 — 用快速判断即可
- **紧急生死时刻**: 来不及四问 — 凭本能
- **明确无风险的小事**: 散步/喝水
- **已深思熟虑的常规决策**: 季度复盘等

### 局限

- **本 skill 是质性框架,不提供量化预测**:
  - "时"无法精确到"今天下午 3 点"这样的精度
  - "位"无法精确到"50.2% 概率"这样的量化
- **不替代专业判断**:
  - 投资决策仍需财务/行业分析
  - 医疗决策仍需医生诊断
  - 法律决策仍需律师意见
- **主观性强**:
  - 四维评分有主观性
  - 不同人对"中"和"正"的理解可能不同
  - 建议与可信的他人讨论 (易经的"拟议")

### 失效条件

- **数据严重不足**: 观象无法做时,四问失基础
- **立场严重偏颇**: 决策者已被情绪/利益绑架,四问失意义
- **时机严重不对**: 在"否"的环境中强行作为,四问不能补救

### 与现代决策科学的整合

| 易经维度 | 现代决策工具 | 整合方式 |
|---|---|---|
| 时 | OKR / 时机窗口分析 | 用"时"判断是否启动 OKR |
| 位 | 角色 / RACI 矩阵 | 用"位"明确决策权 |
| 中 | 平衡计分卡 / 利益相关者分析 | 用"中"平衡多维 |
| 正 | 伦理审查 / 合规检查 | 用"正"做底线 |
