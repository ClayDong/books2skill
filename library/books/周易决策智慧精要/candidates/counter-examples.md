# 周易 — 候选抽取: 反例 (counter-examples)

> 易经使用不当的案例: 失时/失位/失中/失正/失几

---

## ce01 — 不观象而占

```yaml
id: ce01
title: 不观象而占 — 形式化使用易经
type: counter_example
problem: |
  一些使用者把易经当作"算命工具",不观察现实情况,
  直接摇卦或起卦,然后机械地解读卦辞爻辞。
why_wrong: |
  - 违背"君子居则观其象而玩其辞"原则
  - 忽略"象"是"辞"的基础
  - 决策脱离实际
example: |
  有人投资只看"卦象吉凶",不研究公司基本面、行业趋势、财务数据,
  结果完全凭"卦"决策,损失惨重。
lesson: |
  易经是"思考框架"而非"预测工具"。使用易经时:
  1. 先观察现实 (观象)
  2. 再研究规律 (玩辞)
  3. 最后在不确定中决策 (占变)
```

## ce02 — 不知时位而妄动

```yaml
id: ce02
title: 不知时位而妄动 — 潜龙阶段强行冒头
type: counter_example
problem: |
  在"潜龙勿用"阶段强行行动,急于显露。
why_wrong: |
  - 违背"潜龙勿用,阳气潜藏"原则
  - 积累不够,时机未到
  - 强出头必受挫
example: |
  刚毕业 1 年的员工,专业能力不足,却急于表现,
  强行申请重大项目负责人,结果搞砸项目,损害职业声誉。
lesson: |
  任何阶段都有其时位:
  - 潜龙: 积累期,不强行出头
  - 见龙: 小试期,可以试探
  - 飞龙: 大展期,全力投入
  - 亢龙: 退守期,主动退让
```

## ce03 — 知进而不知退

```yaml
id: ce03
title: 亢龙阶段不知退 — 顶峰时继续扩张
type: counter_example
problem: |
  处在"飞龙在天"阶段,本应巩固基础,但继续激进扩张。
why_wrong: |
  - 违背"亢龙有悔,贵而无位,高而无民"原则
  - 顶峰时没有群众基础,继续扩张必败
  - 投资中的"加仓最重仓位" / 职业中的"超负荷"
example: |
  企业顶峰时盲目多元化,资源分散,
  主营业务被拖累,新业务未站稳,最终全线崩溃。
lesson: |
  "知进退存亡而不失其正者,其唯圣人乎" — 顶峰时要主动损
```

## ce04 — 失中 — 决策偏激

```yaml
id: ce04
title: 失中 — 决策偏激
type: counter_example
problem: |
  决策时走极端,要么全押要么全弃,没有中间方案。
why_wrong: |
  - 违背"中"的原则 — 不偏不倚
  - 失去调节空间
  - 风险/收益不匹配
example: |
  投资中"全仓一只股票"或"完全空仓观望",
  都是失中的表现。应该按"时位"分批建仓/分批减仓。
lesson: |
  "中"是动态的 — 风险/收益/时机的平衡。
  决策时自问:
  1. 这是不是最极端的方案?
  2. 有没有折中方案?
  3. 折中方案的成本/收益比?
```

## ce05 — 失正 — 不正当手段

```yaml
id: ce05
title: 失正 — 用不正当手段达成目标
type: counter_example
problem: |
  即使目标正确,使用欺骗、违规、内幕等手段。
why_wrong: |
  - 违背"贞" (正) 原则
  - 短期可能获利,长期必受损
  - 损害"位"的基础
example: |
  内幕交易虽然可能短期获利,但一旦曝光,
  不仅是法律后果,职业信誉也彻底毁掉。
lesson: |
  "利贞"中的"贞"是"正" — 决策必须合乎正道。
  这是"位"的根本 — 没有"正",所有"位"都是空中楼阁。
```

## ce06 — 不知几而失机

```yaml
id: ce06
title: 不知几而失机 — 错过早期信号
type: counter_example
problem: |
  错过了"几微"信号,等事情完全明朗才行动,已经太晚。
why_wrong: |
  - 违背"君子见几而作,不俟终日"原则
  - 信息充分时,机会也消失
  - 决策时机的"几"已经过去
example: |
  错过行业风口:
  - 2010 年移动互联网苗头已现 (几)
  - 2014 年移动应用爆发 (明朗)
  - 2018 年红利消退 (消失)
  - 2010 年布局的公司赚得盆满钵满
  - 2014 年跟进的也有肉吃
  - 2018 年才进入的红海搏杀
lesson: |
  优秀的决策者在"几微"时就开始行动。
  这需要:
  1. 持续学习 — 保持对趋势的敏感
  2. 小步试错 — 用小成本测试"几"是否真的到来
  3. 果断决策 — 一旦"几"出现,不拖泥带水
```

## ce07 — 不知恒而频繁变

```yaml
id: ce07
title: 不知恒而频繁变 — 缺乏坚持
type: counter_example
problem: |
  决策频繁变更,没有坚持和耐心。
why_wrong: |
  - 违背"恒,德之固也"原则
  - 没有积累效应
  - 难以形成核心能力
example: |
  创业 1 年换 3 个方向,每个方向都没做深。
  投资 3 个月换 5 只股票,没有长期持有。
lesson: |
  "恒"和"变"不矛盾:
  - 大方向要恒 (核心战略)
  - 小执行要变 (具体战术)
  - 在恒的基础上灵活调整
```

## ce08 — 不知损而过度积累

```yaml
id: ce08
title: 不知损而过度积累 — 不懂减法
type: counter_example
problem: |
  过度积累 (财富/库存/承诺/关系),不懂"损"。
why_wrong: |
  - 违背"损,德之修也"原则
  - 积累超过承载能力,反而成负担
  - 错过"损"带来的"益"机会
example: |
  持有太多股票,每只都不愿卖,最终套牢。
  同时做太多项目,每个都做不好。
lesson: |
  "损"是"益"的开始:
  - 减掉不重要的,才能聚焦重要的
  - 减掉不适应的,才能迎接新机会
  - "先难后易" — 短期痛苦换来长期轻松
```
