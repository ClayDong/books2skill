# 周易 — 候选抽取: 术语表 (glossary)

> 易经核心术语,用于跨书知识图谱

---

## g01 — 时 (shí)

```yaml
id: g01
term: 时
pinyin: shí
source: 彖传/文言
definition: |
  决策的时机/天时。乾卦"时乘六龙以御天" — 把握时机才能驾驭变化。
  "时止则止,时行则行" — 该动则动,该停则停。
related_concepts: [位, 中, 正, 几]
modern_equivalent: Timing / Window of Opportunity
```

## g02 — 位 (wèi)

```yaml
id: g02
term: 位
pinyin: wèi
source: 爻辞通例
definition: |
  决策者所处的位置/角色。初、三、五为阳位,二、四、上为阴位。
  阳爻居阳位、阴爻居阴位为"正",反之为"不正"。
related_concepts: [时, 中, 正]
modern_equivalent: Role / Stakeholder Position
```

## g03 — 中 (zhōng)

```yaml
id: g03
term: 中
pinyin: zhōng
source: 爻辞通例
definition: |
  决策的不偏不倚。二爻和五爻为"中"位(下卦和上卦之中)。
  居中多吉,失中多凶。
related_concepts: [时, 位, 正, 和]
modern_equivalent: Balance / Neutrality
```

## g04 — 正 (zhèng)

```yaml
id: g04
term: 正
pinyin: zhèng
source: 乾卦"元亨利贞" / 各卦"利贞"
definition: |
  决策的正当性。"贞"不仅是"正",还包括"坚定"。
  决策要合乎正道,且坚持不懈。
related_concepts: [时, 位, 中]
modern_equivalent: Legitimacy / Right Path
```

## g05 — 几 (jī)

```yaml
id: g05
term: 几
pinyin: jī
source: 系辞下
definition: |
  决策的"几微" — 事物初显的微妙征兆。
  "几者,动之微,吉之先见者也"
  识别"几"是决策的最高境界。
related_concepts: [时, 位, 中, 变]
modern_equivalent: Signal / Early Indicator
```

## g06 — 应 (yìng)

```yaml
id: g06
term: 应
pinyin: yìng
source: 爻辞通例
definition: |
  爻位之间的对应关系。初与四、二与五、三与上为"应位"。
  对应位上的阴爻阳爻互应(异性)为吉,同性不应为凶。
related_concepts: [位, 中, 承]
modern_equivalent: Stakeholder Alignment
```

## g07 — 承 (chéng)

```yaml
id: g07
term: 承
pinyin: chéng
source: 爻辞通例
definition: |
  阴爻对阳爻的承托关系。阴爻在阳爻之下为"承"为吉,
  阴爻在阳爻之上为"乘"为凶(阳承阴)。
related_concepts: [位, 应, 上下]
modern_equivalent: Support / Subordination
```

## g08 — 变 (biàn)

```yaml
id: g08
term: 变
pinyin: biàn
source: 系辞
definition: |
  事物的变化。"穷则变,变则通,通则久" — 变是常态,不变是例外。
  决策要顺势而变,但变要"当" (适时适当)。
related_concepts: [时, 通, 几]
modern_equivalent: Change / Adaptability
```

## g09 — 通 (tōng)

```yaml
id: g09
term: 通
pinyin: tōng
source: 系辞
definition: |
  事物的通达。变则通,通则久。"泰"是天地相交万物通,
  "否"是天地不交万物不通。
related_concepts: [变, 泰, 否]
modern_equivalent: Flow / Throughput
```

## g10 — 悔 (huǐ)

```yaml
id: g10
term: 悔
pinyin: huǐ
source: 系辞/各卦爻辞
definition: |
  决策失误后的后悔。悔可补救,吝难补救,咎尚可救,厉则危,凶则败。
  "震无咎者存乎悔" — 懂得后悔才能无咎。
related_concepts: [吝, 咎, 厉, 凶]
modern_equivalent: Regret / Course Correction
```

## g11 — 亢 (kàng)

```yaml
id: g11
term: 亢
pinyin: kàng
source: 乾卦上九
definition: |
  过度、顶点。亢龙有悔 — 到达顶点后不知退让必有悔。
  "贵而无位,高而无民" — 顶点时失去了支撑。
related_concepts: [悔, 进退, 否]
modern_equivalent: Hubris / Overreach
```

## g12 — 潜 (qián)

```yaml
id: g12
term: 潜
pinyin: qián
source: 乾卦初九
definition: |
  潜伏、隐藏。潜龙勿用 — 在时机未到时,不要硬出头。
  "不易乎世,不成乎名,遁世无闷" — 隐居而不郁闷。
related_concepts: [时, 亢, 见龙]
modern_equivalent: Stealth / Latency
```
