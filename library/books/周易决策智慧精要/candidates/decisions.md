## d01 — 职业转型决策

```yaml
id: d01
title: 职业转型决策
type: decision
related_hexagrams: [革, 鼎, 乾, 坤]
trigger_keywords:
  - "该不该换工作"
  - "要不要转行"
  - "我想转行"
  - "现在的职业方向对吗"
  - "该不该裸辞"
decision_framework: |
  革卦 (变革) + 鼎卦 (鼎新) + 乾卦 (生命周期) + 坤卦 (顺承)
  1. 革的时机: 当前事业是否真的需要革?
  2. 鼎的新方向: 革之后的"鼎"是什么?
  3. 乾的阶段: 处在职业生命周期的哪一阶段?
  4. 坤的顺承: 转型是否顺应大势?
```
