## c01 — 文王演易,羑里之困

```yaml
id: c01
title: 文王被囚羑里演周易
type: case
period: 商末周初
problem: |
  周文王被商纣王囚禁在羑里 (今河南汤阴),
  面临生死困境。其子伯邑考被纣王杀害,
  文王被囚七年,需在绝境中保全自己并为周国谋划未来。
method: |
  - 在囚禁中研究易理,将八卦重为六十四卦
  - 通过"时" (待商朝气数尽) 调整决策节奏
  - 通过"位" (臣位) 坚守"臣道"而不逾越
  - 用"潜龙勿用"精神保存实力
result: |
  - 七年囚禁中完成周易经部
  - 后获释,辅佐武王灭商
  - 周朝建立后被追封为"文王"
applicability: 人在绝境中如何用"时位中正"原则保全自己并图长远
```

## c02 — 孔子韦编三绝,作十翼

```yaml
id: c02
title: 孔子晚年读易,作十翼
type: case
period: 春秋
problem: |
  孔子周游列国十四年,理想未能实现,回到鲁国后
  已68岁,深感"不复梦见周公"。此时需要找到
  新的精神支柱和传承文化的载体。
method: |
  - "加我数年,五十以学易,可以无大过矣" — 把学易作为晚年的功课
  - 韦编三绝 — 反复研读,把编竹简的熟牛皮绳磨断多次
  - 撰写系辞/说卦/序卦/杂卦/文言/象/彖 等十翼
  - 通过"系辞焉以尽其言" — 用文字把易的智慧传给后人
result: |
  - 完成《易传》(十翼),奠定易学哲学基础
  - 易经从"占卜之书"升华为"哲学之书"
  - 影响中国文化 2500 年
applicability: 在人生晚期通过系统总结创造永恒价值
```

## c03 — 诸葛亮借东风,观天时

```yaml
id: c03
title: 赤壁之战,诸葛亮借东风
type: case
period: 三国
problem: |
  208 年,孙刘联军在赤壁对曹操 80 万大军,
  长江北风不利于火攻。需要改变天时。
method: |
  - 诸葛亮"观天时" — 观察长江冬季的气候规律
  - 知道隆冬时节偶尔会有"暖流"导致短期转南风
  - 通过"几" (早期信号) 把握转风的精确时间
  - "借"东风 — 把自然规律为我所用
result: |
  - 火攻成功,曹操大败
  - 三国鼎立格局形成
applicability: 在必败的局面下,通过精准的"时"判断翻盘
modern_translation: 数据分析+时机判断 改变战局
```

## c06 — 王莽篡汉,失时失位

```yaml
id: c06
title: 王莽改革失败
type: case
period: 西汉末
problem: |
  王莽以外戚身份代汉称帝,推行激进改革,
  面对的是已经腐烂的西汉社会。
method: 反面教材:
  - 失时: 改革时机不对,民众对汉朝还有感情
  - 失位: 没有真正的民意基础,靠外戚权势
  - 失中: 改革措施过于激进,不符合"中道"
  - 失几: 没识别出民众的实际需求
result: |
  - 改革失败,引发绿林赤眉起义
  - 王莽被杀,新朝 15 年而亡
applicability: 变革需要"时位中正"四要素俱全,缺一不可
```

## c11 — 个人案例:职业转型时机的判断

```yaml
id: c11
title: 软件工程师到产品经理的转型
type: case
type: personal_decision
problem: |
  一位 30 岁的工程师,工作 7 年,技术能力成熟,
  但感觉职业天花板。考虑是否转产品经理。
method: |
  - "时": 当前是 AI 时代,产品经理比工程师更稀缺
  - "位": 工程师的"位"已到顶 (高级工程师)
  - "中**: 是否过度乐观?评估转型成本
  - "正**: 是否逃避工程师的真正问题 (技术管理岗?)
  - "几**: 测试早期信号 — 业余时间做产品原型
  - "潜龙勿用**: 不裸辞,先副业试水
result: |
  - 副业 1 年后确认兴趣和能力匹配
  - 内部转岗成功,无风险完成转型
applicability: 重大职业决策的"时位中正几"五维评估
```
