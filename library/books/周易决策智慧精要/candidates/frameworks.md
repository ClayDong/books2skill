## f01 — 乾卦六爻生命周期模型 (Life-Cycle Decision Model)

```yaml
id: f01
title: 乾卦六爻生命周期模型
type: framework
source: 乾卦卦辞+爻辞+文言传
description: |
  人生/事业六阶段决策模型。每阶段对应不同行动:
  初九 潜龙勿用 → 潜伏期 (学习/积累)
  九二 见龙在田 → 显露期 (小试牛刀)
  九三 终日乾乾 → 奋斗期 (谨慎精进)
  九四 或跃在渊 → 抉择期 (进退选择)
  九五 飞龙在天 → 鼎盛期 (大展宏图)
  上九 亢龙有悔 → 衰退期 (知退)
core_principle: "知进退存亡而不失其正者,其唯圣人乎"
applicability: 创业/职业/投资/关系 各领域
```

## f02 — 时位中正决策矩阵 (Time-Position-Balance Decision Matrix)

```yaml
id: f02
title: 时位中正决策矩阵
type: framework
source: 系辞+彖传+爻辞
description: |
  决策四维评估框架:
  1. 时: 是不是合适的时机? (天时)
  2. 位: 你处在什么位置/角色? (角色)
  3. 中: 决策是否不偏不倚? (平衡)
  4. 正: 决策是否合乎正道? (正当性)
core_principle: "二与四同功而异位,三与五同功而异位" — 不同位置有不同风险
applicability: 任何重大决策前的"四问"自检
```

## f03 — 序卦因果链模型 (Causal Chain of Life Events)

```yaml
id: f03
title: 序卦因果链模型
type: framework
source: 序卦传
description: |
  64 卦构成完整因果链,前一卦是后一卦的因:
  天地 → 万物 → 男女 → 夫妇 → 父子 → 君臣 → 上下 → 礼义
  师 → 比 → 小畜 → 履 → 泰 → 否 → 同人 → 大有
  核心: "有 A 必 B,B 必 C" — 任何结果都有前因
core_principle: "物不可以终通,故受之以否" — 物极必反
applicability: 长周期规划/产业链分析/职业发展路径
```

## f04 — 系辞"观象玩辞占变"三阶决策法 (Three-Stage Decision Method)

```yaml
id: f04
title: 观象玩辞占变三阶决策法
type: framework
source: 系辞上传
description: |
  决策三阶段:
  1. 观象: 观察外在表现 (数据/现象)
  2. 玩辞: 反复研究内在规律 (推演/分析)
  3. 占变: 在不确定中做判断 (决策/承担)
core_principle: "君子居则观其象而玩其辞,动则观其变而玩其占"
applicability: 任何复杂决策的标准流程
```

## f06 — 坎困危机转化模型 (Crisis Transformation Model)

```yaml
id: f06
title: 坎困危机转化模型
type: framework
source: 坎卦+困卦+系辞
description: |
  危机的两层转化:
  1. 坎(习坎): 重险之下,维心亨,行有尚 — 内心通透方能脱险
  2. 困(亨贞): 困而不动,大人吉 — 困境中坚守正道方能脱困
  转化关键:
  - 维心亨: 保持内心清醒
  - 不动: 不在困境中乱动
  - 行有尚: 行动要有目标
core_principle: "穷则变,变则通,通则久" — 困是变的契机
applicability: 危机管理/逆境决策/转型期
```

## f08 — 几微信号识别模型 (Early Signal Recognition)

```yaml
id: f08
title: 几微信号识别模型
type: framework
source: 系辞下+文言
description: |
  "几"是决策的最高境界:

```

## f09 — 二五爻中正模型 (Middle-Balance Decision Model)

```yaml
id: f09
title: 二五爻中正模型
type: framework
source: 爻辞通例
description: |
  六爻中,二爻和五爻为"中"位:
  - 二在下卦之中 (基层中坚)
  - 五在上卦之中 (领导核心)
  - 居中多吉,失中多凶
  决策含义:
  - 重要角色 (二/五) 的决策要"中" — 不偏不倚
  - 失中 (偏离中位) 必有悔
core_principle: "黄中通理,正位居体" — 黄是中之色,通理是中之用
applicability: 关键岗位决策/团队中层管理
```

## f10 — 损益盛衰循环模型 (Gain-Loss Cycle Model)

```yaml
id: f10
title: 损益盛衰循环模型
type: framework
source: 损卦+益卦+杂卦传
description: |
  损益是盛衰的起点:
  - 损: 减损过剩,适度回退
  - 益: 增益不足,适度前进
  - 循环: 损 → 益 → 损 (周而复始)
  决策含义:
  - 顶峰时要主动损 (减仓/退让/精简)
  - 谷底时要主动益 (加仓/学习/积累)
  - 损是"先难后易" (短痛长甜)
  - 益是"长裕不设" (持续收益)
core_principle: "损益,盛衰之始也" — 损是衰的起点,益是盛的起点
applicability: 投资周期/职业循环/健康管理
```
