# 周易 — 候选抽取: 原则 (principles)

> 易经原则提取: 来自系辞/文言/九卦君子之道/六十四卦卦辞

---

## p01 — 几者,见几而作,不俟终日

```yaml
id: p01
title: 见几而作,不俟终日
type: principle
source: 系辞下+文言
quote: |
  "几者,动之微,吉之先见者也。君子见几而作,不俟终日。
   易曰:'介于石,不终日,贞吉。'"
decisional_meaning: 在事情刚刚显出苗头时就行动,不要等到完全明朗
modern_translation: 看到早期信号就决策,不要等到所有信息齐备
```

## p02 — 时行则行,时止则止

```yaml
id: p02
title: 时行则行,时止则止
type: principle
source: 艮卦+文言
quote: |
  "时止则止,时行则行,动静不失其时,其道光明"
decisional_meaning: 该动就动,该停就停,关键是把握时机
modern_translation: 行动与停止都要符合时机,顺时则吉
```

## p03 — 亢龙有悔,知进退存亡

```yaml
id: p03
title: 知进退存亡而不失其正
type: principle
source: 乾卦文言
quote: |
  "亢之为言也,知进而不知退,知存而不知亡,知得而不知丧。
   其唯圣人乎!知进退存亡而不失其正者,其唯圣人乎!"
decisional_meaning: 知道进退、存亡、得失,在顶峰主动退让才是圣人
modern_translation: 顶峰时要主动退让,见好就收
```

## p04 — 潜龙勿用,待时而动

```yaml
id: p04
title: 潜龙勿用,待时而动
type: principle
source: 乾卦初九+文言
quote: |
  "初九曰'潜龙勿用',何谓也?子曰:'龙德而隐者也。
   不易乎世,不成乎名,遁世无闷,不见是而无闷。
   乐则行之,忧则违之,确乎其不可拔,潜龙也。'"
decisional_meaning: 在时机不成熟时,坚守内心不妄动
modern_translation: 时机未到时不要硬上,等机会来了再动
```

## p05 — 履信思顺,尚贤受益

```yaml
id: p05
title: 履信思顺,尚贤受益
type: principle
source: 系辞上
quote: |
  "佑者,助也。天之所助者,顺也。人之所助者,信也。
   履信思乎顺,又以尚贤也。是以自天佑之,吉无不利也。"
decisional_meaning: 讲信用、顺天时、尊贤人,自天保佑
modern_translation: 决策要守信用、顺客观规律、尊重专业人士
```

## p06 — 谦谦君子,德之光也

```yaml
id: p06
title: 谦德受益,满招损
type: principle
source: 谦卦+杂卦
quote: |
  "谦,亨,君子有终。" / "谦轻而豫怠也"
decisional_meaning: 谦卦是唯一六爻皆吉的卦
modern_translation: 保持谦让是最安全的长期策略
```

## p07 — 亢龙有悔,贵而无位

```yaml
id: p07
title: 亢龙有悔,过犹不及
type: principle
source: 乾卦上九
quote: |
  "贵而无位,高而无民,贤人在下位而无辅,是以动而有悔也。"
decisional_meaning: 顶峰时没有群众和辅助,动则有悔
modern_translation: 成功后要巩固基础,不要孤军深入
```

## p08 — 革而当,其悔乃亡

```yaml
id: p08
title: 革而当,其悔乃亡
type: principle
source: 革卦彖传
quote: |
  "革,水火相息,二女同居,其志不相得,曰革。巳日乃孚,
   革而信之。文明以说,大亨以正。革而当,其悔乃亡。"
decisional_meaning: 变革要"当" (时机对、方式对),悔恨才能消失
modern_translation: 变革要选对时机和方式,否则后患无穷
```

## p09 — 穷则变,变则通,通则久

```yaml
id: p09
title: 穷则变,变则通,通则久
type: principle
source: 系辞下
quote: |
  "易穷则变,变则通,通则久。是以自天佑之,吉无不利。"
decisional_meaning: 走到尽头就要变化,变化才能通达,通达才能持久
modern_translation: 困境中要主动求变,死守现状是死路
```

## p10 — 见几而作,君子之道

```yaml
id: p10
title: 见几而作
type: principle
source: 系辞下
quote: |
  "君子见几而作,不俟终日。易曰:'介于石,不终日,贞吉。'

decisional_meaning: 识别"几微"就行动,这是君子的决策特征
modern_translation: 优秀的决策者在信号微弱时就行动
```

## p11 — 二多誉,四多惧,五多功

```yaml
id: p11
title: 位不同则誉惧功异
type: principle
source: 系辞下
quote: |
  "二与四同功而异位,其善不同。二多誉,四多惧,近也。
   三与五同功而异位,三多凶,五多功,贵贱之等也。"
decisional_meaning: 同样的行动在不同位置有不同结果
modern_translation: 决策要考虑你所处的位置,不同位置风险收益不同
```

## p12 — 君子安身,易心,定交

```yaml
id: p12
title: 安身/易心/定交三步走
type: principle
source: 系辞下
quote: |
  "君子安其身而后动,易其心而后语,定其交而后求。
   君子修此三者,故全也。"
decisional_meaning: 决策前先稳定自身 (安身)、平和心态 (易心)、确定关系 (定交)
modern_translation: 决策三准备:自身稳定、情绪平和、关系清楚
```

## p13 — 小惩而大诫,小人之福

```yaml
id: p13
title: 小惩大诫,小人之福
type: principle
source: 系辞下
quote: |
  "小人不耻不仁,不畏不义,不见利不劝,不威不惩。
   小惩而大诫,此小人之福也。易曰:'履校灭趾,无咎。'此之谓也。"
decisional_meaning: 小的惩罚避免大错,这是福气
modern_translation: 早期小损失是福气,避免大损失
```

## p14 — 履虎尾不咥人,谦和而行

```yaml
id: p14
title: 履险而和,行权不危
type: principle
source: 履卦+系辞下
quote: |
  "履以和行,谦以制礼" / "履虎尾,不咥人,亨"
decisional_meaning: 即便走在危险境地,只要保持和气谦逊,也能亨通
modern_translation: 在风险中保持冷静和礼节,化险为夷
```

## p15 — 善不积不足以成名

```yaml
id: p15
title: 积善成名,积恶灭身
type: principle
source: 系辞下+坤卦文言
quote: |
  "善不积不足以成名,恶不积不足以灭身。
   小人以小善为无益而弗为也,以小恶为无伤而弗去也。
   故恶积而不可掩,罪大而不可解。易曰:'何校灭耳,凶。'"
decisional_meaning: 小善要积累,小恶要避免,否则会质变
modern_translation: 日常小决策的积累决定最终结果
```

## p16 — 安不忘危,存不忘亡

```yaml
id: p16
title: 安不忘危,存不忘亡
type: principle
source: 系辞下
quote: |
  "是故君子安而不忘危,存而不忘亡,治而不忘乱,
   是以身安而国家可保也。易曰:'其亡其亡,系于苞桑。'"
decisional_meaning: 平安时要想到危险,存在时要想到消亡
modern_translation: 居安思危,长期主义
```

## p17 — 德薄位尊,鲜不及矣

```yaml
id: p17
title: 德薄位尊,必逢其咎
type: principle
source: 系辞下
quote: |
  "德薄而位尊,知小而谋大,力小而任重,鲜不及矣。
   易曰:'鼎折足,覆公餗,其形渥,凶。'言不胜其任也。"
decisional_meaning: 能力配不上位置,迟早出事
modern_translation: 不要高攀不匹配的位置,德不配位必有灾
```

## p18 — 拟议以成变化

```yaml
id: p18
title: 拟议以成变化
type: principle
source: 系辞上
quote: |
  "言天下之至赜而不可恶也,言天下之至动而不可乱也。
   拟之而后言,议之而后动,拟议以成其变化。"
decisional_meaning: 决策要"拟" (推演) "议" (讨论) 之后再行动
modern_translation: 重要决策必须先推演讨论,再行动
```
