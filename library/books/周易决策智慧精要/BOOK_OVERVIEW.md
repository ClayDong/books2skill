## 三、诠释分析 (Interpretive Analysis)

### 核心理念 (15 关键概念)

| 概念 | 出处 | 决策含义 |
|---|---|---|
| **一阴一阳之谓道** | 系辞上 | 任何事都有正反两面,决策要看两面 |
| **时** | 彖传/文言 | 决策的核心是"时"——什么时候做 |
| **位** | 爻辞 | 处在什么位置 (初/中/上) 决定吉凶 |
| **中** | 爻辞 | 二/五爻为"中",不偏不倚 |
| **正** | 爻辞 | 阳爻居阳位为"正",阴爻居阴位为"正" |
| **几** | 系辞下 | 决策的"几微"——在事情未显时抓住征兆 |
| **应** | 爻辞 | 初与四、二与五、三与上为"应"位 |
| **承** | 爻辞 | 阴爻承阳爻为吉,逆承为凶 |
| **变** | 系辞 | 穷则变,变则通,通则久 |
| **通** | 系辞 | 不通则痛,通则不痛 |
| **悔** | 系辞 | 决策失误的修正机制 |
| **吝** | 系辞 | 决策不顺的轻微表现 |
| **咎** | 系辞 | 决策问题,但可补救 |
| **厉** | 系辞 | 决策危险信号 |
| **亢** | 乾卦上九 | 知进而不知退,知存而不知亡 |

### 决策三层结构

**第一层: 观象 (Observe Signs)**
- "君子居则观其象而玩其辞"
- 决策前先观察"象" (外在表现) + "辞" (内在规律)
- 现代对应: 数据收集 + 模式识别

**第二层: 玩辞 (Study Principles)**
- "动则观其变而玩其占"
- 决策中要"玩" (反复研究) 变化规律
- 现代对应: 反复推演 + 沙盘模拟

**第三层: 占变 (Divine Changes)**
- "易有四象,所以示也。系辞焉,所以告也。定之以吉凶,所以断也"
- 决策终要"断"——在不确定性中给出判断
- 现代对应: 风险评估 + 决策勇气

### 核心方法论: "君子之道"

> 系辞下: "履,德之基也。谦,德之柄也。复,德之本也。恒,德之固也。损,德之修也。益,德之裕也。困,德之辨也。井,德之地也。巽,德之制也。"

**九卦君子之道**:
1. **履** (履虎尾不咥人) → 礼是德的根基
2. **谦** (亨,君子有终) → 谦让是德的关键
3. **复** (出入无疾) → 复归是德的本源
4. **恒** (利有攸往) → 恒久是德的巩固
5. **损** (元吉无咎) → 减损是德的修炼
6. **益** (利有攸往) → 增益是德的扩充
7. **困** (亨,大人吉) → 困顿是德的检验
8. **井** (往来井井) → 井养是德的根基
9. **巽** (利有攸往) → 顺入是德的制宜

### 乾卦六爻: 人生/决策生命周期模型

```
初九 潜龙勿用        — 潜伏期,不动
九二 见龙在田        — 显露期,小试
九三 终日乾乾,夕惕若厉 — 奋斗期,日乾夕惕
九四 或跃在渊        — 抉择期,可上可下
九五 飞龙在天        — 鼎盛期,大展
上九 亢龙有悔        — 衰退期,知退
```

**核心原则**: "知进退存亡而不失其正者,其唯圣人乎"

## 四、批判分析 (Critical Evaluation)

### 三强批判 (3 Strong Critiques)

#### 批判 1: 概率迷信 vs 概率思维
- **问题**: 易经被简化为"算命",但其核心是"概率思维 + 决策框架"
- **表现**: 多数使用者关注"卦象吉凶",忽视"卦象背后的时位中正"
- **现代批判**: 决策书不是预测书,吉凶是"概率提示"而非"必然结果"
- **适配方案**: skill 中明确"占=概率评估",非"占=命运预言"

#### 批判 2: 时代脱节 (Gender/Class/Technology)
- **问题**: 卦辞爻辞基于周代社会(男尊女卑、君臣父子)
- **表现**: "臣道""妻道""子道"等表述不适合现代平等社会
- **现代批判**: 不能直接套用,需"取其精神,变其形式"
- **适配方案**: skill 中"位"概念重新定义为"角色定位",去除等级含义

#### 批判 3: 多义性 vs 可执行性
- **问题**: 卦辞/爻辞高度多义,不同注家有不同解读
- **表现**: 同一卦不同人解出不同决策
- **现代批判**: 易经提供"思考框架"而非"标准答案"
- **适配方案**: skill 中提供"3 种主流解读"而非"唯一正解"

### 弱批判 (Weak Critiques)

- 神秘色彩 (蓍草占卜): 剥离占卜形式,保留决策智慧
- 文字晦涩: 通过白话注释解决 (程颐朱熹已做)
- 缺乏实证: 经验性框架,不需科学验证

## 六、提取策略调整 (Extraction Strategy)

### 经典古籍 vs 现代书 差异

| 维度 | 现代书 (如炒股的智慧) | 经典古籍 (周易) |
|---|---|---|
| 作者洞见 | 个人视角,有独特方法论 | 集体创作,无单一作者 |
| 案例 | 真实案例丰富 | 缺乏现代案例 |
| 心理学 | 个人经验心理学 | 无现代心理学框架 |
| 可证伪 | 经验性,可验证 | 框架性,难证伪 |
| 行动指南 | 具体可执行 | 多义需解读 |

### 适配提取策略

1. **framework**: 提取"卦象 → 决策情境"映射,作为情境框架
2. **principle**: 提取"九卦君子之道"等可行动原则
3. **case**: 历史应用 (文王演易/孔子用易/历代易案) + 现代应用 (商战/管理)
4. **counter-example**: 不当使用的反例 (不观象/不知时/不守中)
5. **decision**: 64 卦 → 64 种决策场景映射
6. **glossary**: 时/位/中/正/几/应/承 等术语

### 验证方法调整

- V1 跨域: 现代决策科学中的对应 (时机理论/权变理论)
- V2 预测力: 历史案例中的验证 (历代易案)
- V3 独特性: 与"通用决策框架"对比的独特贡献
