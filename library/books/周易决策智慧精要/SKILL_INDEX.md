# 周易决策智慧精要 — 6个 Skill 索引

> 基于《周易》(公版) 提炼的 6 个核心决策 skill,覆盖从阶段判断到具体执行的全流程。

---

## 核心定位

周易系列 skill 是**东方决策哲学的现代翻译**。它不替代具体业务 skill(如股票/创业/管理),而是提供**底层决策框架**,与具体业务 skill 协同使用。

**6 个 skill 的总体结构**:

```
i-ching-time-position    ← 决策哲学基础 (时位中正)
        ↓
i-ching-life-cycle       ← 阶段判断 (乾卦六爻)
        ↓
i-ching-decision-omen    ← 信号识别 (见几而作)
        ↓
i-ching-revolution-timing ← 变革决策 (革+鼎)
        ↓
i-ching-advance-retreat  ← 进退取舍 (进退存亡)
        ↓
i-ching-crisis-transformation ← 危机应对 (坎+困)
```

**使用顺序**:
1. 面对决策 → 用 `i-ching-time-position` 做四维自检
2. 不确定阶段 → 用 `i-ching-life-cycle` 定位
3. 面对变革 → 用 `i-ching-revolution-timing` 判断时机
4. 面对进退 → 用 `i-ching-advance-retreat` 决策
5. 面对危机 → 用 `i-ching-crisis-transformation` 应对
6. 全程配合 `i-ching-decision-omen` 监测信号

## 核心原则 (跨 6 个 skill)

### 1. 时位中正 — 决策的四个基础
- **时**: 时机 (外部环境)
- **位**: 位置 (内部角色)
- **中**: 平衡 (各方利益)
- **正**: 正道 (长期价值)

**违反任何一项**,决策都难以成功。

### 2. 进退存亡 — 决策的取舍智慧
- 只知进不知退 → 亢龙有悔
- 只知退不知进 → 功亏一篑
- 关键: 知进退存亡 + 不失其正

### 3. 几微信号 — 决策的时机判断
- **几**: 变化的早期征兆
- **见几而作,不俟终日**: 识别就行动
- **难点**: 几 vs 噪声的辨别

### 4. 革鼎变革 — 决策的革命方法
- **革**: 打破旧秩序
- **鼎**: 建立新秩序
- **顺序不可颠倒**

### 5. 坎困转化 — 决策的危机应对
- **坎** (外险) → 行动突围
- **困** (内困) → 静守求助
- **变则通** → 主动求变

### 6. 乾卦六阶段 — 决策的阶段定位
- **初九 潜龙**: 积累期
- **九二 见龙**: 显露期
- **九三 乾乾**: 奋斗期
- **九四 或跃**: 抉择期
- **九五 飞龙**: 鼎盛期
- **上九 亢龙**: 衰退前期

## 用户旅程推荐

### 首次使用

1. **建立基础**: 阅读 `i-ching-time-position` (决策基础)
2. **理解阶段**: 阅读 `i-ching-life-cycle` (定位自己)
3. **识别信号**: 阅读 `i-ching-decision-omen` (培养观察力)
4. **具体决策**: 根据当前问题选用其他 3 个

### 进阶使用

- **日常自检**: 每周用 `i-ching-time-position` 做 1 次自检
- **阶段复盘**: 每月用 `i-ching-life-cycle` 重新定位
- **信号记录**: 每日用 `i-ching-decision-omen` 记录"几"
- **季度复盘**: 综合 6 个 skill 全面评估

### 团队/组织使用

- **决策机制**: 用 `i-ching-time-position` 作为团队决策框架
- **阶段管理**: 用 `i-ching-life-cycle` 管理项目和团队
- **变革管理**: 用 `i-ching-revolution-timing` 推动组织变革
- **危机管理**: 用 `i-ching-crisis-transformation` 应对危机

## 与炒股的智慧的 skill 整合

### 决策哲学 vs 具体操作

| 决策层级 | 用 i-ching-* | 用 stock-* |
|---|---|---|
| 战略层 | i-ching-life-cycle | stock-learning-stage |
| 战术层 | i-ching-decision-omen | stock-entry-decision |
| 操作层 | i-ching-advance-retreat | stock-stop-loss-decision |
| 心态层 | i-ching-crisis-transformation | stock-psychology-check |

**整合原则**:
- **战略层用周易** (东方哲学视角)
- **操作层用炒股的智慧** (西方技术视角)
- **心态层配合用** (东西方互补)

### 整合示例: 熊市决策

1. **i-ching-time-position** → 自检: 该不该抄底?
2. **i-ching-life-cycle** → 定位: 我处在"见龙"还是"潜龙"?
3. **i-ching-decision-omen** → 几: 政策面/资金面 的早期信号
4. **stock-trend-judgment** → 趋势: 熊市是否接近底部
5. **stock-entry-decision** → 入场: 具体的入场策略
6. **i-ching-advance-retreat** → 退: 止盈止损的时机预设
