# 福利与市场结构分析

## R — 原文 (Reading)

> "消费者剩余是买者愿意支付的量减去其实际支付的量。生产者剩余是卖者得到的量减去其生产成本。总剩余=消费者剩余+生产者剩余。使总剩余最大化的结果就是有效率的。垄断者生产的产量低于社会有效率的水平，产生无谓损失。"
>
> — 曼昆，第7+15章

---

## I — 方法论骨架 (Interpretation)

**福利分析（评估"好不好"）：**
- 消费者剩余 = 愿付价格 - 实付价格（衡量买方福利）
- 生产者剩余 = 实得价格 - 生产成本（衡量卖方福利）
- 总剩余 = 消费者剩余 + 生产者剩余
- 效率 = 总剩余最大化；无谓损失 = 总剩余的减少

**市场结构分析（判断"什么环境"）：**

| 结构 | 企业数 | 产品差异 | 进入壁垒 | 定价能力 | 效率 |
|---|---|---|---|---|---|
| 完全竞争 | 很多 | 同质 | 无 | 无（价格接受者） | 最高 |
| 垄断竞争 | 很多 | 差异化 | 低 | 弱 | 中 |
| 寡头 | 少数 | 同质/差异 | 中高 | 中（策略互动） | 中低 |
| 垄断 | 一个 | 独特 | 高 | 强（价格制定者） | 最低 |

**核心命题：**
- 竞争市场均衡 → 总剩余最大化（有效率）
- 垄断/税收/价格管制 → 产量偏离均衡 → 无谓损失
- 无谓损失与税率平方成正比

## A1 — 书中的应用 (Past Application)

### 案例 1: 垄断定价与无谓损失
- **问题**: 垄断企业如何定价？社会损失多大？
- **方法论的使用**: 垄断者选 MR=MC 的产量，此时 P>MC。与竞争市场（P=MC）相比，产量更低、价格更高。消费者剩余减少，部分变利润、部分消失（无谓损失）。
- **结论**: 垄断导致效率损失，是反垄断政策的理论基础
- **结果**: 反垄断法限制企业合并和市场势力滥用

### 案例 2: 税收的无谓损失
- **问题**: 税收的社会成本多大？
- **方法论的使用**: 税收在买者支付价和卖者得到价之间打入楔子。交易量减少→消失的交易带来的剩余就是无谓损失。税率翻倍→无谓损失四倍。
- **结论**: 税收不仅转移收入，还摧毁交易——高税率的效率代价非线性增长
- **结果**: 拉弗曲线理论——税率过高反而减少税收

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 政策分析者——"这个税/补贴/管制的效率损失多大？"
2. 企业战略师——"我们行业的竞争格局是什么？定价能力来自哪里？"
3. 投资者——"这家公司的护城河（市场势力）能持续吗？"
4. 反垄断评估——"这个合并会损害消费者吗？"

### 语言信号

- "这个政策好不好"
- "效率损失多大"
- "定价能力"
- "竞争激烈吗"
- "垄断/寡头"

### 与相邻 skill 的区分

- 与 `mankiw-supply-demand-elasticity` 的区别: 供求分析关注价格/数量变化，福利分析关注剩余分配和效率损失。
- 与 `mankiw-externality-policy` 的区别: 福利分析假设无外部性，外部性分析处理市场失灵。

## E — 可执行步骤 (Execution)

1. **判断市场结构**
   - 完成标准: 识别企业数量、产品差异度、进入壁垒、定价能力
   - 输出: 归类为完全竞争/垄断竞争/寡头/垄断

2. **计算福利分配（如需）**
   - 完成标准: 估算消费者剩余、生产者剩余、总剩余
   - 判停条件: 如果数据不足，用定性判断（"CS大/PS大/无谓损失明显"）

3. **识别无谓损失来源**
   - 完成标准: 判断是否存在垄断定价、税收扭曲、价格管制
   - 量化: 无谓损失 ≈ 0.5 × 价格差 × 产量减少

4. **评估政策/战略含义**
   - 完成标准:
     - 政策：是否需要反垄断/放松管制/调整税率？
     - 战略：企业的定价能力来自壁垒，壁垒可持续吗？

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 存在外部性——需先用 `mankiw-externality-policy` 修正市场结果
- 信息不对称——福利分析假设信息完全，不对称信息导致额外效率损失
- 行为偏差——消费者剩余基于"愿意支付"，但人的意愿可能被操纵

### 作者在书中警告的失败模式

- 垄断导致无效率——产量低于社会最优（c07, p12）
- 税收带来无谓损失——税率越高损失越大且非线性（p11）

### 作者的盲点 / 时代局限

- "完全竞争"作为基准——现实中大部分市场是垄断竞争，但政策建议常以竞争市场为参照
- 静态效率——动态效率（创新带来的长期福利增长）可能需要短期"无效率"（如专利保护垄断）

### 容易混淆的邻近方法论

- 市场失灵 vs 市场势力：前者包括外部性/公共物品/信息不对称，后者仅指垄断/寡头
