# 比较优势决策

## R — 原文 (Reading)

> "比较优势：一个生产者以低于其他生产者的机会成本生产一种物品。机会成本最低的人应该专业化生产该物品，通过贸易使双方都受益。即使一个人在两方面都更擅长，两人仍可通过专业化获益。"
>
> — 曼昆，第3章

---

## I — 方法论骨架 (Interpretation)

分工的依据不是"谁做得最好"（绝对优势），而是"谁放弃的代价最小"（比较优势）。

核心逻辑：
1. 计算每个人做每件事的机会成本（放弃的其他选项的价值）
2. 机会成本最低的人应该专业化做那件事
3. 通过交换/贸易，双方都获得比自己生产更多的产出

反直觉洞察：即使A在所有方面都强于B（绝对优势），A也不应该什么都做。因为A的时间有限，做X意味着放弃Y，如果A做Y的优势更大，就该让B做X——即使B做X不如A。

## A1 — 书中的应用 (Past Application)

### 案例 1: 农民与牧牛人
- **问题**: 农民和牧牛人各自生产牛肉和土豆，该怎么分工？
- **方法论的使用**: 牧牛人养牛的机会成本低（放弃的土豆少），农民种土豆的机会成本低（放弃的牛肉少）。即使牧牛人在两方面都更强，仍应专业化。
- **结论**: 专业化+贸易使总产量增加，双方都获得更多牛肉和土豆
- **结果**: 贸易不是零和博弈——双方都受益

### 案例 2: 国际贸易
- **问题**: 中国和美国该各自生产什么？
- **方法论的使用**: 中国在劳动密集型产品上机会成本低，美国在技术密集型产品上机会成本低
- **结论**: 各自专业化+贸易，比自给自足更好
- **结果**: 全球化贸易使各国总体福利提升

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 创业者什么都自己做——"我比雇的人做得更好"（但你的时间机会成本更高）
2. 团队leader分配任务——按"谁更擅长"分配而非"谁放弃的代价更小"
3. 个人职业选择——"我什么都会一点"vs"该专精什么"
4. 家庭分工——"谁该做饭谁该打扫"

### 语言信号

- "我自己做得更好"
- "他什么都不会"
- "谁该负责这个"
- "要不要外包"
- "我什么都得管"

### 与相邻 skill 的区分

- 与 `mankiw-opportunity-cost` 的区别: 机会成本是个人决策工具，比较优势是多人分工工具。比较优势建立在机会成本计算之上。

## E — 可执行步骤 (Execution)

1. **列出所有任务和所有人**
   - 完成标准: 形成任务×人的矩阵

2. **计算每个人做每件事的机会成本**
   - 完成标准: 对每个单元格，回答"做这件事意味着放弃什么？放弃的价值多大？"
   - 判停条件: 如果无法量化，用相对排序

3. **识别每个人的比较优势**
   - 完成标准: 每项任务分配给机会成本最低的人

4. **设计交换/协作机制**
   - 完成标准: 明确专业化后如何交换成果（贸易/内部协作/外包合同）

5. **验证总产出是否增加**
   - 完成标准: 专业化+交换的总产出 > 各自自给自足的总产出

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 任务不可分割——某些任务必须由同一人完成（如创意性工作）
- 转换成本过高——频繁切换任务的摩擦成本可能抵消专业化收益
- 学习阶段——初学者需要广泛尝试而非过早专业化
- 强制分工——非自愿的分工（如计划经济）无法实现比较优势的贸易收益

### 作者在书中警告的失败模式

- 用绝对优势代替比较优势——"谁做得好谁就做"忽略了机会成本
- 忽视贸易摩擦——运输成本、信息不对称、制度障碍可能蚕食贸易收益

### 作者的盲点 / 时代局限

- 静态分析——比较优势是静态的，但后天可以创造优势（韩国产业政策）
- 假设完全流动——现实中劳动力不能自由转换行业（摩擦性失业）
- 忽略分配效应——贸易总体受益但有人受损（如美国制造业工人）

### 容易混淆的邻近方法论

- 绝对优势：谁做得更好 ≠ 比较优势：谁放弃的更少
- 规模经济：大量生产降低成本 ≠ 比较优势：机会成本低
