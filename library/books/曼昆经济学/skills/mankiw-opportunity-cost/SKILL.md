# 机会成本思维

## R — 原文 (Reading)

> "一种东西的机会成本是为了得到这种东西所必须放弃的东西。当做出任何一项决策时，决策者都应该认识到每一种可能的行动所带来的机会成本。上大学的最大成本不是学费，而是放弃的工作收入。"
>
> — 曼昆，第1章

---

## I — 方法论骨架 (Interpretation)

做任何决策时，真正的成本不是你花了多少钱（会计成本），而是你放弃了什么最佳替代选项（机会成本）。

机会成本 = 显性成本（实际支出） + 隐性成本（放弃的最佳替代品价值）。

关键洞察：免费的也可能是昂贵的——如果免费的东西占用了你本可用于更高价值活动的时间。

应用公式：面对选择A时，问"选A意味着放弃什么？"，把放弃的选项中价值最高的那个，作为A的真实成本。

---

## A1 — 书中的应用 (Past Application)

### 案例 1: 上大学的决策
- **问题**: 上大学一年花了多少钱？
- **方法论的使用**: 不能只算学费+住宿+伙食。即使不上大学也要吃饭住宿，所以只有"比不上大学多花的部分"才是成本。最大成本是放弃的一年工资收入。
- **结论**: 上大学的机会成本 = 学费 + (放弃的工资) - (不上大学也要花的食宿费)
- **结果**: 大学运动员退学转职业——因为放弃的职业收入（几百万）远超大学收益

### 案例 2: 农民的生产选择
- **问题**: 农民该种土豆还是养牛？
- **方法论的使用**: 种土豆的机会成本 = 放弃的牛肉产量。养牛的机会成本 = 放弃的土豆产量。
- **结论**: 机会成本低的一方应专业化，通过贸易使双方受益（比较优势原理的基础）

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 创业者在算"自己干"的成本时——忘了算放弃的工资
2. 学生在纠结读研vs工作——只比"读研花多少"vs"工作赚多少"，忘了读研也是"放弃两年收入"
3. 投资者在考虑"已经亏了10万要不要继续"——把沉没成本当作决策依据
4. 管理者在决定"自己做还是外包"——只算外包费用，不算自己做的时间机会成本

### 语言信号

- "我已经投入这么多了"
- "自己干不用花钱"
- "免费的不用白不用"
- "读研才花X万，工作能赚Y万"
- "要不要放弃"

### 与相邻 skill 的区分

- 与 `mankiw-marginal-thinking` 的区别: 机会成本问"放弃什么"，边际思维问"多一单位值不值"。前者是选什么，后者是选多少。
- 与 `mankiw-incentive-analysis` 的区别: 机会成本是决策者的自我分析，激励分析是预测他人的行为反应。

## E — 可执行步骤 (Execution)

1. **列出所有替代选项**
   - 完成标准: 至少列出3个被放弃的选项

2. **估算每个替代选项的价值**
   - 完成标准: 对每个选项给出量化的价值估计（收入、时间、学习收益等）
   - 判停条件: 如果无法量化，用排序法（哪个价值最高）

3. **识别最高价值的替代选项 = 机会成本**
   - 完成标准: 明确说出"选A的真实成本是放弃B"

4. **比较决策收益与机会成本**
   - 完成标准: 收益 > 机会成本才值得做；否则选替代选项

---

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 沉没成本已发生且不可收回——沉没成本不是机会成本，不应影响决策
- 纯情感/道德决策——机会成本是经济分析工具，不适用于"要不要原谅某人"
- 紧急情况——需要快速反应时，完整的机会成本分析反而有害

### 作者在书中警告的失败模式

- 混淆会计成本与经济成本——只算花的钱，不算放弃的钱（ce01）
- 混淆会计利润与经济利润——会计利润为正不等于"赚钱"（ce06）

### 作者的盲点 / 时代局限

- "理性人"假设——行为经济学证明人系统性地忽视机会成本（现状偏差）
- 隐性成本难以量化——放弃的"健康""关系""幸福感"如何折算？

### 容易混淆的邻近方法论

- 沉没成本：过去已发生不可收回的成本 ≠ 机会成本（未来的放弃）
- 交易成本：执行交易的费用 ≠ 机会成本（放弃的替代品）
