# 激励分析框架

## R — 原文 (Reading)

> "激励是引起一个人做出行动的某种东西。由于理性人通过比较成本与收益做出决策，所以他们会对激励做出反应。当成本或收益变动时，人们的行为也会改变。"
>
> — 曼昆，第1章

---

## I — 方法论骨架 (Interpretation)

预测任何政策/规则/制度变化的效果时，核心问题是：它如何改变了相关主体的边际成本或边际收益？

分析框架：
1. 识别受影响的所有主体（不只是目标群体）
2. 分析变化如何改变每个主体的边际成本/收益
3. 预测行为反应（人们会怎样调整以应对新的激励）
4. 警惕意外后果（unintended consequences）——激励改变可能引发意料之外的行为

关键洞察：人们总是会对激励做出反应，只是反应方式可能出乎设计者意料。好的政策分析不是假设人们"应该"怎么做，而是分析激励结构下人们"会"怎么做。

## A1 — 书中的应用 (Past Application)

### 案例 1: 租金管制
- **问题**: 政府设定租金上限以保护租客
- **方法论的使用**: 租金上限降低了房东的边际收益→房东减少维护、转做出租→长期供给减少→住房短缺
- **结论**: 好意（保护租客）→ 坏结果（住房短缺+质量下降）
- **结果**: 纽约市租金管制区住房质量显著低于非管制区

### 案例 2: 汽油税与行为调整
- **问题**: 对汽油征税能否减少油耗？
- **方法论的使用**: 汽油税提高了驾驶的边际成本→短期：减少开车；中期：换省油车；长期：住近一点、坐公交
- **结论**: 激励效应随时间增强（弹性长期>短期）
- **结果**: 欧洲高汽油税国家人均油耗远低于美国

### 案例 3: 安全带法与驾驶行为
- **问题**: 强制安全带法减少了车祸伤亡吗？
- **方法论的使用**: 安全带降低了事故的边际成本（受伤概率降低）→驾驶员开得更冒险→事故增加
- **结论**: 佩兹曼效应——安全措施可能被风险补偿行为抵消
- **结果**: 死亡率下降幅度小于预期，行人死亡率上升

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 管理者设计KPI/奖金制度——需要预测员工会怎么应对（会不会只做被考核的、忽略其他？）
2. 政策制定者推出补贴/税收——需要预测市场主体的行为调整
3. 产品经理设计积分/等级系统——需要预测用户会不会"刷分"
4. 家长制定奖惩规则——需要预测孩子会怎么"钻空子"

### 语言信号

- "为什么他们不按预期做？"
- "这个政策本意是好的"
- "没想到会变成这样"
- "怎么设计KPI才有效"
- "他们会不会钻空子"

### 与相邻 skill 的区分

- 与 `mankiw-marginal-thinking` 的区别: 边际思维是自我决策分析，激励分析是预测他人对边际变化的反应。
- 与 `mankiw-supply-demand-elasticity` 的区别: 弹性分析关注"变化多大"，激励分析关注"怎么变"。

## E — 可执行步骤 (Execution)

1. **识别所有受影响的主体**
   - 完成标准: 列出直接受影响方 + 间接受影响方（至少3类主体）

2. **分析激励变化**
   - 完成标准: 对每个主体，回答"这个变化让什么变得更贵了？更便宜了？"
   - 判停条件: 如果没有任何主体的边际成本/收益改变，说明政策无效

3. **预测行为反应**
   - 完成标准: 对每个主体，回答"面对新激励，他们会怎么调整？"
   - 追问: "有没有可能以意料之外的方式反应？"

4. **检查意外后果**
   - 完成标准: 至少列出2个可能的意外后果
   - 追问: "目标群体能不能通过'钻空子'获利？"

5. **综合评估**
   - 完成标准: 预期效果 + 意外后果的净影响是否仍为正？

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 纯个人决策——没有其他主体会做出反应的场景
- 紧急情况——来不及做完整激励分析
- 强情感/非理性主导——人们可能明知激励但选择"不理性"（如成瘾行为）

### 作者在书中警告的失败模式

- 忽视意外后果——只考虑目标效果，不考虑行为反应（ce03）
- 价格管制导致更差结果——租金管制→短缺，最低工资→失业（ce04）
- 合成谬误——对个人有效的激励对整体可能无效（ce08）

### 作者的盲点 / 时代局限

- "理性人"假设——行为经济学证明人有系统性偏差，不一定按激励最优反应
- 忽视权力不对等——强势方可能转嫁激励成本给弱势方（如房东将维修成本转嫁给租客）

### 容易混淆的邻近方法论

- 博弈论：激励分析关注"激励→行为"，博弈论关注"策略互动→均衡"
- 行为经济学：关注"激励→非理性行为"，是对激励分析的补充
