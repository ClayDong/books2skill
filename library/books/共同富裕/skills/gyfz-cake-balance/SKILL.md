# 做大蛋糕与分好蛋糕权衡框架

## R — 原文 (Reading)

> "做大蛋糕是分好蛋糕的物质前提——不发展只分配只会共同贫穷。分好蛋糕是做大蛋糕的激励基础——不分配只发展会导致贫富悬殊、增长停滞。二者不是线性关系，而是相互促进的循环。"
>
> — 厉以宁等，总论

---

## I — 方法论骨架 (Interpretation)

发展与分配不是"先后关系"，而是"动态循环关系"。

核心命题：
1. **发展不足时过度分配 = 共同贫穷**（计划经济教训）
2. **发展充分时不分配 = 贫富悬殊**（拉美陷阱）
3. **正确做法 = 动态平衡**：在不同发展阶段找到不同的平衡点

权衡框架：
- 早期阶段：发展优先，分配保底（避免绝对贫困）
- 中期阶段：发展与分配并重（扩大中等收入群体）
- 后期阶段：分配优化，质量发展（橄榄型结构）

关键洞察：分配不是发展的"拖累"，好的分配是发展的"激励基础"——让劳动者分享成果才能持续创新。

## A1 — 书中的应用 (Past Application)

### 案例 1: 计划经济的教训
- **问题**: 为什么计划经济导致共同贫穷？
- **方法论的使用**: 追求绝对平均→消灭激励→生产积极性下降→蛋糕没做大→分配也没东西可分
- **结论**: 发展不足时过度分配=共同贫穷
- **结果**: 1978年改革转向"先做大蛋糕"

### 案例 2: 拉美中等收入陷阱
- **问题**: 为什么拉美国家陷入中等收入陷阱？
- **方法论的使用**: 只发展不分配→贫富悬殊→消费不足+社会动荡→增长停滞
- **结论**: 发展充分时不分配=贫富悬殊→增长停滞
- **结果**: 巴西、阿根廷基尼系数>0.5，社会动荡频发

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 政策制定者——"该优先发展还是优先分配？"
2. 企业管理者——"利润该用于扩张还是分红？"
3. 家庭决策——"收入该投资还是消费？"
4. 国际比较——"为什么有些国家发展了但不富裕？"

### 语言信号

- "先发展还是先分配"
- "做大蛋糕还是分好蛋糕"
- "增长还是公平"
- "效率还是公平"

### 与相邻 skill 的区分

- 与 `gyfz-three-distribution` 的区别: 本skill是"要不要分配"的战略权衡，三次分配是"怎么分配"的机制设计。

## E — 可执行步骤 (Execution)

1. **评估当前发展阶段**
   - 完成标准: 判断处于早期/中期/后期，蛋糕是否够大

2. **识别分配问题**
   - 完成标准: 是否存在贫富悬殊？基尼系数？社会流动性？

3. **找到动态平衡点**
   - 完成标准: 根据阶段确定发展vs分配的权重
   - 判停条件: 早期→发展70%+分配30%；中期→50%+50%；后期→30%+70%

4. **设计反馈机制**
   - 完成标准: 分配效果如何反哺发展？（如教育投资提升生产力）

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 纯技术决策——无分配维度
- 紧急危机——需快速反应，来不及权衡
- 零和博弈——发展与分配完全对立时（罕见）

### 作者在书中警告的失败模式

- 平均主义导致共同贫穷（ce01）
- 只发展不分配导致贫富悬殊（ce02）

### 作者的盲点 / 时代局限

- 假设"发展"与"分配"可分离——实际上某些发展模式（如资源掠夺）本身加剧不平等
- 对"高质量发展"的定义模糊——什么算"高质量"需要进一步明确

### 容易混淆的邻近方法论

- 效率vs公平：经典经济学命题，本skill强调"动态循环"而非"静态权衡"
