# 橄榄型分配结构评估框架

## R — 原文 (Reading)

> "橄榄型分配结构：中间大、两头小——中等收入群体占多数。不是'均等'，而是'中等为主'，允许合理差距。"
>
> — 厉以宁等，第三篇

---

## I — 方法论骨架 (Interpretation)

评估分配健康度，看是否形成"橄榄型"：

**结构特征:**
- 高收入群体：小（但存在，激励创新）
- 中等收入群体：大（社会稳定基础）
- 低收入群体：小（通过流动可上升）

**评估指标:**
1. **中等收入群体占比** —— 目标>50%
2. **基尼系数** —— 合理区间0.3-0.4
3. **社会流动性** —— 代际收入弹性<0.4（流动性高）
4. **贫困率** —— 绝对贫困=0，相对贫困<10%

**健康度判断:**
- 橄榄型（健康）：中等>50%，基尼0.3-0.4，流动性高
- 金字塔型（不健康）：低收入为主，基尼>0.4，流动性低
- 哑铃型（危险）：两极分化，中产萎缩，基尼>0.5
- 倒T型（危机）：极少数富裕+大多数贫困，基尼>0.6

关键洞察：橄榄型不是"均等"，而是"中等为主+允许合理差距"。差距是激励，但需可控。

## A1 — 书中的应用 (Past Application)

### 案例 1: 中国当前结构
- **问题**: 中国目前的分配结构是什么型？
- **方法论的使用**: 中等收入群体约4亿（约30%），基尼0.47，流动性下降
- **结论**: 介于金字塔型和橄榄型之间，正在转型
- **结果**: 目标2035年中等收入群体翻番至8亿

### 案例 2: 北欧橄榄型
- **问题**: 北欧为什么是典型橄榄型？
- **方法论的使用**: 高税收高福利→中等收入群体占60%+，基尼0.25-0.3
- **结论**: 但面临高税率抑制活力+移民冲击
- **结果**: 北欧模式可借鉴但不能照搬

## E — 可执行步骤 (Execution)

1. **收集分配数据**
   - 完成标准: 获取收入分组数据（五等分或十等分）

2. **计算核心指标**
   - 完成标准: 
     - 中等收入群体占比
     - 基尼系数
     - 社会流动性（代际收入弹性）
     - 贫困率

3. **判断结构类型**
   - 完成标准: 归类为橄榄型/金字塔型/哑铃型/倒T型
   - 判停条件: 参照健康度判断标准

4. **识别问题与对策**
   - 完成标准: 
     - 金字塔型→扩大中等收入群体（人力资本投资）
     - 哑铃型→防止中产萎缩（税收+社保）
     - 倒T型→紧急分配改革

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 个体收入评估——本框架针对群体结构
- 无分组数据——只有平均收入无法判断结构
- 短期波动——需多年数据判断趋势

### 作者在书中警告的失败模式

- 中产阶级萎缩的全球趋势（ce07）——中国能否逆势？
- 追求"均等"而非"橄榄型"——消灭差距=失去活力

### 作者的盲点 / 时代局限

- "橄榄型"作为最优解缺乏理论推导——为何不是其他形状？
- 对中产阶级定义不统一——收入/职业/教育/认同？

### 容易混淆的邻近方法论

- 基尼系数：单一指标，本框架是综合评估
- 洛伦兹曲线：可视化工具，本框架是判断框架
