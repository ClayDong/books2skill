# 共同富裕 — Framework 候选 (阶段 1 产出)

```yaml
- id: f01
  title: 做大蛋糕与分好蛋糕权衡框架
  type: framework
  source_chapter: 总论 + 第一篇
  source_quote: |
    "做大蛋糕是分好蛋糕的物质前提——不发展只分配只会共同贫穷。
    分好蛋糕是做大蛋糕的激励基础——不分配只发展会导致贫富悬殊、增长停滞。
    二者不是线性关系，而是相互促进的循环。"
  summary: |
    任何分配政策都需先问"蛋糕够不够大"。发展不足时过度分配=共同贫穷；
    发展充分时不分配=贫富悬殊。关键是在不同发展阶段找到动态平衡点。
  tags: [development, distribution, tradeoff]

- id: f02
  title: 三大差别诊断框架
  type: framework
  source_chapter: 总论 + 第一篇
  source_quote: |
    "共同富裕要缩小的核心对象是三大差别：区域差别、城乡差别、行业差别。
    三大差别的根源在生产力——技术、资本、管理和要素配置不到位。"
  summary: |
    诊断不平等问题时，不要泛泛谈"贫富差距"，而要拆解为三个维度：
    区域（东部vs西部）、城乡（城市vs农村）、行业（垄断vs竞争）。
    每个维度的根源不同，对策也不同。
  tags: [diagnosis, inequality, regional, urban-rural]

- id: f03
  title: 三次分配协调设计框架
  type: framework
  source_chapter: 第三篇
  source_quote: |
    "初次分配讲效率（市场）、再分配讲公平（政府税收社保）、
    三次分配讲自愿（慈善公益）。三者协调配套，缺一不可。"
  summary: |
    分配不是单一机制，而是三层架构：
    第一层（市场）：按要素贡献分配，激励效率
    第二层（政府）：税收+社保转移支付，纠正市场失灵
    第三层（社会）：慈善公益，自愿补充
    三层各有边界，不能越位或缺位。
  tags: [distribution, market, government, charity]

- id: f04
  title: 渐进式改革路径设计框架
  type: framework
  source_chapter: 总论 + 第一篇
  source_quote: |
    "共同富裕是分阶段推进的——不能犯急性子，不同阶段有不同标准。
    缩小差别不等于消灭差别——基尼系数0.3-0.4是合理区间。"
  summary: |
    改革路径设计需遵循"分阶段、有节奏、可调整"原则：
    1. 明确阶段性目标（而非终极目标）
    2. 设定合理区间（而非绝对均等）
    3. 保留调整空间（根据实际效果动态修正）
  tags: [reform, gradualism, phased-approach]

- id: f05
  title: 人力资本投资优先序框架
  type: framework
  source_chapter: 第三篇
  source_quote: |
    "扩大中等收入群体的关键是提升低收入群体的人力资本。
    人力资本是人的知识、技能、健康等能带来经济回报的属性。"
  summary: |
    缩小差距的核心杠杆是人力资本投资，而非直接收入转移。
    投资优先序：基础教育 > 职业技能 > 高等教育 > 健康。
    关键是提升低收入群体的"造血能力"而非"输血"。
  tags: [human-capital, education, priority]

- id: f06
  title: 企业共同富裕战略转型框架
  type: framework
  source_chapter: 第四篇
  source_quote: |
    "企业是共同富裕的重要参与者——不是被动对象，
    而是通过战略转型主动贡献。企业五个转型方向..."
  summary: |
    企业参与共同富裕不是被动缴税或捐款，而是主动战略转型：
    1. 科技创新（做大蛋糕）
    2. 员工共享（分好蛋糕）
    3. 供应链带动（先富带后富）
    4. ESG治理（长期主义）
    5. 社会责任（三次分配）
  tags: [corporate-strategy, ESG, social-responsibility]

- id: f07
  title: 橄榄型分配结构评估框架
  type: framework
  source_chapter: 第三篇
  source_quote: |
    "橄榄型分配结构：中间大、两头小——中等收入群体占多数。
    不是'均等'，而是'中等为主'，允许合理差距。"
  summary: |
    评估一个社会的分配健康度，看是否形成"橄榄型"：
    - 高收入群体：小（但存在，激励创新）
    - 中等收入群体：大（社会稳定基础）
    - 低收入群体：小（通过流动可上升）
    评估指标：中等收入群体占比、基尼系数、社会流动性。
  tags: [evaluation, distribution, middle-class]
```
