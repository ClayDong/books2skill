# 共同富裕 — Case 候选 (阶段 1 产出)

```yaml
- id: c01
  title: 计划经济平均主义导致共同贫穷
  type: case
  source_chapter: 总论
  source_quote: "计划经济平均主义=共同贫穷→改革开放先富带后富→全面小康→下一步共同富裕"
  summary: |
    1950-70年代中国实行计划经济和平均主义分配，结果不是共同富裕而是共同贫穷。
    原因：消灭了激励，"干多干少一个样"导致生产积极性下降。
    教训：绝对平均=共同贫穷。
  tags: [failure-case, egalitarianism]

- id: c02
  title: 改革开放先富带后富的成功
  type: case
  source_chapter: 第一篇
  source_quote: "允许一部分人先富起来，通过示范、帮扶、转移支付带动后富。"
  summary: |
    1978年后允许部分地区先富（深圳等经济特区），通过产业转移、税收返还、
    对口帮扶带动中西部发展。40年间使8亿人脱贫，证明"先富带后富"路径可行。
  tags: [success-case, reform]

- id: c03
  title: 北欧高福利模式的启示与局限
  type: case
  source_chapter: 总论
  source_quote: "北欧高福利模式、美国贫富分化教训→中国路径的差异化选择。"
  summary: |
    北欧（瑞典、丹麦）通过高税收+高福利实现基尼系数0.25-0.3，但面临：
    1. 高税率抑制创业活力
    2. 移民冲击福利体系
    3. 人口老龄化压力
    中国不能照搬，但可借鉴其"普惠性社保+积极劳动力市场政策"。
  tags: [international, welfare]

- id: c04
  title: 美国贫富分化的教训
  type: case
  source_chapter: 总论
  source_quote: "美国贫富分化教训→中国路径的差异化选择。"
  summary: |
    美国基尼系数0.48，1%人口拥有40%财富。原因：
    1. 金融化导致资本收益远超劳动收益
    2. 教育不平等固化阶层
    3. 工会衰落削弱劳工议价能力
    4. 税收制度偏向资本
    教训：市场至上导致两极分化，需政府主动调节。
  tags: [international, inequality]

- id: c05
  title: 6.1亿人月收入<1000元的现实
  type: case
  source_chapter: 总论
  source_quote: "2020年人均GDP 1.13万美元 vs 美国6.4万、6.1亿人月收入<1000元、基尼系数等→差距仍大"
  summary: |
    2020年总理记者会披露：6.1亿人月收入<1000元。
    说明：虽然总体小康，但底层收入仍低，共同富裕任重道远。
    这组数据成为推动共同富裕政策的现实依据。
  tags: [data, reality]

- id: c06
  title: 2035年中等收入群体翻番目标
  type: case
  source_chapter: 第三篇
  source_quote: "2035年中等收入群体翻番"
  summary: |
    目标：从2020年约4亿人增至2035年约8亿人。
    路径：提升低收入群体人力资本→进入中等收入。
    挑战：疫情后经济增速放缓、学历通胀、AI替代劳动。
  tags: [target, middle-class]

- id: c07
  title: 数字经济赋能共同富裕
  type: case
  source_chapter: 第二篇
  source_quote: "数字经济赋能、新型集体经济"
  summary: |
    数字经济通过以下方式促进共同富裕：
    1. 电商扶贫（农村产品直达城市）
    2. 远程办公（缩小区域差距）
    3. 数字金融（普惠信贷）
    4. 平台经济（灵活就业）
    但也带来：平台垄断、算法剥削、数字鸿沟等新问题。
  tags: [digital, technology]

- id: c08
  title: 新型集体经济的探索
  type: case
  source_chapter: 第二篇
  source_quote: "新型集体经济"
  summary: |
    新型集体经济（如华西村、塘约道路）通过：
    1. 集体资产股份化
    2. 农民持股分红
    3. 集体经营+市场化运作
    实现农村共同富裕。但也面临：产权不清、效率低下、腐败风险。
  tags: [rural, collective]
```
