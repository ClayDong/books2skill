# 共同富裕 — Glossary 候选 (阶段 1 产出)

```yaml
- id: g01
  term: 共同富裕
  definition: 全体人民通过共同劳动、共同创造共享发展成果的渐进过程
  distinction: 不是"平均分配"或"同步富裕"，强调渐进+共同创造+共享

- id: g02
  term: 三大差别
  definition: 区域差别、城乡差别、行业差别——共同富裕要缩小的核心对象
  distinction: 不是泛指"贫富差距"，而是有明确的三维结构

- id: g03
  term: 三次分配
  definition: 初次分配讲效率（市场）、再分配讲公平（政府税收社保）、三次分配讲自愿（慈善公益）
  distinction: 三次分配是独立维度，非强制，区别于税收

- id: g04
  term: 做大蛋糕与分好蛋糕
  definition: 发展是前提，分配是保障，二者动态兼顾
  distinction: 不是"先发展后分配"的线性关系，而是相互促进的循环

- id: g05
  term: 橄榄型分配结构
  definition: 中间大、两头小——中等收入群体占多数
  distinction: 不是"均等"，而是"中等为主"，允许合理差距

- id: g06
  term: 两高三均衡
  definition: 高质量发展+高效率治理；区域均衡+城乡均衡+收入均衡
  distinction: 作者自创的政策目标框架

- id: g07
  term: 人力资本
  definition: 人的知识、技能、健康等能带来经济回报的属性
  distinction: 不是"劳动力数量"，而是"劳动力质量"

- id: g08
  term: 先富带后富
  definition: 允许一部分人先富起来，通过示范、帮扶、转移支付带动后富
  distinction: 不是"劫富济贫"，而是"示范效应+制度引导+区域协调"

- id: g09
  term: 人本逻辑
  definition: 以人的全面发展为中心的发展逻辑
  distinction: 对立于"资本逻辑"或"物本逻辑"，强调发展目的是人

- id: g10
  term: 高质量发展
  definition: 创新驱动、协调、绿色、开放、共享的发展模式
  distinction: 不是单纯GDP增长，而是五大理念指导的发展

- id: g11
  term: 基尼系数
  definition: 衡量收入分配不平等程度的指标，0=绝对均等，1=绝对不均
  distinction: 0.3-0.4为合理区间，超过0.4预警，超过0.6高度不均

- id: g12
  term: 中等收入群体
  definition: 收入处于社会中间水平的群体，中国标准约家庭年收入10-50万
  distinction: 不是"中产阶级"（含资产维度），纯收入维度
```
