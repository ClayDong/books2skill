# 共同富裕 — Principle 候选 (阶段 1 产出)

```yaml
- id: p01
  title: 发展是前提，分配是保障
  type: principle
  source_chapter: 总论
  source_quote: "做大蛋糕是分好蛋糕的物质前提——不发展只分配只会共同贫穷。"
  summary: 发展与分配是动态循环关系，不是先后关系。发展不足时分配=共同贫穷；发展充分时不分配=贫富悬殊。
  tags: [development, distribution]

- id: p02
  title: 不是平均主义，允许合理差距
  type: principle
  source_chapter: 总论
  source_quote: "共同富裕不是平均主义也不是同步富裕...缩小差别不等于消灭差别。"
  summary: 允许合理差距存在（基尼0.3-0.4），激励创新和努力。绝对平均=共同贫穷。
  tags: [equality, incentive]

- id: p03
  title: 分阶段推进，不犯急性子
  type: principle
  source_chapter: 总论
  source_quote: "共同富裕是分阶段推进的——不能犯急性子，不同阶段有不同标准。"
  summary: 改革需循序渐进，设定阶段性目标而非一蹴而就。急性子改革导致社会动荡。
  tags: [gradualism, reform]

- id: p04
  title: 先富带后富，非劫富济贫
  type: principle
  source_chapter: 第一篇
  source_quote: "先富带后富——允许一部分人先富起来，通过示范、帮扶、转移支付带动后富。"
  summary: 不是劫富济贫，而是通过示范效应+制度引导+区域协调，让先富者带动后富者。
  tags: [inequality, mechanism]

- id: p05
  title: 人力资本是缩小差距的核心杠杆
  type: principle
  source_chapter: 第三篇
  source_quote: "扩大中等收入群体的关键是提升低收入群体的人力资本。"
  summary: 直接收入转移治标不治本，提升人力资本（教育、技能、健康）才能治本。
  tags: [human-capital, education]

- id: p06
  title: 市场效率+政府公平+社会自愿三者协调
  type: principle
  source_chapter: 第三篇
  source_quote: "初次分配讲效率、再分配讲公平、三次分配讲自愿。三者协调配套，缺一不可。"
  summary: 三层分配机制各有边界，不能越位或缺位。市场管效率，政府管公平，社会管补充。
  tags: [distribution, coordination]

- id: p07
  title: 高质量发展是共同富裕的基础
  type: principle
  source_chapter: 总论 + 第二篇
  source_quote: "高质量发展为基础...共同富裕是高质量发展的特征。"
  summary: 不是任何发展都能实现共同富裕，必须是创新驱动、协调、绿色、开放、共享的高质量发展。
  tags: [quality, development]

- id: p08
  title: 缩小差别不等于消灭差别
  type: principle
  source_chapter: 总论
  source_quote: "缩小差别不等于消灭差别——基尼系数0.3-0.4、区域/城乡差距控制在2倍以内是合理区间。"
  summary: 差距是激励创新的动力，完全消灭差距=失去活力。目标是"可控的合理差距"而非"绝对均等"。
  tags: [equality, boundary]
```
