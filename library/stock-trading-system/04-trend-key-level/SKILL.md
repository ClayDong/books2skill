# 趋势关键位层 — 道氏趋势 + 三级强度 + 周期定位

## R — 原文 (Reading)

> 趋势的定义：高点高于前高、低点高于前低 = 上升趋势；反之 = 下降趋势。趋势分三级：主要趋势（数月-数年）、次要趋势（数周-数月）、短期波动（数日-数周）。
>
> — 爱德华兹，《股市趋势技术分析》道氏理论章节（OCR原文）

> 四种周期：政策底 → 市场底 → 经济底 → 盈利底。三种杠杆轮动：高财务杠杆（金融地产）→ 高运营杠杆（周期股）→ 高估值杠杆（概念股）。
>
> — 邱国鹭，《投资中最简单的事》（周期章节）

---

## I — 方法论骨架 (Interpretation)

趋势是朋友，逆趋势是敌人——除非你确认趋势正在反转。

1. **道氏趋势定义**：
   - 上升：高点高于前高 + 低点高于前低
   - 下降：高点低于前高 + 低点低于前低
   - 震荡：无明确方向
2. **趋势三级**：主要趋势（数月-数年，最重要）/ 次要趋势（数周-数月）/ 短期波动（数日-数周，噪音）
3. **趋势强度三级量化**（关键修复）：
   - 强：价格在 20 日线上方 且 20 日线上行
   - 中：价格在 20 日线上方 但 20 日线走平
   - 弱：价格在 20 日线下方
4. **四种周期定位**（邱国鹭）：政策底 → 市场底 → 经济底 → 盈利底
5. **三种杠杆轮动**（邱国鹭）：高财务杠杆 → 高运营杠杆 → 高估值杠杆
6. **价值锚**（关键修复）：PE/PB 历史分位（< 30% 低估 / 30%-70% 合理 / > 70% 高估）

核心心法：**顺主要趋势，过滤短期波动，用价值锚定位置**。

## A1 — 书中的应用 (Past Application)

### 案例 1: 道氏趋势的顺势原则
- **问题**: 散户喜欢抄底逃顶，逆趋势操作
- **方法论的使用**: 用高点/低点序列判断趋势方向，只顺主要趋势操作
- **结论**: 趋势是你的朋友，不要逆趋势
- **结果**: 顺势操作的胜率显著高于逆势

### 案例 2: 邱国鹭的四种周期定位
- **问题**: 如何判断市场处于什么阶段
- **方法论的使用**: 政策底最先出现 → 市场底 → 经济底 → 盈利底最后
- **结论**: 政策底到市场底之间是布局期，盈利底确认后是主升浪
- **结果**: 在政策底后布局，盈利底后加仓，获得最佳收益

### 案例 3: 杠杆轮动的行业配置
- **问题**: 不同周期阶段该配置什么行业
- **方法论的使用**: 早期高财务杠杆（金融地产）→ 中期高运营杠杆（周期股）→ 后期高估值杠杆（概念股）
- **结论**: 顺杠杆轮动配置，不逆周期
- **结果**: 避免在错误阶段配置错误行业

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 判断当前市场/个股趋势方向，问"是涨是跌"
2. 评估趋势强度，问"趋势强不强""还能持续多久"
3. 找关键支撑位/阻力位，问"在哪买/在哪卖"
4. 判断市场处于什么周期阶段
5. 决定行业配置方向，问"该配什么行业"

### 语言信号

- "现在是什么趋势"
- "支撑位/阻力位在哪"
- "20日线/60日线怎么看"
- "大盘处于什么周期"
- "该看多还是看空"
- "趋势强不强"
- "杠杆轮动到哪个阶段了"

### 与相邻 skill 的区分

- 与 `01-risk-control-baseline` 的区别: 风控判断"买多少"，趋势判断"什么方向"——风控在前
- 与 `02-stock-screening` 的区别: 选股看公司，趋势看市场——前者微观，后者宏观
- 与 `03-kline-signal` 的区别: K线看短期信号，趋势看长期方向——前者是触发点，后者是大格局

### 下一步调用指引

- ✅ 强趋势 + 价值锚低估 → 调用 `03-kline-signal` 找入场点
- ⚠️ 中趋势 → 等待趋势明朗，不急于操作
- ❌ 弱趋势/下降趋势 → 空仓等待，不逆势

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验**
   - 完成标准: 确认已通过 `01-risk-control-baseline`
   - 判停条件: 若未跑风控 → 强制先跑 01

2. **收集输入参数**
   - 完成标准: 明确价格序列（近 60 日 OHLC）、20 日均线、60 日均线、PE/PB 历史分位
   - 判停条件: 若缺均线或分位数据 → ⚠️待补充

3. **趋势方向判断**（道氏）
   - 完成标准: 用高点/低点序列判断方向
   - 上升：高点高于前高 + 低点高于前低
   - 下降：高点低于前高 + 低点低于前低
   - 震荡：无明确方向
   - 判停条件: 若为震荡 → ⚠️ 不操作，等待方向明确

4. **趋势强度分级**（关键修复）
   - 完成标准: 按 20 日线状态分级
   - 强：价格 > 20 日线 且 20 日线上行（斜率 > 0）
   - 中：价格 > 20 日线 但 20 日线走平（斜率 ≈ 0）
   - 弱：价格 < 20 日线
   - 判停条件: 若为弱 → ❌ 不做多，空仓等待

5. **关键支撑/阻力位识别**
   - 完成标准: 标出近期关键价位
   - 支撑位：近期低点 / 20 日线 / 60 日线
   - 阻力位：近期高点 / 前期密集成交区
   - 判停条件: 若无明确关键位 → ⚠️待补充

6. **周期定位**（邱国鹭四种周期）
   - 完成标准: 判断当前处于哪个阶段
   - 政策底（最先）→ 市场底 → 经济底 → 盈利底（最后）
   - 政策底到市场底：布局期
   - 市场底到经济底：加仓期
   - 经济底到盈利底：主升浪
   - 盈利底后：持有期

7. **价值锚评估**（关键修复）
   - 完成标准: 检查 PE/PB 历史分位
   - < 30% 分位：低估，✅ 优先
   - 30%-70% 分位：合理，⚠️ 谨慎
   - > 70% 分位：高估，❌ 回避

8. **杠杆轮动判断**（邱国鹭）
   - 完成标准: 判断当前轮动阶段
   - 早期：高财务杠杆行业（金融地产）
   - 中期：高运营杠杆行业（周期股）
   - 后期：高估值杠杆行业（概念股）

9. **输出综合结果**
   - 完成标准: 给出三类输出之一 + 趋势详情
   - ✅ 强趋势 + 低估：趋势方向 + 强度 + 关键价位 + 周期阶段 + 建议操作
   - ⚠️ 中趋势/合理估值：等待建议 + 观察点
   - ❌ 弱趋势/下降/高估：空仓建议 + 理由

   输出格式：
   ```
   ✅强趋势 | 方向:上升 | 强度:强(20日线上行) | 关键位:支撑XX/阻力XX | 周期:布局期 | 价值锚:低估 | 下一步:03K线
   ⚠️中趋势 | 方向:震荡 | 强度:中(20日线走平) | 等待:趋势明朗
   ❌弱趋势/下降 | 原因:价格<20日线/PE>70%分位 | 建议:空仓等待
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯选股**: 用户问"公司好不好"不涉及趋势 → 走 02-stock-screening
- **纯K线形态**: 用户问"这个形态什么意思"不涉及大趋势 → 走 03-kline-signal
- **未跑风控**: 前置条件未满足
- **无均线数据**: 无法做强度分级

### 作者在书中警告的失败模式

- **逆趋势操作**: 下降趋势中抄底，大概率被套（道氏警告）
- **忽视主要趋势**: 用次要趋势/短期波动代替主要趋势判断
- **周期定位错误**: 把政策底当盈利底，过早加仓（邱国鹭警告）
- **杠杆轮动错配**: 在早期配置概念股，在后期配置金融地产

### 作者的盲点 / 时代局限

- **道氏理论的滞后性**: 趋势确认需要时间，转折点附近信号滞后
- **20 日线的参数依赖**: 不同市场/品种最优参数不同
- **A 股政策市特征**: 政策底到市场底的间隔不固定，有时政策底即市场底

### 容易混淆的邻近方法论

- **K线信号**: K线看短期触发，趋势看长期方向——不要用 K线代替趋势
- **均线系统**: 均线是趋势的工具之一，本 skill 综合多种方法

### 禁忌红线（不可逾越）

- 弱趋势（价格 < 20 日线）→ ❌ 禁止做多
- 下降趋势 → ❌ 禁止逆势抄底
- PE/PB > 70% 分位 → ❌ 禁止买入
- 未跑风控 → ❌ 禁止进入
