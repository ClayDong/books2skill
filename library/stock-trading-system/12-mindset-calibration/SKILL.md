# 心态校准层 — 时位中正心境引擎（严格限界）

## R — 原文 (Reading)

> "时位中正。时，是时机；位，是位置；中，是不偏不倚；正，是正当合理。做人做事，要看时机对不对、位置合不合、态度中不中、方向正不正。"
>
> — 曾仕强，《易经的奥秘》

> "亢龙有悔。龙飞得太高，必然有悔。盈不可久，高不可极。知进退存亡而不失其正者，其唯圣人乎。"
>
> — 《周易·乾卦》上九

---

## I — 方法论骨架 (Interpretation)

易经的心态校准哲学，核心是"时位中正"四字：

- **时**：判断当前处于什么阶段——是上升期、鼎盛期、还是转折期？不同阶段心态不同
- **位**：判断自己处于什么位置——是新手、熟手、还是老手？不同位置策略不同
- **中**：不偏不倚——既不冒进也不退缩，保持中道
- **正**：正当合理——做该做的事，不做不该做的事

周易七原则提供具体的心态校准工具：
1. **趣时**：变通顺应时机，不逆势而为
2. **见几而作**：捕捉微小征兆，见微知著
3. **藏器待时**：能力就绪但等时机，不急于出手
4. **安不忘危**：安顺时不忘记危险，居安思危
5. **亢龙有悔**：高位必收，盈不可久
6. **惧以终始**：全程保持戒惧，从开始到结束都不松懈
7. **穷则变**：极限将变，变则通，通则久

本 skill 的目标是"无咎"——不出大错，而非"大吉"——追求暴利。心态校准是防御性的，不是进攻性的。

**严格限界**：本 skill 只调心境，不输出交易指令。所有交易决策走量化系统。

## A1 — 书中的应用 (Past Application)

### 案例1: 曾仕强的"时位中正"应用
- **问题**: 一个交易者在牛市中赚了钱，觉得自己能力很强，准备加杠杆
- **方法论的使用**: 曾仕强的"时位"判断——当前处于牛市鼎盛期（时=鼎盛），但鼎盛意味着接近转折（极反而变）。用户的位置是"赚了钱但经验不足"（位=新手），不应在鼎盛期加杠杆
- **结论**: "中正"建议——保持当前仓位，不加杠杆。鼎盛期加杠杆=亢龙有悔
- **结果**: 用户保持仓位，随后市场回调时没有爆仓，保住了利润

### 案例2: 周易"藏器待时"应用
- **问题**: 一个交易者学了很多技术分析方法，但没有好的交易机会，急于出手
- **方法论的使用**: 周易"藏器待时"——能力已就绪（器已藏），但时机未到（待时）。急于出手=违背"趣时"原则
- **结论**: 心态校准——接受"等待也是交易的一部分"，不急于出手
- **结果**: 用户耐心等待，2周后出现明确信号才进场，避免了震荡市的反复止损

### 案例3: 周易"安不忘危"应用
- **问题**: 一个交易者连续盈利6个月，觉得自己的系统无敌了，开始放松风控
- **方法论的使用**: 周易"安不忘危"——安顺时（连续盈利）最危险，因为容易放松警惕。"惧以终始"——从开始到结束都要保持戒惧
- **结论**: 心态校准——盈利时反而要收紧风控，因为"盈不可久"
- **结果**: 用户恢复严格风控，随后市场出现黑天鹅时没有大亏

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **盈利后心态飘了**：连续赚钱后觉得自己"开窍了""无敌了"，想加杠杆/放松风控
2. **亏损后心态崩了**：连续亏钱后自我怀疑、焦虑、失眠，需要心态调整
3. **纠结该激进还是保守**：用户在"要不要加仓""要不要休息"之间犹豫，需要心境层面的指引
4. **等待中焦虑**：没有交易机会但急于出手，需要"藏器待时"的心态校准
5. **高位犹豫**：持仓盈利丰厚但不知道该不该收，需要"亢龙有悔"的心态校准

### 语言信号 (用户的话里出现这些就应激活)

- "赚了钱心态飘了怎么办"
- "亏了之后怎么调整心态"
- "现在该激进还是保守"
- "怎么保持平常心"
- "没有机会但忍不住想交易"
- "赚了这么多该不该收"
- "我是不是不适合做交易"
- "怎么才能不焦虑"

### 与相邻 skill 的区分

- 与 `09-emotion-control` 的区别：情绪控制检测具体认知偏差并给操作建议（减仓/暂停），心态校准是更宏观的"时位判断"和心境调适，**不输出交易指令**。情绪失控→情绪控制；心境需要调适→心态校准
- 与 `10-system-optimization` 的区别：系统优化输出量化判断（可交易/不可交易），心态校准输出心境建议（不输出交易指令）
- 与 `11-trade-review` 的区别：复盘输出"错误分类+改进措施"（可执行），心态校准输出"心境建议"（不输出交易指令）

### 严格限界声明（必须遵守）

1. **只输出心智状态校准，不输出交易信号**
2. **不判断买卖点、不判断趋势方向、不计算仓位**
3. **若周易说"该收"但量化系统说"趋势未破"，以量化系统为准**
4. **去占卜化：不涉及蓍草/占筮/算卦**
5. **目标是"无咎"（不出大错），不是"大吉"（暴利）**

### 下一步调用指引

- 心态校准后需要检测具体认知偏差 → 调用 `09-emotion-control`
- 心态校准后需要评估系统是否可交易 → 调用 `10-system-optimization`
- 心态校准后需要复盘近期交易找原因 → 调用 `11-trade-review`
- **本 skill 不输出"该买/该卖"的指令——交易决策走量化系统**

## E — 可执行步骤 (Execution)

**输入**：用户描述的心理状态 / 当前交易处境（盈利/亏损/等待/犹豫）

### 步骤1: 状态识别

根据用户描述，识别当前心理状态属于以下哪类：

| 状态类型 | 典型表现 | 对应周易原则 |
|---------|---------|------------|
| 飘（过度自信） | 连续盈利后觉得自己无敌、想加杠杆 | 亢龙有悔、安不忘危 |
| 崩（过度悲观） | 连续亏损后自我怀疑、焦虑失眠 | 穷则变、惧以终始 |
| 急（急于出手） | 没有机会但忍不住想交易 | 藏器待时、趣时 |
| 犹豫（进退失据） | 不知道该加仓还是该收、该激进还是保守 | 时位中正 |
| 麻痹（放松警惕） | 盈利后放松风控、觉得系统无敌 | 安不忘危、惧以终始 |

- **完成标准**: 明确用户当前状态类型（飘/崩/急/犹疑/麻痹）
- **判停条件**: 若用户描述的是具体交易决策需求（"该不该买""该不该卖"）→ 明确告知"本 skill 不输出交易指令，请走量化系统"，不强行做心态校准

### 步骤2: 时位判断

根据用户处境，做"时位"分析（仅心态层面，不输出交易指令）：

**时（时机判断）**：
- 上升期：市场向好，但需"安不忘危"
- 鼎盛期：接近转折，需"亢龙有悔"
- 转折期：方向不明，需"见几而作"
- 下降期：需"藏器待时"，不逆势

**位（位置判断）**：
- 新手位：经验不足，需"惧以终始"
- 熟手位：有经验但需"中正"
- 老手位：需"亢龙有悔"防飘

- **完成标准**: 输出时位判断（仅心境层面，不含交易指令）
- **判停条件**: 若时位判断与量化系统信号冲突 → 明确标注"以量化系统为准"，本 skill 的时位判断仅作心态参考

### 步骤3: 心态校准建议输出

根据步骤1的状态类型 + 步骤2的时位判断，输出心态校准建议：

```
心态校准建议:
═══════════════════════════════════
当前状态: {飘/崩/急/犹疑/麻痹}
时位判断: 时={上升/鼎盛/转折/下降} 位={新手/熟手/老手}

校准原则: {亢龙有悔/安不忘危/藏器待时/穷则变/时位中正/惧以终始}
心境建议: {具体的心态调整建议}

⚠️ 限界声明: 本建议仅限心态层面，不构成交易指令。
   交易决策（买卖点/仓位/止损）请走量化系统。
   若本建议与量化系统信号冲突，以量化系统为准。
═══════════════════════════════════
```

- **完成标准**: 输出心态校准建议（仅心境层面）+ 限界声明
- **输出三类**: ✅可执行（心态建议明确） / ❌不可执行（用户需求超出心态范畴，需走量化系统） / ⚠️待补充（状态描述不足，需更多信息）

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **具体交易决策**：用户在问"该不该买""该不该卖""买多少"时，走量化系统——本 skill 不输出交易指令
- **趋势方向判断**：用户在问"现在是牛市还是熊市""趋势破了没有"时，走量化系统
- **仓位计算**：用户在问"该买多少股""仓位怎么分配"时，走仓位管理 skill
- **情绪实时检测**：用户在交易中情绪失控时，走 `09-emotion-control`（本 skill 是心境调适，不是实时情绪检测）
- **系统评估**：用户在问"我的系统期望值多少"时，走 `10-system-optimization`

### 禁忌红线（严格限界）

- ❌ **绝不输出买卖信号**：本 skill 只调心境，不判断"该买还是该卖"
- ❌ **绝不判断趋势方向**：不输出"趋势向上/向下"的判断
- ❌ **绝不计算仓位**：不输出"该买X股""仓位Y%"的建议
- ❌ **绝不与量化系统冲突**：若周易说"该收"但量化系统说"趋势未破"，以量化系统为准
- ❌ **绝不涉及占卜**：不涉及蓍草/占筮/算卦/预测吉凶
- ❌ **目标是无咎不是大吉**：不追求暴利，追求不出大错

### 作者在书中警告的失败模式

- 曾仕强警告：易经不是算命工具，是"借象明理"——用卦象帮助思考，不是预测未来
- 曾仕强警告："时位中正"不是消极等待，是"在该做的时候做该做的事"——但做什么事走量化系统
- 周易警告："亢龙有悔"不是说"高位必跌"，是"高位时心态要收"——具体操作走量化系统

### 作者的盲点 / 时代局限

- 易经的智慧产生于农业社会，节奏慢——现代金融市场节奏快，"藏器待时"可能等几周，但金融市场机会转瞬即逝
- 易经强调"中道"，但交易有时需要极端操作（如黑天鹅时清仓）——本 skill 的"中道"是心态层面的，不限制量化系统的极端操作
- 周易的"见几而作"是定性描述，缺乏量化标准——具体征兆的识别走量化系统

### 容易混淆的邻近方法论

- **"易经算命"**：错误。本 skill 去占卜化，不涉及蓍草/占筮，只借易经的"时位中正"哲学做心态校准
- **"心态好就能赚钱"**：错误。心态校准是必要条件不是充分条件，没有正期望系统再好的心态也亏
- **"周易预测市场"**：错误。本 skill 不预测市场方向，只校准交易者的心境状态
