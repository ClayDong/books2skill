# 趋势出场层 — Trend Exit

## R — 原文 (Reading)

> "炒股最难的功夫之一就是让利润奔跑。大多数人的问题是：赚一点就跑，亏了就死扛。正确的做法是截短亏损，让利润奔跑——亏损时快走，盈利时慢走。"
>
> — 陈江挺，《炒股的智慧》

> "卖出六技术要领：1.趋势破坏 2.量价背离 3.关键位破位 4.均线死叉 5.指标顶背离 6.放量滞涨。出现任一信号，考虑减仓或清仓。"
>
> — 陈江挺，《炒股的智慧》

---

## I — 方法论骨架 (Interpretation)

趋势出场的核心原则是"让利润奔跑"——盈利时不要急于卖出，用回撤保护代替固定止盈。

**出场优先级（从高到低）**：
1. **止损**（07-atr-stop-loss 负责，本 skill 不重复）
2. **10日低点出场**：收盘价跌破10日最低价 → 减仓50%
3. **浮盈回撤保护**：
   - 浮盈 > 20%：回撤 10% 减仓50%（如浮盈从30%回撤到20%）
   - 浮盈 > 50%：回撤 15% 清仓止盈（如浮盈从60%回撤到45%）
4. **趋势破坏**：收盘价跌破20日均线 → 清仓

**冲突仲裁规则**：若同时触发减仓信号和清仓信号，取更保守的操作（清仓）。

**A股特色**：
- 涨停不卖（继续持有，涨停=多头强势）
- 跌停排队卖（跌停时无法成交，排队卖出）

## A1 — 书中的应用 (Past Application)

### 案例 1: 陈江挺的截短亏损
- **问题**: 散户常见错误是赚一点就跑、亏了死扛
- **方法论的使用**: 亏损时用ATR止损快速出场（截短亏损），盈利时用回撤保护让利润奔跑
- **结论**: 盈亏比 > 2:1 才能长期盈利，截短亏损是保证盈亏比的关键
- **结果**: 用回撤保护代替固定止盈，陈江挺在趋势股上捕获了3-5倍利润

### 案例 2: 卖出六技术要领实战
- **问题**: 趋势股何时该卖出
- **方法论的使用**: 监控六技术信号，出现任一信号考虑减仓，出现两个以上清仓
- **结论**: 六信号是趋势破坏的早期预警，组合使用胜率更高
- **结果**: 陈江挺用六信号在2007年牛市顶部成功逃顶，保住了大部分利润

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验：确认风险参数已就绪**
   - 检查 01-risk-control-baseline 是否已运行
   - 完成标准：总资金、持仓信息已确认
   - 判停条件：若未跑 01-risk-control-baseline，则提示用户先运行，跳到结束

2. **输入采集**
   - 当前价（元）
   - 10日最低价（元）
   - 入场价（元）
   - 浮盈比例 = (当前价 - 入场价) / 入场价 × 100%
   - 20日均线价（元）
   - 是否涨停（涨停=涨幅≥9.9%且封板）
   - 是否跌停（跌停=跌幅≥9.9%且封板）
   - 持仓期间最高价（元）
   - 持仓数量（股）
   - 完成标准：以上9项参数全部获取
   - 判停条件：若缺价格数据→⚠️待补充，要求用户补全当前价/10日最低价/入场价/20日均线价；若缺持仓数量→⚠️待补充，要求用户补全持仓数量

3. **账户级回撤校验与涨跌停处理（最高优先级）**
   - 账户级回撤校验（前置）：
     - 若账户回撤≥20%→输出❌清仓冷静（触发01回撤熔断）
     - 若15%≤回撤<20%→输出⚠️强制减仓至半仓
   - 若涨停且封板 → 输出 ✅持有（涨停=多头强势，继续持有）
   - 若跌停且封板 → 输出 ⚠️跌停排队卖（限价单挂跌停价，连续≥3日改市价单）
   - 连续跌停≥3日→触发账户级回撤校验（委托01）：若回撤≥20%→清仓冷静；若回撤<20%→继续排队+标记实际风险敞口（按跌停价估算）
   - 判停条件：若涨停，直接输出持有，跳到步骤8
   - 完成标准：账户级回撤与涨跌停状态判定完成

4. **止损检查（委托 07-atr-stop-loss）**
   - 若当前价 ≤ 止损价（入场价 - 2×ATR）→ 输出 ❌清仓（止损出场，委托07处理）
   - 判停条件：若触发止损，跳到步骤8输出清仓
   - 完成标准：止损状态判定完成

5. **10日低点检查**
   - 10日最低价 = 近10个交易日的最低收盘价（用收盘价，不用日内最低价）
   - 若收盘价 < 10日最低收盘价 → 输出 ⚠️减仓50%（趋势短期走弱信号）
   - 完成标准：10日低点信号判定完成

6. **浮盈回撤保护检查**
   - 浮盈回撤保护计算：
     - 持仓期间最高价 = 持仓至当日的最高收盘价（用收盘价而非日内最高价，避免假突破干扰）
     - 最高浮盈 = (持仓期间最高收盘价 - 入场价) / 入场价 × 100%
     - 当前浮盈 = (当前价 - 入场价) / 入场价 × 100%
     - 当前回撤 = 最高浮盈 - 当前浮盈
   - 若最高浮盈 > 50% 且 当前回撤 ≥ 15% → 输出 ❌清仓（回撤保护止盈）
   - 若最高浮盈 > 20% 且 当前回撤 ≥ 10% → 输出 ⚠️减仓50%（回撤保护减仓）
   - 完成标准：浮盈回撤保护判定完成

7. **趋势破坏检查**
   - 20日均线 = 近20个交易日收盘价的移动平均
   - 若收盘价 < 20日均线 → 标记"趋势预警"
   - 若连续2日收盘价 < 20日均线 → 输出 ❌清仓（趋势破坏确认）
   - 若仅1日跌破但次日收回 → 维持持有（假突破过滤）
   - 若收盘价 ≥ 20日均线 → 输出 ✅持有（趋势完好，继续持有）
   - 完成标准：趋势状态判定完成

8. **交易成本计算（出场时必算）**
   - 佣金 = 成交金额 × 0.025%（双向，最低5元）
   - 印花税 = 卖出成交金额 × 0.05%（单向，仅卖出）
   - 滑点 = 成交金额 × 0.1%（估计值，涨停/跌停时×2）
   - 总交易成本 = 佣金 + 印花税 + 滑点
   - 净盈亏 = 毛盈亏 - 总交易成本
   - 完成标准：净盈亏已计算

9. **输出格式**
   ```
   ✅持有 | 股票:XXXX | 当前价:XX元 | 浮盈:X% | 趋势完好(20日均线上方)
   ```
   ```
   ⚠️减仓50% | 股票:XXXX | 原因:跌破10日低点/浮盈回撤10% | 出场价:XX元 | 减仓后持有:NNN股
   ```
   ```
   ❌清仓 | 股票:XXXX | 原因:趋势破坏(跌破20日均线)/浮盈回撤15% | 出场价:XX元 | 毛盈亏:X% | 交易成本:XX元 | 净盈亏:X%
   ```
   ```
   ⚠️跌停排队 | 股票:XXXX | 策略:限价单挂跌停价(连续≥3日改市价单) | 预估滑点×2
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 止损判断（走 07-atr-stop-loss，本 skill 只管盈利出场和趋势出场）
- 入场判断（走 05-breakout-entry）
- 头寸计算（走 06-position-sizing）
- 价值投资长期持有（不靠技术信号出场，靠基本面变化决定）

### 作者在书中警告的失败模式

- **赚一点就跑**：陈江挺警告"最常见的错误是赚3%就卖，亏10%还死扛"。正确做法是让利润奔跑，用回撤保护代替固定止盈
- **不设止盈规则**：没有明确的出场规则，涨了想再涨、跌了想回本，最终坐过山车
- **忽视量价背离**：陈江挺卖出六要领之一，价格新高但成交量萎缩是趋势衰竭信号

### 作者的盲点 / 时代局限

- 陈江挺的经验基于2000年代A股，10日低点参数对大盘股可能偏敏感
- 浮盈回撤10%/15%是经验值，对高波动股票可适当放宽至15%/20%
- 卖出六技术要领是定性信号，未量化，需结合具体市场调整

### 容易混淆的邻近方法论

- "趋势出场"vs"止损"：趋势出场是盈利保护（本skill），止损是亏损控制（07-atr-stop-loss）
- "浮盈回撤保护"vs"移动止损"：浮盈回撤保护基于盈利比例，移动止损基于价格绝对值
- "10日低点出场"vs"2×ATR止损"：10日低点是趋势信号，2×ATR是风险控制，两者可同时使用
