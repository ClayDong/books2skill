# 均值回归层 — Mean Reversion

## R — 原文 (Reading)

> "我现在非常相信系统可以被视为是它们所产生的R乘数的分布。因为系统可以被看成R乘数的分布，所以利用这些分布来模拟你将来的交易结果如何就同样是可能的。"

> — 范·K·撒普，《通向财务自由之路》第2版前言（OCR原文，page 318-322）

> "有一个期望收益为正的系统，该系统产生于确保把你的损失控制在最低限度的想法：交易可以获得非常不错的回报—风险比率。"

> — 范·K·撒普，《通向财务自由之路》（OCR原文，page 202-205）

---

## I — 方法论骨架 (Interpretation)

均值回归的核心逻辑是"价格偏离均值后会回归"——当价格被恐慌情绪打到均值下方极端位置，理性买盘介入后价格会向均值（如20日均线、布林带中轨）回归。

**三层确认体系**：
1. **RSI 超卖**：RSI14 < 30 → 超卖信号；RSI14 < 20 → 极端超卖，信号更强
2. **布林带破下轨**：收盘价 < 布林带下轨 → 价格偏离均值达到统计极端
3. **成交量萎缩**：当日成交量 < 5日均量 × 0.8 → 恐慌抛售结束信号（抛压衰竭）

**与 05-breakout-entry 的区分**：
- 05 是趋势追涨——价格突破前高时入场，赌趋势延续
- 13 是超跌抄底——价格跌破均值极端时入场，赌价格回归
- 两者市场状态互斥：05 用于趋势市，13 用于震荡市
- 冲突仲裁：若用户同时看到突破和超卖信号，先判断市场状态——趋势市走 05，震荡市走 13

**A股适配**：
- 涨跌停处理：跌停封板不入场（买不到+次日可能继续跌停）；跌停开板可观察
- T+1 限制：当日买入次日才能卖，所以入场时机要在尾盘确认（14:50后），避免盘中信号反转
- ST 股票涨跌停 5%，RSI 和布林带信号失真，禁用

## A1 — 书中的应用 (Past Application)

### 案例 1: 撒普的系统设计原则
- **问题**: 如何在震荡市场构建正期望收益系统
- **方法论的使用**: 撒普指出系统可以被视为R乘数的分布，关键是识别市场状态——用波动率分位数判断震荡市，再在超卖区入场，目标回归均值
- **结论**: 均值回归系统的期望收益 = 胜率 × 平均盈利 - 败率 × 平均亏损，震荡市胜率较高
- **结果**: 撒普强调系统设计需匹配市场状态，均值回归在区间市场有效，但在趋势市场会连续止损

### 案例 2: 技术分析常识中的超卖反弹
- **问题**: 超跌后价格是否会反弹
- **方法论的使用**: RSI 跌破 30 + 成交量萎缩 → 识别抛压衰竭，在反弹启动时入场
- **结论**: 超卖反弹是技术分析的经典形态，但必须配合成交量萎缩确认
- **结果**: 超卖反弹在震荡市中概率较高，但极端趋势中失效（A股适配经验）

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 用户看到某只股票 RSI 很低（如 RSI<30），问"超跌了能不能抄底"
2. 用户看到某只股票跌破布林带下轨，问"是不是机会"
3. 用户判断当前是震荡市，问"震荡市怎么做"

### 语言信号 (用户的话里出现这些就应激活)

- "超跌""超卖""RSI 低""能不能抄底"
- "布林带跌破下轨""跌破布林带"
- "震荡市""区间震荡""横盘"
- "反弹""超跌反弹"

### 与相邻 skill 的区分

- 与 `05-breakout-entry` 的区别：05 是趋势追涨（突破前高买入），本 skill 是超跌抄底（跌破均值买入）；05 用于趋势市，本 skill 用于震荡市
- 与 `06-position-sizing` 的区别：本 skill 判断"要不要买"，06 判断"买多少"
- 与 `07-atr-stop-loss` 的区别：07 用 ATR 止损（趋势策略），本 skill 用布林带止损（均值回归策略），止损逻辑不同
- 与 `08-trend-exit` 的区别：08 让利润奔跑（趋势出场），本 skill 回到均值就走（均值回归出场），出场哲学不同

### 下一步调用指引

- ✅ 入场信号 → 调用 `06-position-sizing` 计算具体股数 → 本 skill 自带止损和出场规则（基于布林带和RSI），不委托 07/08
- ⚠️ 跌停封板 → 次日集合竞价观察（跌停开板且信号仍在 → 可入场；继续跌停 → 放弃）
- ❌ 不入场 → 等待信号或重新选股（调用 `02-stock-screening`）

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验：确认风险参数与市场状态**
   - 检查 01-risk-control-baseline 是否已运行
   - 完成标准：总资金、单笔风险上限（1%）、最大持仓限制已确认
   - 判停条件：若未跑 01-risk-control-baseline，则提示用户先运行，跳到结束
   - 大盘状态校验：震荡市才用此策略，趋势市走 05-breakout-entry（详见步骤3）

2. **输入采集**
   - 股票代码
   - 当前价（收盘价或实时价）
   - RSI14（14日相对强弱指标）
   - 布林带上轨、中轨、下轨（20日，2倍标准差）
   - 20日均价
   - 当日成交量、5日均量
   - 板块类型（主板/创业板/科创板）
   - 是否涨停（涨停=涨幅≥板块涨跌停限制×99%且封板）
   - 是否跌停（跌停=跌幅≥板块涨跌停限制×99%且封板）
   - 完成标准：以上10项参数全部获取
   - 判停条件：若缺价格数据→⚠️待补充，要求用户补全当前价/布林带三轨/20日均价；若缺指标数据→⚠️待补充，要求用户补全 RSI14/当日成交量/5日均量

3. **市场状态判断**（与05-breakout-entry使用相同指标：波动率分位 + ADX辅助）
   - 大盘20日波动率 = 近20个交易日大盘日收益率的标准差
   - 历史波动率分位 = 当前20日波动率在过去1年波动率序列中的百分位
   - ADX（平均定向指数）辅助：ADX<15确认极端无趋势（最佳均值回归环境），ADX<20确认无趋势，ADX>25确认有趋势
   - 若历史波动率分位 < 40% 且 ADX<20 → 震荡市 → ✅ 适用本策略
   - 若历史波动率分位 > 60% 且 ADX>25 → 趋势市 → ❌ 不适用，提示用户走 05-breakout-entry
   - 若 40% ≤ 分位 ≤ 60% 或 ADX在20-25之间 → 中性区间 → ⚠️ 谨慎使用，建议降低仓位至首仓20%
   - 判停条件：若判定为趋势市，输出"不适用+建议走05"，跳到结束
   - 完成标准：市场状态判定完成

4. **超卖信号确认**
   - RSI14 < 30 + 收盘价 < 布林带下轨 → 超卖信号 ✅
   - RSI14 < 20 → 极端超卖，信号更强（标记"强信号"）
   - RSI14 ≥ 30 或 收盘价 ≥ 布林带下轨 → ❌ 不满足超卖条件，输出不入场
   - 完成标准：超卖信号判定完成

5. **成交量确认**
   - 当日成交量 < 5日均量 × 0.8 → 恐慌抛售结束信号 ✅（抛压衰竭）
   - 当日成交量 > 5日均量 × 1.5 → 恐慌仍在继续 ❌ 不入场（抛压未衰竭，可能继续下跌）
   - **恐慌放量见底例外**：若RSI<20 且 成交量 > 5日均量 × 2.0 → ⚠️恐慌放量见底信号，允许入场但仓位降至首仓15%（极端恐慌往往是见底信号，如2018年10月、2022年10月）
   - 5日均量 × 0.8 ≤ 当日成交量 ≤ 5日均量 × 1.5 → 中性 ✅（量能温和，允许入场）
   - 完成标准：成交量有效性判定完成

6. **入场信号输出**
   - 跌停封板（跌幅≥板块限制×99%且封板）→ ⚠️ 跌停待次日，跳到步骤8输出
   - 满足超卖信号 + 成交量确认 → ✅ 入场信号
     - 入场价 = 当前价（或次日开盘价）
     - 首仓比例 = 计划仓位 × 30%（震荡市，试探性建仓，均值回归胜率高但需防极端趋势）；若步骤3判定为中性区间，降为计划仓位 × 20%
     - 止损价 = min(入场价 × 0.97, 布林带下轨 × 0.98)（取较低者，更保守）
     - **止损价锁定规则**：入场时锁定止损价，不随布林带下移而下移（防止风险放大）；若布林带下轨上移，止损价可随之上移（跟踪止损逻辑，锁定利润）
     - 目标价 = 布林带中轨（均值回归目标）
     - 下一步：调用 06-position-sizing 计算具体股数（传入布林带止损价，由06校验1%风险红线）
   - 不满足条件 → ❌ 不入场，输出原因
   - 完成标准：输出明确的入场/不入场/待次日信号

7. **出场规则（本 skill 自带，不委托 07/08——均值回归出场哲学与趋势出场不同）**
   - **7.0 账户级风控校验（最高优先级，委托 01 黑天鹅熔断 + 回撤校验）**：
     - 黑天鹅熔断：单日账户跌幅≥8%→强制减仓至半仓；≥10%→清仓冷静3日（触发条件：大盘跌≥3%或行业跌≥5%或个股跌≥8%且非个股利空）
     - 账户级回撤：回撤≥20%→清仓冷静；15%≤回撤<20%→减仓至半仓
     - 若触发熔断/回撤校验 → 直接输出对应信号，跳过以下步骤
   - **7.1 涨停处理**：若涨停且封板 → ⚠️涨停待次日（无法成交，次日集合竞价补执行出场信号）；涨停未封死 → 正常执行出场信号
   - **7.2 时间止损**：持有10个交易日未回到布林带中轨 → ❌清仓（避免资金低效占用，撒普强调期望收益需考虑时间成本）
   - 价格回到布林带中轨 → ⚠️ 减仓50%（均值回归目标达成一半）
   - 价格回到布林带上轨 → ❌ 清仓（均值回归完成，超越均值）
   - RSI14 > 70 → ❌ 清仓（超买信号，反向极端）
   - 收盘价 < 止损价 → ❌ 清仓（止损出场，止损价由步骤6定义）
   - 跌停封板 → ⚠️ 跌停排队卖（限价单挂跌停价，连续≥3日改市价单）
   - 完成标准：出场规则已明确，本 skill 自带止损和出场，不委托 07/08（但账户级风控委托01）

8. **输出格式**
   ```
   ✅入场信号 | 股票:XXXX | 入场价:XX元 | 首仓比例:30% | 信号强度:强信号(RSI<20) | 止损价:XX元 | 目标价:XX元(布林带中轨) | 量能:恐慌抛售结束
   ```
   ```
   ⚠️跌停待次日 | 股票:XXXX | 次日策略:集合竞价观察 | 跌停开板且信号仍在→可入场 | 继续跌停→放弃
   ```
   ```
   ❌不入场 | 股票:XXXX | 原因:未满足超卖条件/恐慌仍在继续/趋势市不适用
   ```
   ```
   ⚠️减仓50% | 股票:XXXX | 原因:价格回到布林带中轨 | 出场价:XX元 | 减仓后持有:NNN股
   ```
   ```
   ❌清仓 | 股票:XXXX | 原因:价格回到布林带上轨/RSI>70/跌破止损 | 出场价:XX元 | 毛盈亏:X% | 交易成本:XX元 | 净盈亏:X%
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 趋势市（大盘波动率分位 > 60%，走 05-breakout-entry）
- ST/*ST股票（涨跌停5%，RSI 和布林带信号失真）
- 上市首日及次新股（无历史数据，20日均价和布林带无意义）
- 基本面恶化的股票（超跌可能是价值毁灭，不是均值回归）
- 已持仓股票的加仓判断（走 06-position-sizing）

### 作者在书中警告的失败模式

- **在趋势市用均值回归**：撒普警告系统需匹配市场状态——趋势股超卖后可能继续超卖，抄底抄在半山腰
- **忽视成交量**：没有成交量萎缩确认的超卖反弹不可靠——恐慌未止的反弹是死猫跳（技术分析常识）
- **抄底抄成扛单**：超跌不等于见底，撒普强调"必须设止损，否则一次极端趋势就会爆仓"

### 作者的盲点 / 时代局限

- **均值回归在极端趋势中会失效**：撒普承认系统在极端趋势中会连续止损，这是方法的内在局限
- 撒普的经验基于美国市场，A股的涨跌停和 T+1 制度会影响入场时机
- RSI 参数（14日）对 A 股短线可能偏慢，部分交易者用 RSI6，但本 skill 保守采用 RSI14
- 布林带 2 倍标准差是经验值，对高波动股票可放宽至 2.5 倍

### 容易混淆的邻近方法论

- "均值回归"vs"趋势跟随"：均值回归赌价格回归均值（本skill），趋势跟随赌价格延续趋势（05-breakout-entry），两者哲学对立
- "超卖反弹"vs"抄底"：超卖反弹是技术信号驱动的短线交易（本skill），抄底常指基本面低估的左侧买入，方法论不同
- "试探性建仓"vs"分批建仓"：试探是先小仓验证方向，分批是按计划分次买入
