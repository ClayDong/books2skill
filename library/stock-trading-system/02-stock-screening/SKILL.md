# 选股分析层 — 估值优先 + 行业格局 + 陷阱排查

## R — 原文 (Reading)

> 选股三要素：估值（最易把握）> 品质（定价权）> 时机（淡化）。投资三板斧：波特五力看议价权、杜邦分析看盈利模式、估值分析看贵不贵。行业格局四等级：月朗星稀 > 一超多强 > 两分天下 > 百花齐放。
>
> — 邱国鹭，《投资中最简单的事》（选股章节）

> CANSLIM：C 当季 EPS ≥ 25%、A 年度 EPS ≥ 25%、N 新产品/新高、S 供需、L 龙头、I 机构持仓、M 市场方向。
>
> — 欧奈尔，《笑傲股市》（CANSLIM 章节，训练数据）

---

## I — 方法论骨架 (Interpretation)

选股不是"找好公司"，而是"找便宜的好公司"。

1. **大盘过滤**（关键修复）：指数 > 60 日线才选股，否则空仓等待
2. **行业格局评估**：月朗星稀（最优）> 一超多强 > 三足鼎立 > 百花齐放（最差）
   - 月朗星稀典型案例：白酒（茅台 48%）、MDI（万华 70%）、梯媒（分众 84%）
   - 一超多强典型案例：新能源电池（宁德 47%）、安防（海康 38%）、新能源车（比亚迪 33%）
   - 三足鼎立典型案例：白电（格力 32% / 美的 28% / 海尔 10%）——原"两分天下"应修正为"三足鼎立"
   - 百花齐放典型案例：银行（36 家上市银行）、房地产
3. **基本面三板斧**：
   - 波特五力：议价权 / 比较优势 / 门槛高低
   - 杜邦分析：连续 5 年盈利模式是否稳定
   - 估值分析：同业横比 + 历史纵比（A股估值区间适配见下）
   - A股估值区间适配：
     · 金融（银行/保险）：PE 4-8 倍 + PB<1 + 股息率>4%
     · 消费（白酒/家电/食品）：PE 15-25 倍
     · 医药（创新药）：PE 25-50 倍
     · 科技（半导体/新能源/消费电子）：PE 25-50 倍
     · 周期（化工/资源/建材）：PB 1-3 倍 + 周期位置判断
     · 传媒：PE 20-30 倍
4. **定价权 6 来源**：垄断 / 品牌 / 技术专利 / 资源矿产 / 牌照准入 / 规模效应（新增）
   - 规模效应案例：比亚迪垂直整合 + 宁德时代全球 47% 市占率
5. **CANSLIM 成长过滤**（A股适配）：EPS 阈值按行业分类，新产品/新高、龙头、机构持仓
   - 成长股（医药/新能源/半导体）：EPS ≥ 15% + ROE ≥ 15%
   - 白马股（白酒/家电/消费）：EPS ≥ 10% + ROE ≥ 15% + 市占率稳固
   - 周期股（化工/资源/建材）：不适用 CANSLIM，看 PB + 周期位置
   - 金融股（银行/保险/证券）：不适用 CANSLIM，看不良率 + 股息率
6. **陷阱排查**：价值陷阱 6 类 + 成长陷阱 9 类（负面清单）

核心心法：**估值最易把握，品质决定持久，时机可以淡化**。先排雷再选股。

## A1 — 书中的应用 (Past Application)

### 案例 1: 邱国鹭的估值优先
- **问题**: 散户追"好公司"不看价格，结果买在山顶
- **方法论的使用**: 先看估值（PE/PB 历史分位），再看品质（定价权），最后看时机
- **结论**: 便宜的好公司 > 贵的好公司
- **结果**: 避免了"好公司但买贵了"的常见陷阱

### 案例 2: 行业格局决定胜负
- **问题**: 同样是龙头，为什么有的赚钱有的不赚
- **方法论的使用**: 月朗星稀行业（如白酒）龙头溢价高；百花齐放行业（如家电早期）内卷严重
- **结论**: 选行业比选公司更重要
- **结果**: 优先配置月朗星稀行业的龙头

### 案例 3: CANSLIM 过滤成长股
- **问题**: 如何识别真正在成长的公司
- **方法论的使用**: 当季 EPS ≥ 25% + 年度 EPS ≥ 25% + 新产品催化 + 机构进场
- **结论**: 成长股必须有盈利支撑，不是讲故事
- **结果**: 过滤掉讲概念无盈利的伪成长股

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 看到一只股票，问"值不值得买""估值贵不贵"
2. 评估某行业的竞争格局，问"这行业好不好做"
3. 排查某只股票是不是价值陷阱/成长陷阱
4. 用 CANSLIM 筛选成长股
5. 比较同行业几只股票哪个更优

### 语言信号

- "这只股票估值贵不贵"
- "PE 多少算合理"
- "这个行业格局怎么样"
- "有没有定价权/护城河"
- "是不是价值陷阱"
- "CANSLIM 怎么用"
- "基本面好不好"

### 与相邻 skill 的区分

- 与 `01-risk-control-baseline` 的区别: 风控判断"买多少"，选股判断"买什么"——风控在前
- 与 `03-kline-signal` 的区别: 选股看基本面，K线看技术面——前者是"值不值"，后者是"何时买"
- 与 `04-trend-key-level` 的区别: 选股看公司，趋势看市场——前者是微观，后者是宏观

### 下一步调用指引

- ✅ 选股通过 → 调用 `03-kline-signal` 找入场点 或 `04-trend-key-level` 判断趋势
- ❌ 选股淘汰 → 停止，换标的或空仓
- ⚠️ 大盘过滤未过 → 空仓等待，不选股

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验**
   - 完成标准: 确认已通过 `01-risk-control-baseline`
   - 判停条件: 若未跑风控 → 强制先跑 01，否则停止

2. **收集输入参数**
   - 完成标准: 明确股票代码、行业、PE/PB/ROE、近 5 年财报数据、大盘指数 vs 60 日线
   - 判停条件: 若缺财报数据 → ⚠️待补充

3. **大盘过滤**（关键修复）
   - 完成标准: 检查大盘指数 vs 60 日线
   - 指数 > 60 日线 → ✅ 可选股
   - 指数 < 60 日线 → ❌ 空仓等待，不选股
   - 判停条件: 若大盘未过 → 直接输出 ❌，停止

4. **行业格局评估**
   - 完成标准: 判断行业属于四等级之一
   - 月朗星稀 → ✅ 优先
   - 一超多强 → ✅ 可选龙头
   - 三足鼎立 → ⚠️ 谨慎
   - 百花齐放 → ❌ 回避

5. **基本面三板斧**
   - 完成标准: 执行三项分析
   - 波特五力：议价权 / 比较优势 / 门槛
   - 杜邦分析：连续 5 年 ROE 稳定性
   - 估值分析：PE/PB 同业横比 + 历史纵比（5 年分位）
   - A股估值区间适配（按行业分类）：
     · 金融（银行/保险）：PE 4-8 倍 + PB<1 + 股息率>4%
     · 消费（白酒/家电/食品）：PE 15-25 倍
     · 医药（创新药）：PE 25-50 倍
     · 科技（半导体/新能源/消费电子）：PE 25-50 倍
     · 周期（化工/资源/建材）：PB 1-3 倍 + 周期位置判断
     · 传媒：PE 20-30 倍

6. **定价权检查**
   - 完成标准: 确认定价权来源（垄断/品牌/专利/资源/牌照/规模效应）
   - 有定价权 → ✅ 加分
   - 无定价权 → ⚠️ 谨慎

7. **CANSLIM 成长过滤**（若选成长股）
   - 完成标准: 逐项检查 7 要素
   - EPS 阈值按行业分类（A股适配）：
     · 成长股（医药/新能源/半导体）：EPS ≥ 15% + ROE ≥ 15%
     · 白马股（白酒/家电/消费）：EPS ≥ 10% + ROE ≥ 15% + 市占率稳固
     · 周期股/金融股：不适用 CANSLIM，走步骤 7.5
   - 其余要素：新产品 / 新高 / 供需 / 龙头 / 机构持仓
   - ≥ 5 项满足 → ✅ 通过
   - < 5 项 → ❌ 淘汰

7.5 **周期股特殊处理**（化工/资源/建材，CANSLIM 不适用时走此步）
   - 完成标准: 用周期股专属指标替代 PE 估值
   - 不看 PE，看 PB 历史分位（<20% 为底部区域）
   - 看核心产品价格周期位置（底部/中部/顶部）
   - 看产能利用率（<70% 为底部信号）
   - PB 历史分位 <20% + 价格周期底部 + 产能利用率 <70% → ✅ 通过
   - 否则 → ⚠️ 谨慎或 ❌ 淘汰

8. **陷阱排查**（负面清单）
   - 完成标准: 排查价值陷阱 6 类 + 成长陷阱 9 类
   - 价值陷阱：周期股顶点/技术淘汰/高估值假便宜/夕阳行业/会计造假/价值毁灭
   - 成长陷阱：讲故事无盈利/高增长不可持续/烧钱无尽头/管理层套现/估值透支/竞争恶化/政策打压/技术路径错误/规模不经济
   - 命中任一 → ❌ 淘汰

9. **输出综合结果**
   - 完成标准: 给出三类输出之一 + 估值区间
   - ✅ 通过：列出通过项 + 估值区间（PE/PB 合理范围）
   - ❌ 淘汰：列出淘汰原因
   - ⚠️ 待补充：列出缺失数据

   输出格式：
   ```
   ✅选股通过 | 标的:XXXX | 行业格局:月朗星稀 | 估值区间:PE XX-XX | 下一步:04趋势→03K线
   ⚠️待补充 | 缺失数据:XXX | 补全建议:XXX
   ❌淘汰 | 原因:大盘<60日线/陷阱/格局差
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯仓位/风控问题**: 用户问"买多少"不涉及选股 → 走 01-risk-control-baseline
- **纯技术分析**: 用户问"K线形态如何"不涉及基本面 → 走 03-kline-signal
- **纯趋势判断**: 用户问"大盘方向"不涉及个股 → 走 04-trend-key-level
- **未跑风控**: 前置条件未满足，禁止直接选股

### 作者在书中警告的失败模式

- **追好公司不看估值**: 贵的好公司也是陷阱（邱国鹭警告）
- **忽视行业格局**: 百花齐放行业再好的公司也难赚（邱国鹭警告）
- **讲故事无盈利**: CANSLIM 缺 EPS 支撑的伪成长（欧奈尔警告）
- **价值陷阱**: 看着便宜实则价值毁灭（邱国鹭警告）

### 作者的盲点 / 时代局限

- **CANSLIM 的 A 股适配**: EPS 25% 阈值在 A 股偏高，需结合行业调整
- **60 日线的滞后性**: 趋势转折点附近 60 日线信号滞后
- **财报造假风险**: 基本面分析依赖财报真实性，A 股有造假风险

### 定价权判断注意事项

- **代工模式陷阱**: 代工企业（如立讯精密/工业富联）虽有技术但无自主品牌，议价权有限，应标注"弱定价权"
- **品牌定价权减弱标志**: 行业价格战加剧、毛利率持续下滑（如格力空调）

### 容易混淆的邻近方法论

- **趋势分析**: 趋势看价格方向，选股看公司价值——不要用趋势代替选股

### 禁忌红线（不可逾越）

- 大盘 < 60 日线 → ❌ 禁止选股
- 命中价值陷阱/成长陷阱 → ❌ 禁止买入
- 未跑风控 → ❌ 禁止进入
