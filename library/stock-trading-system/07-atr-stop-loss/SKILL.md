# ATR止损层 — ATR Stop Loss

## R — 原文 (Reading)

> "海龟止损规则：任何头寸的止损价 = 入场价 - 2×ATR。如果价格触及止损，立即平仓，没有例外。ATR让止损距离随波动自适应——波动大的市场止损宽，波动小的市场止损窄。"
>
> — 柯蒂斯·费思，《海龟交易法则》第7章

> "加仓后，所有单位的止损价上移2.5%至新入场价-2×ATR。止损只能上移，不能下移——这是铁律。"
>
> — 柯蒂斯·费思，《海龟交易法则》第7章

---

## I — 方法论骨架 (Interpretation)

ATR止损的核心是"波动自适应"——止损距离随市场波动自动调整，而非固定百分比。

**止损价公式**：
```
初始止损价 = 入场价 - 2 × ATR(14日)
```

**加仓止损修正（关键修复）**：
```
加仓后止损价 = max(原止损价, 新入场价 - 2 × 新ATR)
```
止损只能上移不能下移——加仓后若新止损低于原止损，维持原止损；若新止损高于原止损，上移至新止损。

**止损红线**：
- 单笔亏损 ≤ 总资金 1%（目标值）
- 单笔亏损 ≤ 总资金 2%（硬上限，超过必须立即止损）

**A股特殊处理**：
- T+1：买入当日无法止损，次日集合竞价（9:25）处理
- 跌停：跌停排队卖出，连续跌停≥3日改市价单（避免排队过久扩大亏损）

## A1 — 书中的应用 (Past Application)

### 案例 1: 海龟咖啡糖止损
- **问题**: 咖啡糖入场后价格回落，何时止损
- **方法论的使用**: 止损价=入场价-2×ATR，价格触及止损立即平仓，不犹豫
- **结论**: ATR自适应波动，咖啡糖波动大时止损宽，避免被噪音震出
- **结果**: 海龟在咖啡糖上小亏退出，但保住了本金，后续趋势品种大赚覆盖损失

### 案例 2: 加仓止损上移
- **问题**: 海龟加仓后，原止损价是否需要调整
- **方法论的使用**: 加仓后所有单位止损上移至新入场价-2×ATR，锁定部分利润
- **结论**: 止损上移后，即使趋势反转，加仓部分也不会亏
- **结果**: 海龟在原油趋势中加仓3次，止损上移后最终在反转时保住了70%利润

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验：确认风险参数已就绪**
   - 检查 01-risk-control-baseline 是否已运行
   - 完成标准：总资金、单笔风险上限（1%）、硬上限（2%）已确认
   - 判停条件：若未跑 01-risk-control-baseline，则提示用户先运行，跳到结束

2. **输入采集**
   - 入场价（元）
   - 当前ATR（14日真实波幅均值，元）
   - 持仓数量（股）
   - 是否跌停（跌停=跌幅≥板块涨跌停限制×99%且封板；主板10%/创业板科创板20%）
   - 已有止损价（元，加仓场景需要）
   - 是否为买入当日（T+0，无法止损）
   - 连续跌停天数（若跌停）
   - 完成标准：以上7项参数全部获取
   - 判停条件：若缺入场价→⚠️待补充，要求用户补全入场价；若缺ATR→⚠️待补充，要求用户补全14日ATR；若缺持仓数量→⚠️待补充，要求用户补全持仓数量

3. **T+1 校验**
   - 若为买入当日（T+0）→ 输出 ⚠️"T+1限制：买入当日无法止损，次日集合竞价（9:25）处理"
   - 判停条件：若T+0，跳到步骤7输出
   - 完成标准：T+1状态判定完成

4. **计算止损价**
   - 初始止损价 = 入场价 - 2 × ATR
   - 加仓场景：新止损价 = max(原止损价, 新入场价 - 2 × 新ATR)
   - 止损价上移校验：新止损价 ≥ 原止损价（止损只能上移不能下移）
   - 完成标准：止损价已计算

5. **止损红线校验**
   - 止损金额 = (入场价 - 止损价) × 持仓数量
   - 止损占比 = 止损金额 / 总资金
   - 若当前跌停且无法成交→实际止损金额=(入场价-跌停价)×持仓数量，按此值校验红线
   - 校验1：止损占比 ≤ 1% → ✅ 风险正常
   - 校验2：1% < 止损占比 ≤ 2% → ⚠️ 风险偏高，建议减仓至1%
   - 校验3：止损占比 > 2% → ❌ 超硬上限，必须立即止损或减仓
   - 完成标准：止损红线校验完成

6. **跌停处理**
   - 若未跌停 → 正常止损指令
   - 若跌停且连续跌停 < 3日 → 排队卖出（限价单挂跌停价）
   - 若跌停且连续跌停 ≥ 3日 → 改市价单（避免排队过久扩大亏损）
   - 连续跌停≥3日→触发账户级回撤校验（委托01）：若回撤≥20%→清仓冷静；若回撤<20%→继续排队+标记实际风险敞口（按跌停价估算）
   - 完成标准：跌停处理策略已确定

7. **输出格式**
   ```
   ✅止损已设定 | 股票:XXXX | 止损价:XX元 | 止损金额:XXXX元(总资金X%)
   当前价:XX元 | 距止损:X%(XX元) | 操作:持有待涨
   ```
   ```
   ⚠️T+1限制 | 股票:XXXX | 买入当日无法止损 | 次日策略:集合竞价9:25处理
   ```
   ```
   ⚠️跌停排队 | 股票:XXXX | 连续跌停X日 | 策略:限价单挂跌停价(连续≥3日改市价单)
   ```
   ```
   ❌超硬上限 | 股票:XXXX | 止损占比X%>2% | 操作:立即止损/减仓至1%
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 入场判断（走 05-breakout-entry）
- 头寸计算（走 06-position-sizing）
- 盈利出场/趋势出场（走 08-trend-exit，本 skill 只管亏损出场）
- 价值投资长期持有（不止损，靠基本面持有）

### 作者在书中警告的失败模式

- **止损下移**：海龟铁律"止损只能上移不能下移"。常见错误是亏损后把止损往下移"再等等"，导致小亏变大亏
- **不止损**：海龟强调"触及止损立即平仓，没有例外"。最常见的失败是"再等等看"，结果亏损扩大
- **固定百分比止损**：用固定5%止损而非ATR止损，波动大的市场会被噪音震出，波动小的市场止损太宽

### 作者的盲点 / 时代局限

- 海龟设计于T+0市场，止损可即时执行；A股T+1限制买入当日无法止损
- 海龟无涨跌停限制；A股跌停时排队卖出可能数日无法成交
- 2×ATR参数对低波动蓝筹股可能偏宽，可考虑1.5×ATR

### 容易混淆的邻近方法论

- "ATR止损"vs"固定百分比止损"：ATR止损随波动自适应，固定百分比不随波动调整
- "止损"vs"止盈"：止损是亏损出场（本skill），止盈是盈利出场（08-trend-exit）
- "加仓止损上移"vs"移动止损"：加仓止损上移是加仓触发的，移动止损是价格新高触发的
