# K线信号层 — 形态识别 + ATR 位置确认 + 强度分级

## R — 原文 (Reading)

> "它们的下影线较长，而实体较小，并且在其全天价格区间里，实体处在接近顶端的位置上……具体情况要由它们在趋势中所处的位置来决定。只要它出现在下降趋势中，那么它就是下降趋势即将结束的信号。"

> — 尼森，《日本蜡烛图技术》第4章（OCR原文，page 1463-1468）

---

## I — 方法论骨架 (Interpretation)

K线形态是"语言"，位置和成交量是"语境"——脱离语境的形态毫无意义。

1. **核心反转形态**：
   - 单根：锤子线（底反转）/ 上吊线（顶反转）
   - 双根：吞没形态（强反转）/ 乌云盖顶（顶反转）/ 刺透形态（底反转）
   - 三根：晨星（底反转）/ 暮星（顶反转）
2. **持续形态**：三法 / 窗口 / 十字星
3. **位置确认**（关键修复）：形态必须在关键支撑/阻力的 ATR 范围内才有效
   - 关键位 ± 1×ATR 内 → 位置有效
   - 关键位 ± 1×ATR 外 → 位置无效，形态降级
4. **成交量确认**：反转形态需成交量放大（≥ 5 日均量 1.5 倍）
5. **信号强度分级**：
   - 强：关键位置 + 成交量确认 + 多形态共振（3 项全满足）
   - 中：满足 2 项
   - 弱：仅满足 1 项

核心心法：**形态是必要条件，位置+量能是充分条件**。

## A1 — 书中的应用 (Past Application)

### 案例 1: 锤子线的位置依赖
- **问题**: 同样是锤子线，为什么有时反转有时继续跌
- **方法论的使用**: 检查锤子线是否出现在关键支撑位的 ATR 范围内
- **结论**: 关键位 + ATR 内 + 放量 = 有效反转；否则只是噪音
- **结果**: 过滤掉 70% 的假信号

### 案例 2: 吞没形态的成交量确认
- **问题**: 吞没形态出现后却不反转
- **方法论的使用**: 检查成交量是否 ≥ 5 日均量 1.5 倍
- **结论**: 无量吞没不可信，需等待量能确认
- **结果**: 避免了无量假反转的陷阱

### 案例 3: 晨星三形态共振
- **问题**: 单根形态可靠性低
- **方法论的使用**: 晨星（三根形态）+ 关键位置 + 放量 = 三重共振
- **结论**: 多形态共振的信号强度最高
- **结果**: 共振信号的胜率显著高于单根形态

## E — 可执行步骤 (Execution)

当 skill 被激活后，agent 应按以下步骤执行：

1. **前置校验**
   - 完成标准: 确认已通过 `01-risk-control-baseline`
   - 判停条件: 若未跑风控 → 强制先跑 01

2. **收集输入参数**
   - 完成标准: 明确 K线数据（OHLC）、ATR、成交量、当前位置 vs 关键支撑/阻力
   - 判停条件: 若缺 ATR 或关键位数据 → ⚠️待补充

3. **形态识别**
   - 完成标准: 识别当前 K线形态类型
   - 反转形态：锤子线 / 上吊线 / 吞没 / 乌云盖顶 / 刺透 / 晨星 / 暮星
   - 持续形态：三法 / 窗口 / 十字星
   - 无明确形态 → ❌ 无信号

4. **位置确认**（关键修复）
   - 完成标准: 检查形态是否在关键位的 ATR 范围内
   - 公式: `|当前价 - 关键位| ≤ 1×ATR` → 位置有效
   - `|当前价 - 关键位| > 1×ATR` → 位置无效，形态降级
   - 判停条件: 若位置无效 → 信号强度直接降为弱

5. **成交量确认**
   - 完成标准: 检查成交量是否 ≥ 5 日均量 × 1.5
   - 放量 → ✅ 确认
   - 缩量 → ❌ 不确认
   - 判停条件: 若无成交量数据 → ⚠️待补充

6. **信号强度分级**
   - 完成标准: 按三项满足程度分级
   - 强（3 项全满足）：关键位置 + 成交量确认 + 多形态共振
   - 中（满足 2 项）
   - 弱（仅满足 1 项）
   - 判停条件: 若为弱信号 → ❌ 不交易

7. **输出综合结果**
   - 完成标准: 给出三类输出之一 + 信号详情
   - ✅ 强信号：信号类型 + 方向（多/空）+ 强度等级 + 确认条件
   - ⚠️ 中信号：列出缺失项 + 等待建议
   - ❌ 弱信号/无信号：忽略，不交易

   输出格式：
   ```
   ✅强信号 | 形态:锤子线 | 方向:多 | 强度:强 | 位置:关键位ATR内 | 量比:1.8x | 下一步:05入场
   ⚠️中信号 | 形态:吞没 | 缺失项:量能不足 | 等待建议:放量确认
   ❌弱信号/无信号 | 原因:位置无效/无量/无形态
   ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯基本面分析**: 用户问"公司好不好"不涉及 K线 → 走 02-stock-screening
- **纯趋势判断**: 用户问"大方向是什么"不涉及具体形态 → 走 04-trend-key-level
- **未跑风控**: 前置条件未满足
- **无 ATR 数据**: 无法做位置确认，形态判断无意义

### 作者在书中警告的失败模式

- **脱离位置看形态**: 锤子线在下跌中继只是停顿，不是反转
- **无量形态**: 无量反转不可信，必须等量能确认
- **单根形态过度解读**: 单根形态可靠性低，需多形态共振
- **忽视趋势方向**: 逆趋势的反转信号要打折扣

### 作者的盲点 / 时代局限

- **A 股 T+1 限制**: 当天形态确认后无法当天买入，次日跳空可能让信号失效
- **涨跌停板影响**: 涨停/跌停时的 K线形态失真
- **量化交易冲击**: 高频交易让传统 K线形态的可靠性下降

### 容易混淆的邻近方法论

- **趋势分析**: 趋势看长期方向，K线看短期信号——不要用 K线代替趋势判断
- **西方技术指标**: MACD/RSI 等是衍生指标，K线是原始价格——本 skill 专注 K线

### 禁忌红线（不可逾越）

- 位置无效（距关键位 > 1×ATR）→ ❌ 信号降级为弱
- 无成交量确认 → ❌ 信号降级为弱
- 弱信号 → ❌ 禁止交易
- 逆趋势的反转信号 → ⚠️ 需额外确认
