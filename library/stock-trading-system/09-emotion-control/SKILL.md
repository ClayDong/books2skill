# 情绪控制层 — 交易心理防火墙

## R — 原文 (Reading)

> "交易的冷酷现实是每笔交易的结果都不确定，除非你学会彻底接受结果不确定的可能性，否则你会有意或无意地避免你定义为痛苦的可能性。交易者学会根据概率思考后，会对自己的整体成功很有信心。"

> — 马克·道格拉斯，《交易心理分析》（OCR原文，page 127-131）

> "炒股的三个心理障碍是恐惧、贪婪和希望。恐惧让你该买不敢买，贪婪让你该卖不肯卖，希望让你该止损不止损。"

> — 陈江挺，《炒股的智慧》

> "被剥夺超级反应倾向：一个人失去已有的东西比未得到更痛苦，导致急于扳平的非理性行为。社会认同倾向：人在不确定时倾向于模仿他人的行为。避免怀疑倾向：人脑天生倾向于尽快消除怀疑，做出草率决策。"

> — 查理·芒格，《穷查理宝典》第十一讲"人类误判心理学"

## I — 方法论骨架 (Interpretation)

交易情绪失控的根因是"把概率事件当成了对错评判"。道格拉斯的概率思维是地基：每笔交易只是样本空间的一个采样，盈亏是统计结果不是个人成败。

陈江挺的三障碍模型（恐惧/贪婪/希望）描述了情绪如何扭曲决策：恐惧锁手、贪婪延卖、希望扛亏。学股四阶段（蛮干→摸索→体验→领悟）说明情绪控制是渐进习得的。

芒格的5个关键心理偏误提供了可检测的偏差信号：被剥夺超级反应（亏损后急于扳平）、社会认同（从众）、避免怀疑（急于决策）、压力影响（压力下决策失调）、过度乐观（连续盈利后飘）。

本 skill 的核心是"3问快速评估 + 认知偏差检测清单 + 情绪分级"，把模糊的心理状态转成可操作的绿/黄/红三档建议。

## A1 — 书中的应用 (Past Application)

### 案例1: 道格拉斯的概率思维训练
- **问题**: 交易者连续亏损3笔后，第4笔信号出现却不敢执行
- **方法论的使用**: 道格拉斯让交易者把每笔交易视为抛硬币——单次结果不可预测，但大量样本下概率会显现。关键不是"这次对不对"，而是"这个信号长期正期望吗"
- **结论**: 把注意力从"单笔对错"转移到"系统期望值"，情绪干扰降低
- **结果**: 交易者能按信号执行，不再因前一笔亏损而跳过下一笔

### 案例2: 芒格的"被剥夺超级反应"识别
- **问题**: 投资者亏损后加大仓位试图"扳回来"
- **方法论的使用**: 芒格指出这是"被剥夺超级反应倾向"——失去已有的东西比未得到更痛苦，导致急于扳平的非理性加仓
- **结论**: 识别出这是心理偏差而非理性判断，强制冷静24小时再决策
- **结果**: 避免了"越亏越加"的毁灭性螺旋

### 案例3: 陈江挺的"希望"陷阱
- **问题**: 持仓跌破止损位，交易者"希望"它反弹而不执行止损
- **方法论的使用**: 陈江挺区分"希望"和"分析"——止损位是分析得出的，"希望反弹"是情绪。当希望替代分析时，必须执行纪律
- **结论**: 止损是系统的一部分，不是对错判断
- **结果**: 执行止损后，资金保住，能在下一个信号重新进场

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **亏损后情绪化操作**：刚亏了一笔，想加大仓位扳回来，或想立刻换股追另一个
2. **连续盈利后飘了**：连赚几笔后觉得自己"开窍了"，准备加杠杆或放松风控
3. **市场狂热时从众**：看到身边人/论坛都在赚钱，忍不住追高买入
4. **压力下强行交易**：生活压力大（缺钱/有急事），试图靠交易解决，但状态明显不对
5. **急于决策跳过分析**：怕错过行情，没做完检查清单就冲进去

### 语言信号 (用户的话里出现这些就应激活)

- "亏了想赶紧扳回来"
- "连续赚了好几笔，感觉找到感觉了"
- "大家都在买，我不买怕踏空"
- "压力太大但还是想交易"
- "来不及分析了，先冲进去再说"
- "这次肯定能涨"（过度乐观）
- "止损了不甘心"

### 与相邻 skill 的区分

- 与 `10-system-optimization` 的区别：本 skill 管的是"情绪状态评估和心理偏差检测"，系统优化管的是"R乘数和期望值计算"。情绪不对时先调本 skill，情绪稳定后再调系统优化
- 与 `11-trade-review` 的区别：本 skill 是"事前/事中"的情绪防火墙，复盘是"事后"的归因分析。交易前情绪不对→本 skill；交易后要找问题→复盘
- 与 `12-mindset-calibration` 的区别：本 skill 检测具体认知偏差并给操作建议（减仓/暂停），心态校准是更宏观的"时位判断"和心境调适，不输出交易指令

### 下一步调用指引

- 情绪评估为🟢绿色（状态正常）→ 可继续交易，调用 `10-system-optimization` 优化系统
- 情绪评估为🟡黄色（有偏差苗头）→ 减仓至50%，调用 `11-trade-review` 复盘近期交易找原因
- 情绪评估为🔴红色（严重偏差）→ 暂停交易24小时，调用 `12-mindset-calibration` 做心态校准，冷静24小时后，必须重新调用 `01-risk-control-baseline` 通过后方可恢复交易

## E — 可执行步骤 (Execution)

**输入**：近期交易记录（至少3笔）、当前持仓盈亏、用户描述的心理状态

### 步骤1: 3问快速评估（≤2分钟）

对用户提出以下3个问题，根据回答快速判断情绪基线：

1. **"你现在的交易决策，是基于信号还是基于情绪？"**
   - 回答"信号" → 计0分；回答"情绪/感觉" → 计2分
2. **"如果这笔交易亏了，你的第一反应是什么？"**
   - 回答"按计划止损" → 计0分；回答"不甘心/想扳回来" → 计2分
3. **"你最近一次违背自己交易系统是什么时候？"**
   - 回答"没有/很久以前" → 计0分；回答"最近/今天" → 计2分

- **完成标准**: 3问得分汇总（0-6分）
- **判停条件**: 若用户拒绝回答或回答明显敷衍 → 直接标🔴红色，建议暂停交易

### 步骤2: 认知偏差检测清单

逐项核对以下5项偏差，命中任意1项即标记：

| 偏差 | 检测信号 | 命中动作 |
|------|---------|---------|
| 被剥夺超级反应 | 近期有亏损且当前想加仓/换股扳回 | 强制冷静24小时 |
| 社会认同 | 决策依据含"别人都在买/论坛说/群里推荐" | 要求独立验证信号来源 |
| 避免怀疑 | 从产生想法到下单<10分钟且未走检查清单 | 退回走清单 |
| 压力影响 | 用户自述生活压力大/缺钱/有急事 | 建议降仓至30%以下 |
| 过度乐观 | 近期连续盈利≥3笔且自述"感觉对了/开窍了" | 触发回撤保护：下笔仓位减半 |

- **完成标准**: 5项全部核对完毕，记录命中的偏差类型和数量
- **判停条件**: 若命中≥2项偏差 → 直接标🔴红色，跳到步骤3输出"暂停交易"

### 步骤3: 情绪分级与建议输出

根据步骤1得分 + 步骤2命中数综合分级：

| 分级 | 条件 | 建议措施 |
|------|------|---------|
| 🟢 绿色 | 3问得分0分 且 偏差命中0项 | ✅可执行：正常交易，按系统信号操作 |
| 🟡 黄色 | 3问得分2分 或 偏差命中1项 | ⚠️待补充：减仓至50%，完成偏差修正后再恢复 |
| 🔴 红色 | 3问得分≥4分 或 偏差命中≥2项 | ❌不可执行：暂停交易24小时，冷静24小时后必须重新调用01-risk-control-baseline通过后方可恢复交易 |

- **完成标准**: 输出情绪状态（绿/黄/红）+ 检测到的偏差列表 + 建议措施（持有/减仓/暂停）
- **输出格式**:
  ```
  情绪状态: 🟢/🟡/🔴
  检测偏差: [偏差1, 偏差2] 或 [无]
  建议措施: {具体操作}
  下一步: {调用哪个skill}
  ```

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **纯技术面/基本面分析**：用户在讨论K线形态、财务指标、估值模型时，不调用本 skill（走量化系统）
- **仓位计算**：用户在问"该买多少股""仓位怎么分配"时，不调用本 skill（走仓位管理 skill）
- **选股逻辑**：用户在问"这只股票基本面怎么样"时，不调用本 skill
- **趋势判断**：用户在问"现在是牛市还是熊市"时，不调用本 skill

### 禁忌红线

- ❌ **不输出买卖信号**：本 skill 只评估情绪状态，不判断"该买还是该卖"
- ❌ **不替代止损纪律**：情绪评估为绿色不等于可以不设止损
- ❌ **不鼓励"扛单"**：即使情绪稳定，跌破止损位仍必须执行
- ❌ **不作为加杠杆的依据**：情绪好不等于可以放大风险

### 作者在书中警告的失败模式

- 道格拉斯警告：把概率思维当成"每次都亏一点没关系"的借口——概率思维的前提是系统有正期望值
- 陈江挺警告：学股四阶段的"蛮干期"最容易爆仓，因为无知+自信=灾难
- 芒格警告：心理偏误的可怕之处在于"你不知道自己中了"，自我感觉良好时恰恰最危险

### 作者的盲点 / 时代局限

- 道格拉斯的概率思维假设市场是相对有效的随机游走，但极端行情（如熔断、黑天鹅）下概率分布会失效
- 陈江挺的经验来自美股个人投资者时代，A股的T+1和涨跌停板制度会放大情绪效应
- 芒格的偏误清单是定性描述，缺乏量化阈值——本 skill 用"连续盈利≥3笔""下单<10分钟"等量化标准做了补强

### 容易混淆的邻近方法论

- **"心态好就能赚钱"**：错误。情绪控制是必要条件不是充分条件，没有正期望系统再冷静也亏
- **"冥想/深呼吸就能控制情绪"**：不够。本 skill 强调的是"偏差检测+分级+操作建议"，不是简单的放松技巧
