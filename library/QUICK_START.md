## 三、5 个核心 skill (入门必看)

如果你时间有限,**先看这 5 个**:

### 1. dashixiong-keep-one-bite (留一口)
- **为什么先看**: 风险控制是基础
- **核心**: 永远不满仓 + 永远不加杠杆
- **应用**: 任何投资决策

### 2. dashixiong-long-termism (长期主义)
- **为什么先看**: 心态是基础
- **核心**: 短期不重要,长期才重要
- **应用**: 任何长期决策

### 3. i-ching-time-position (时位中正)
- **为什么先看**: 时机是基础
- **核心**: 时/位/中/正 四维自检
- **应用**: 任何重大决策

### 4. mankiw-opportunity-cost (机会成本)
- **为什么先看**: 成本思维是基础
- **核心**: 选A的真实成本是放弃什么
- **应用**: 任何选择决策

### 5. gyfz-cake-balance (蛋糕权衡)
- **为什么先看**: 发展分配权衡是基础
- **核心**: 做大蛋糕+分好蛋糕动态平衡
- **应用**: 任何发展/分配决策

## 五、调用方式

### 方式 1: 直接调用 skill-router
- 当用户说"我该不该做X"时,自动调用 skill-router
- skill-router 会按 5 层调用相关 skill

### 方式 2: 直接调用具体 skill
- 当用户问题明确指向某个 skill 时,直接调用
- 例如:"我该不该加仓" → 直接 dashixiong-keep-one-bite

### 方式 3: 场景组合
- 当用户场景复杂时,按"场景速查"组合调用

## 六、最重要的 5 条原则

### 1. 5 层决策

```
价值观 → 哲学 → 操作 → 学科 → 政策
```

### 2. 长期视角

- 短期不重要
- 长期才重要
- 10 年视角

### 3. 留一口

- 永远不满仓
- 永远不加杠杆
- 永远有子弹

### 4. 机会成本思维

- 选A的真实成本是放弃B
- 不只看显性成本,看隐性成本

### 5. 动态平衡

- 发展与分配动态平衡
- 效率与公平阶段性调整
- 不走极端
