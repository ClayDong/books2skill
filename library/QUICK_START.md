## 三、4 个核心 skill (入门必看)

如果你时间有限,**先看这 4 个**:

### 1. dashixiong-keep-one-bite (留一口)
- **为什么先看**: 风险控制是基础
- **核心**: 永远不满仓 + 永远不加杠杆
- **应用**: 任何投资决策

### 2. dashixiong-long-termism (长期主义)
- **为什么先看**: 心态是基础
- **核心**: 短期不重要,长期才重要
- **应用**: 任何长期决策

### 3. i-ching-time-position (时位中正)
- **为什么先看**: 时机是基础
- **核心**: 时/位/中/正 四维自检
- **应用**: 任何重大决策

### 4. dashixiong-giving (付出)
- **为什么先看**: 价值观是基础
- **核心**: 人生 = 付出
- **应用**: 任何人生/职业决策

## 五、调用方式

### 方式 1: 直接调用 skill-router
- 当用户说"我该不该做X"时,自动调用 skill-router
- skill-router 会按 4 层调用相关 skill

### 方式 2: 直接调用具体 skill
- 当用户问题明确指向某个 skill 时,直接调用
- 例如:"我该不该加仓" → 直接 dashixiong-keep-one-bite

### 方式 3: 场景组合
- 当用户场景复杂时,按"场景速查"组合调用

## 六、最重要的 5 条原则

### 1. 4 层决策

```
价值观 → 哲学 → 操作 → 学科
```

### 2. 长期视角

- 短期不重要
- 长期才重要
- 10 年视角

### 3. 留一口

- 永远不满仓
- 永远不加杠杆
- 永远有子弹

### 4. 付出哲学

- 人生 = 付出
- 不求回报
- 长期复利

### 5. 时位中正

- 时: 时机
- 位: 位置
- 中: 不偏激
- 正: 合规合理

## 七、避免的 5 个错误

### 1. 满仓 + 杠杆
- 任何情况下都不要
- 即使再看好

### 2. 短期主义
- 短期不重要
- 不要被短期波动影响

### 3. 亢龙有悔
- 不要顶峰时张扬
- 不要顶峰时听不进意见

### 4. 不付出
- 索取是痛苦的根源
- 付出是快乐的根源

### 5. 急功近利
- 长期主义
- 慢就是快
- 扫地僧境界
