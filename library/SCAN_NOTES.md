## OCR 方案建议

对于曼昆经济学和共同富裕，有 3 个 OCR 方案:

1. **Tesseract OCR (免费)**: `pip install pytesseract` + Tesseract-OCR 软件。中文识别准确率约 85-90%。
2. **PaddleOCR (免费)**: 百度开源，中文识别准确率约 95%+。`pip install paddlepaddle paddleocr`
3. **付费 OCR API**: 如百度文字识别、腾讯云 OCR。准确率最高，但有成本。

建议用 PaddleOCR，性价比最高。
